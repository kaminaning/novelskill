# 第116-122章 摘要

## 第116章 品牌整合

**概要**：陆明召开高层会议，宣布全面启用"云梦泽"作为集团核心品牌矩阵，将万家福更名"云梦泽生活广场"，西山温泉定名"云梦泽·温泉小镇"。同时部署编制《云梦泽投资白皮书》，汇总所有已落地和规划项目，作为高铁站争取的核心筹码。

**关键事件**：
1. 陆明宣布全面启用"云梦泽"品牌矩阵，万家福更名为"云梦泽生活广场"
2. 温泉小镇定名"云梦泽·温泉小镇"，东区住宅定名"云梦泽·云境天著"
3. 赵一舟担忧换牌流失客户，陆明认为认的是格局和服务而非名字
4. 陆明部署方瑜和陆鸢牵头编制《云梦泽投资白皮书》，汇总所有项目
5. 团队通宵加班完成百页白皮书，涵盖生活广场、温泉小镇、产业园等全部项目

**出场人物**：陆明、方瑜、陆鸢、沈璃、赵一舟、秦业、陈思甜、李曼、孙长明

主题标签：品牌战略 | 基调：振奋、高效

## 第117章 高铁之争，正主到场

**概要**：孙长明连夜赴省城向孟昭华汇报白皮书，孟昭华对陆明实力大感震惊。孟昭华指出平林和新泰都带企业代表参会，批评孙长明不应防备陆明，要求陆明亲自参加论证会。孙长明意识到防备心理过重，打电话邀请陆明赴省城。

**关键事件**：
1. 孙长明连夜赴省城，向孟昭华展示《云梦泽投资白皮书》
2. 孟昭华翻阅白皮书后震惊于陆明三个月投入近二十亿真金白银
3. 孟昭华指出平林县带三家制造企业董事长、新泰县带物流巨头参会
4. 孟昭华批评孙长明防备陆明接触省一级资源是本末倒置
5. 孟昭华提醒孙长明退休前高配副厅的最后一次机会，劝其放下成见

**出场人物**：孙长明、孟昭华、陆明、沈璃

主题标签：省城博弈 | 基调：紧迫、深思

## 第118章 立足当下，放眼未来

**概要**：省发改委高铁站论证会上，平林县提出三十亿投资但要地要政策要免税，新泰县要省里拨款修路。陆明代表云梦县发言，直指对手方案本质是套取政策红利，亮出云梦县不要一分钱补贴、实打实投资的底牌，以压倒性优势碾压对手。

**关键事件**：
1. 平林县提出三十亿意向投资但要五年到位、要土地指标倾斜和税收放宽
2. 新泰县提出十五亿物流园但要省里专项拨款修双向八车道连通线
3. 孙长明递交《云梦泽投资白皮书》，展示全部已落地项目
4. 陆明当场揭穿平林方案是"套取政策红利"，亮出不要省里一分钱的底牌
5. 陆明以"立足当下，放眼未来"作结语，周主任等三人默契点头，胜负已分

**出场人物**：陆明、孙长明、周主任、孟昭华、赵总、平林书记、新泰县长、各企业家

主题标签：论证会碾压 | 基调：霸气、振奋

## 第119章 中心路收工！

**概要**：论证会后孟昭华越过孙长明直接与陆明握手。孙长明破例乘坐陆明的库里南回程，途中接到周启明电话报告中心路完工。孙长明现场组织验收，各局确认质量超标。陆明随后招揽周启明，要求其将恒达建筑总部迁至云梦县，锁定百亿基建合作。

**关键事件**：
1. 孟昭华越过孙长明直接与陆明握手，暗示结果有利
2. 孙长明坐陆明的库里南回程，陆明调侃"这次不会有泥头车了吧"
3. 中心路现场验收，沥青加厚至12公分、管网加粗30%，质量全面超标
4. 老百姓围观新路交口称赞，孙长明公开感谢陆明
5. 陆明招揽周启明，以未来三年百亿基建为筹码，要求恒达建筑迁至云梦县

**出场人物**：陆明、孙长明、孟昭华、周启明、白崇文、方珩、方瑜

主题标签：验收招揽 | 基调：振奋、务实

## 第120章 突击考察

**概要**：周启明同意将恒达建筑迁至云梦县。农产品深加工产业园奠基仪式盛大举行，陆明三句话"造福百姓"引爆全场。仪式进行中突发消息——省发改委周主任带队突击考察，车队已进云梦县界。孙长明和陆明临危不乱，决定让考察组直接看最真实的场景。

**关键事件**：
1. 周启明被陆明三年百亿基建的前景打动，同意半个月内将公司迁至云梦县
2. 产业园奠基仪式盛大举行，孙长明致辞，陆明喊出"三件事都是造福百姓"
3. 张广华接孟昭华急电：周主任带队突击考察，车队二十分钟内到达
4. 考察组不打招呼直奔现场，摆明要看最真实的云梦县
5. 陆明安抚孙长明"眼前的场景就是最好的汇报"，安排考察组直奔奠基现场

**出场人物**：陆明、周启明、孙长明、张广华、沈璃、陆鸢、白崇文、孟昭华、周主任

主题标签：奠基仪式、突击考察 | 基调：紧张、果敢

## 第121章 高下立判

**概要**：考察组抵达产业园现场，周主任直接询问有无现成200亩地——意味着云梦县已成首选。随后考察云梦泽生活广场和温泉小镇，服务质量、员工薪资、拆迁零阻力等事实令专家组印象深刻。周主任心中已将云梦县与平林、新泰"高下立判"。当晚孟昭华短信通知陆明"基本大局已定"。

**关键事件**：
1. 周主任在现场直接问有无200亩现成土地，意味着云梦县已成首选
2. 考察组考察云梦泽生活广场，亲见客流火爆、员工底薪五千、服务质量一流
3. 周主任与超市员工交谈，了解到返乡就业、薪资不输南方的真实情况
4. 考察温泉小镇工地，孙长明确认拆迁零阻力，陆明承诺成本不设上限做品质
5. 孟昭华当晚发短信告知陆明"基本大局已定"，专家组对平林和新泰未过多停留

**出场人物**：陆明、孙长明、周主任、孟昭华、方瑜

主题标签：考察决胜 | 基调：自信、期待

## 第122章 高铁站正式落户！

**概要**：省发改委召开决议会议，各方一致同意豫中南高铁枢纽站落户云梦县。市委高国强电话通知孙长明，省政府常务会议全票通过。孙长明激动地冲出办公室直奔新城大厦，当面向陆明宣布喜讯。陆明下令全场八折庆祝三天、全体员工发双薪，站在落地窗前展望正在苏醒的县城。

**关键事件**：
1. 省发改委决议会议上，副主任、赵总、财政厅张厅长、孟昭华一致支持云梦县
2. 周主任一锤定音，豫中南高铁枢纽站落户云梦县，上报省政府批复
3. 市委高国强电话通知孙长明，省政府常务会议全票通过
4. 孙长明激动地推掉常委会，直奔新城大厦当面告知陆明
5. 陆明下令云梦泽全场八折庆祝三天、全体员工双薪，展望商业帝国最重要拼图完成

**出场人物**：周主任、副主任、赵总、张厅长、孟昭华、高国强、孙长明、陆明、秦业、周启明、陆鸢、沈璃、方瑜

主题标签：高潮决胜 | 基调：激动、辉煌
