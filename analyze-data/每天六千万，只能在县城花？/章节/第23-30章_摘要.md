# 第23-30章 摘要合集

## 第23章 法院院长是我姑父

**概要**：陆明就万家福抵押登记问题致电方瑜调查。查明两年前王大发向胡奎借款三百万并以万家福土地使用权做抵押，借款已还清但胡奎拒不配合注销。方瑜提出诉前保全加速裁方案，七天内解决。方瑜与法院赵院长为姑侄关系。

**关键事件**：1.查明抵押登记来龙去脉 2.方瑜提出诉前保全方案 3.赵院长姑侄关系浮出水面

**出场人物**：陆明、方瑜、陆鸢、沈璃

**情节点**：
P1 抵押缘由查明：方瑜查明两年前王大发向胡奎借三百万以土地使用权抵押
P2 还款未注销：王大发确认借款已还清但胡奎拖着不签字
P3 路径一：起诉确认抵押权消灭，普通程序三个月
P4 路径二：带押过户但不解决根本问题
P5 路径三（选择）：诉前保全加速裁，七天解决，需三百万担保
P6 赵院长关系揭露：方瑜确认赵院长是其姑父，强调合法框架内提速
P7 陆鸢质疑：陆鸢担心胡奎使阴招，陆明表示规则之内奉陪
P8 退礼指令：陆明命沈璃退回胡奎的茅台和大闸蟹

主题标签：法律博弈 | 基调：紧张

## 第24章 诉前保全

**概要**：沈璃将胡奎礼品原封退回，方瑜到法院立案当天获得诉前保全裁定。胡奎求助法院内部关系"张哥"，张哥听闻原告律师是方瑜后拒绝帮忙。

**关键事件**：1.沈璃退礼 2.方瑜当天获得保全裁定 3.胡奎求助被拒

**出场人物**：沈璃、方瑜、胡奎、张哥（法院内部关系人）

**情节点**：
P1 退礼执行：沈璃将礼品退至奎盛建材并附等价回礼
P2 消息外传：胡奎手下在朋友圈发"敬酒不吃吃罚酒"
P3 法院立案：方瑜带完整还款凭证申请诉前保全
P4 保全裁定下达：法院当天下午下达诉前保全裁定
P5 万家福工期推演：沈璃制定35天改造甘特图
P6 胡奎求助被拒：法院关系人张哥得知原告是方瑜后拒绝帮忙

主题标签：法律攻防 | 基调：紧张

## 第25章 胡奎的妥协

**概要**：胡奎求助失败后主动致电陆明，陆明以"茶是我泡的，脸是你撕的"点明先送礼后设卡的实质，提出当天签字注销即撤诉。胡奎权衡后同意，双方在不动产登记中心完成抵押权注销。

**关键事件**：1.陆明电话施压迫使胡奎妥协 2.房管局完成抵押注销 3.万家福过户障碍清除

**出场人物**：陆明、胡奎、方瑜、陆鸢

**情节点**：
P1 胡奎致电质问：陆明以茶面之喻点明对方先礼后兵
P2 陆明开出条件：今天签字注销即撤诉，否则速裁败诉损失更大
P3 胡奎权衡后同意
P4 陆鸢评价陆明为"笑面虎"
P5 房管局面签完成抵押权注销
P6 方瑜预警胡奎不会罢休

主题标签：正面交锋 | 基调：紧张

## 第26章 穷不与富斗

**关键事件**：1.三叔讲述胡奎发迹史 2.父亲劝圆滑处世 3.母亲安排相亲

**出场人物**：陆明、陆建国（父）、陆建军（三叔）、王爱玲（母）

**情节点**：
P1 家宴场景：三叔和父亲喝酒等陆明
P2 三叔讲述胡奎二十年发迹史
P3 父亲以"穷不与富斗"劝子圆滑

P5 陆明表态不退让但方式更灵活
P6 母亲安排相亲

## 第27章 环评是个筐，什么都能装

**概要**：陆明前往住建局办理万家福改造报备，被建管科钱志刚以"环评报告"为由卡住。方瑜确认补充通知适用范围不包括内部翻新，钱志刚故意扩大解释。陆明判断是胡奎在背后推动。

**关键事件**：1.报备被钱志刚以环评卡住 2.胡奎与钱志刚疑似勾连 3.方瑜确认文件被故意扩大解释

**出场人物**：陆明、胡奎、钱志刚（建管科长）、方瑜

**情节点**：
P1 住建局停车场看到胡奎的路虎
P2 钱志刚以补充通知要求环评报告
P3 陆明援引法规反驳，钱志刚称基层有自由裁量权
P4 钱志刚拒绝出示书面意见
P5 方瑜查明补充通知不适用内部翻新
P6 陆明判断胡奎从住建局堵上新口子
P7 母亲催相亲

主题标签：行政阻力 | 基调：紧张
