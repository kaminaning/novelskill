# 第101-110章 摘要

---

## 第101章 各方反应

**概要**：陆明遭遇无牌泥头车撞击被送医抢救，消息传开后云梦县官场震动。县委书记孙长明连夜封锁全城追查肇事者，各局长纷纷赶往医院表态。陆明经抢救脱离生命危险，方瑜在医院冷静酝酿法律反击，三叔陆建军提着钢管赶到医院暴怒发飙。

**关键事件**：
1. 无牌泥头车在建设路十字路口撞击陆明的迈巴赫，肇事车逃逸
2. 孙长明下令公安局长陈建平封锁全县出城路口，亲自赶往医院
3. 财政局长王卫国、商务局长张广华、住建局长白崇文等连夜赶到医院
4. 陆明经抢救无生命危险，昏迷因高血压引发脑供血不足和轻微脑震荡
5. 陆建军提着实心钢管冲到医院走廊，怒砸墙壁，歇斯底里暴喝

**出场人物**：陆明、孙长明、小郑、陈建平、王卫国、张广华、马局长、周岩、白崇文、方珩、郑强、刘彪、沈璃、陆鸢、方瑜、赵国志、方娟、陆建国、王爱玲、陆建军

主题标签：车祸惊变 | 基调：紧张、愤怒

## 第102章 我只要他死

**概要**：陆明在医院苏醒，沈璃因紧绷过久脱力晕倒。陆明任命方珩为安保部门经理并加薪，孙长明亲赴公安局指挥中心督导办案。方瑜在医院走廊向姑父赵国志表明要对苏文发起刑事诉讼的决心，不顾赵国志劝阻，坚定表示"我只要他死"。

**关键事件**：
1. 陆明苏醒后第一句话调侃泥头车"有力气"，沈璃因长期紧绷脱力晕倒
2. 陆明任命方珩为安保部门经理，每月加薪一万
3. 孙长明在公安局指挥中心亲自督导，陈建平汇报泥头车提前踩点等候四十分钟
4. 方瑜向赵国志表明要对苏文发起刑事诉讼，赵国志劝其从大局考虑
5. 方瑜不顾劝阻，一字一顿说出"我只要他死"

**出场人物**：陆明、沈璃、陆鸢、方珩、孙长明、陈建平、方瑜、赵国志、陆建军

主题标签：觉醒反击 | 基调：冷静、坚定、激烈

## 第103章 全部抹除

**概要**：方瑜彻夜起草三份法律文书——刑事控告书、立案监督申请书、诉前财产保全申请书，一上午跑遍公安局、检察院、法院三个部门，将苏文所有法律退路全部锁死。陆明出院后收到方瑜发来的回执，赞其"得加工资了"。泥头车司机张伟被抓后迅速供认受苏文指使。

**关键事件**：
1. 方瑜通宵起草刑事控告书、立案监督申请书、诉前财产保全申请书三份文书
2. 方瑜一上午跑遍公安局、检察院、法院，三个部门全部办妥
3. 法院十点四十五分即出具财产保全裁定书，冻结苏文全部财产
4. 陆明出院，交代陆鸢采购安全车辆，配给公司核心人员
5. 泥头车司机张伟被抓后坦然承认受苏文指使

**出场人物**：方瑜、陆明、陆鸢、方珩、于科长、陈建平、张伟

主题标签：法律雷霆 | 基调：凌厉、高效

## 第104章 我冤枉啊

**概要**：苏国栋得知消息后怒扇苏文耳光，苏文极力辩解自己没有杀人动机——此时已被医保局调查，再买凶杀人毫无逻辑。父子二人意识到有人栽赃嫁祸，苏国栋决定让苏文逃亡。就在苏文准备翻墙逃跑的瞬间，警察破门而入将其带走。

**关键事件**：
1. 苏国栋怒扇苏文，质问是否雇泥头车撞陆明
2. 苏文冷静分析自己没有杀人动机——医保案最多判三五年，买凶杀人十年起步
3. 父子二人意识到有人借苏文与陆明的矛盾栽赃嫁祸
4. 苏国栋拿出二十五根金条，安排苏文走乡道翻太行山逃往山西
5. 苏文跪地磕头告别，正要出门时警察破门抓人

**出场人物**：苏文、苏国栋

主题标签：栽赃陷害 | 基调：绝望、紧迫

## 第105章 案件进展很顺利

**概要**：苏文在审讯中被陈建平亲自问话，最终在所有笔录上签字认罪，供出医保套现利益链条和十几名在职干部。陆明出院后恢复工作，安排赵一舟以一块钱一斤收购全县一千万斤滞销蒜薹、一毛钱一斤卖给市民，花一千万买全县民心。省报林汐联系采访。

**关键事件**：
1. 苏文在审讯中全面认罪，交代医保套现利益链条和涉案干部名单
2. 陆鸢从郑州调来防弹库里南等三辆豪车，给陆明及核心团队配车
3. 方珩买了问界M9，与陆鸢关系升温
4. 孙长明亲自到陆明办公室通报案件进展，表示从严从快顶格处理
5. 赵一舟汇报蒜薹严重滞销，陆明决定一块钱收购、一毛钱销售

**出场人物**：苏文、陈建平、陆明、陆鸢、方珩、沈璃、孙长明、赵一舟

主题标签：案件定论、助农决策 | 基调：顺畅、豪迈

## 第106章 买个口碑吧

**概要**：万家福超市以一块钱一斤收购、一毛钱一斤出售蒜薹的助农行动火爆全县，农户排起两公里长龙。赵一舟同步开通全平台直播，线上订单蜂拥而至，万家福品牌名扬全国。方瑜调查肇事司机张伟的底细，发现他与苏文之间没有任何交集。

**关键事件**：
1. 万家福清晨五点半开始收购蒜薹，排起两公里长龙，当场现结
2. 超市蒜薹以0.1元/斤销售，市民疯狂抢购，乐购老板赵得财叹服
3. 赵一舟开通抖音、快手、视频号全平台直播助农，弹幕刷屏
4. 线上渠道同步开通，全国包邮，下单蜂拥而至
5. 方瑜查清肇事司机张伟与苏文无任何交集——无通话、转账、共同联系人

**出场人物**：陆明、赵一舟、方瑜、老李头、赵得财、沈璃

主题标签：助农热潮 | 基调：热闹、振奋

## 第107章 雷霆手段

**概要**：方瑜向陆明汇报张伟是白血病患者父亲，完美死士人选，但一口咬定受苏文指使，且陈建平对此视而不见。两人分析后震惊地推测幕后主使可能是孙长明——借泥头车事件给苏文扣杀人帽子，抢在医保局之前结案。方瑜表示要一查到底，陆明劝阻无果。

**关键事件**：
1. 方瑜揭示张伟是白血病患者父亲，是最理想的死士人选
2. 两人分析陈建平看出张伟撒谎却仍认定苏文是幕后主使，疑点重重
3. 陆明和方瑜推测幕后主使可能是孙长明，动机是借命案抢在医保局前结案
4. 陆明指出泥头车只撞一次就跑，说明不是真想杀人，只需一个由头
5. 方瑜表示无论牵扯到谁都会一查到底，情绪崩溃落泪

**出场人物**：陆明、方瑜

主题标签：真相浮出 | 基调：紧张、心惊、温情

## 第108章 定要你见识见识人心险恶

**概要**：陆明安排方珩加强所有核心人员的安保。万家福蒜薹助农持续火爆，省报记者林汐来电采访。乐购超市赵得财心生歪念，派员工从万家福低价买入蒜薹再高价卖回，企图薅羊毛牟利。

**关键事件**：
1. 陆明命令方珩给方瑜、陆鸢、沈璃及父母三叔全部加强安保
2. 方珩给方瑜发消息"批评过陆总了"，方瑜回复"？"
3. 万家福蒜薹销售持续火爆，赵一舟嗓子喊哑
4. 省报记者林汐主动联系沈璃要求专访，陆明同意并要求准备深加工产业园规划书
5. 乐购老板赵得财心生邪念，计划从万家福进货再卖回给万家福套利

**出场人物**：陆明、方珩、方瑜、赵一舟、沈璃、林汐、赵得财

主题标签：安保升级、暗流涌动 | 基调：忙碌、紧张

## 第109章 招笑

**概要**：省报记者林汐到访采访，陆明借机发表犀利言论，批评营商环境缺乏安全感和法治，暗指孙长明的手段。赵一舟在超市现场抓获乐购派来薅羊毛的员工，当众拆穿其套路。

**关键事件**：
1. 林汐到访，注意到安保等级大幅提升，陆明以"下雨天路滑"回应伤情
2. 陆明接受专访时批评营商环境缺乏安全感，提出"法治和规则才是底线"
3. 陆明表态"原话照发，一字不改"，借省报向孙长明喊话
4. 赵一舟在超市现场抓获乐购派来反复买卖蒜薹的员工
5. 当众拆穿其"闭环经济"套利行为

**出场人物**：陆明、林汐、沈璃、赵一舟、乐购员工

主题标签：舆论布局 | 基调：犀利、机敏

## 第110章 跟我一起助助农吧

**概要**：乐购员工供出赵得财指使，陆明当众给赵得财两个选择——报警或每天收购一万斤蒜薹连收十天。赵得财被迫接受。随后孙长明约陆明到县委大院，暗示泥头车事件是其秘书小郑所为，将调往乡镇。陆明当场向孙长明发出警告——身边人出事将平等报复每一个嫌疑人。

**关键事件**：
1. 乐购员工当众供认受赵得财指使，赵一舟当场电话录音取证
2. 陆明给赵得财两个选择，赵得财被迫答应每天收一万斤蒜薹连收十天
3. 林汐拟定标题《从竞争到共助》，三方各取所需
4. 孙长明约陆明到县委，暗示秘书小郑是泥头车事件操办者，将调往乡镇
5. 陆明当面警告孙长明：身边人出任何问题，都会怪罪每一个可能有关系的人

**出场人物**：陆明、赵一舟、赵得财、林汐、方珩、孙长明、小郑

主题标签：正面交锋 | 基调：霸气、暗涌
