# 第21章 摘要：就用这个设计团队

**概要**：鼎元空间设计公司创始人贺铭远带队勘察万家福购物广场，给出三层改造方案：一楼精品生鲜市集、二楼开放式品牌集合区、三楼体验区，工期三十五天、总成本近千万，陆明当场拍板签约。考察期间一名开白色路虎的中年男子到新城大厦找陆明未果，沈璃拍下车牌号，周岩看到照片后语气紧张。

**关键事件**：1.贺铭远团队勘察万家福并出具改造方案 2.陆明接受三十五天工期并预付设计费 3.神秘白色路虎车主到访

**出场人物**：

|------|----------|--------|
| 陆明 | 甲方决策者，拍板签约 | 无 |
| 贺铭远 | 设计方主创，提出改造方案 | 无 |
| 沈璃 | 接待、记录神秘来访者 | 无 |
| 陆鸢 | 旁听方案，记录数字 | 无 |

P1 **鼎元团队到场**：类型{信息揭示} | 贺铭远带两名年轻设计师从许昌抵达云梦县 | 涉及贺铭远 | 地点{新城大厦}

P3 **二楼隔断判定**：类型{信息揭示} | 贺铭远判定二楼玻璃隔断全部拆除 | 涉及贺铭远 | 地点{万家福二楼}

P7 **签约预付**：类型{行动} | 陆明承诺签合同当天付清设计费和监理费，贺铭远五天内出正式方案 | 涉及陆明、贺铭远 | 地点{新城大厦会议室}
P8 **神秘来客**：类型{铺垫} | 一名四十来岁穿黑色夹克男子开白色路虎到新城大厦找陆明未遇，拍照后离开 | 涉及沈璃 | 地点{新城大厦广场}
P9 **车牌照片转发**：类型{行动} | 陆明将沈璃拍摄的路虎车牌照片转发给周岩询问 | 涉及陆明、周岩 | 地点{新城大厦}
P10 **周岩即时回电**：类型{转折点} | 周岩看到照片后直接打电话，语气紧张问"这人怎么会找上你" | 涉及周岩 | 地点{电话}
