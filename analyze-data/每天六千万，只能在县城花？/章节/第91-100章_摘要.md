# 第91-100章 摘要

## 第91章 书记不高兴了

**概要**：孙长明得知央视再次曝光云梦县，震怒之下召集卫健委、市场监管局、医保局三部门负责人紧急开会。孙长明逐一点名质问，揭露济世大药房"刷一百返八十"全县皆知唯独监管部门不知的荒谬。最终追问出济世大药房实际控制人正是苏文。

**关键事件**：
1. 孙长明一个月内两度遭央视曝光，深感政治危机逼近
2. 紧急召见刘德厚、郑战勇、陈可为三人，会议室无茶无纸无议程，气氛肃杀
3. 孙长明以"违规不违规"连续追问郑战勇，步步紧逼不留喘息
4. 孙长明讽刺"全县老百姓都知道的事，就你们市场监管局不知道"
5. 郑战勇被迫承认济世大药房实际控制人为苏文

**出场人物**：孙长明、小郑、刘德厚、郑战勇、陈可为

主题标签：官场问责、危机处理 | 基调：高压震慑、如坐针毡

## 第92章 又是好事？

**概要**：孙长明部署三部门封存档案、全力配合国家医保局检查组，随后亲赴云梦宾馆拜访检查组组长高建平表态"绝不姑息"。孙长明电话试探苏国栋，苏国栋果断表态"绝不包庇"。陆明判断不宜介入苏文之事，但暗中安排方瑜调查济世大药房股权结构准备接盘。

**关键事件**：
1. 孙长明命令三部门封存所有档案，全力配合检查组工作
2. 孙长明亲赴云梦宾馆向检查组长高建平表态全力配合
3. 苏国栋接到孙长明试探电话，表态"绝不包庇，该抓抓，该判判"，随后自扇耳光
4. 沈璃建议趁机落井下石，陆明否决，判断"不该掺和"
5. 陆明安排方瑜调查济世大药房股权结构，意图在苏文倒台后接盘开设正规药房

**出场人物**：孙长明、小郑、高建平、苏国栋、陆明、方瑜、沈璃、郑战勇、陈可为

主题标签：政治表态、冷静旁观 | 基调：紧张肃杀、暗藏算计

## 第93章 就等今天

**概要**：苏文对医保风暴非但不惧，反而认为是"好事"。他向父亲分析：央视一个月两次曝光云梦县并非巧合，而是背后大佬在斗法，矛头指向孙长明的靠山。苏文提出联合被清洗的官员实名举报孙长明，并抛出2021年矿难的杀手锏——当年矿难死二十人，孙长明是始作俑者却借机上位。

**关键事件**：
1. 苏文判断央视两次曝光是"神仙打架"，孙长明背后的大佬已落下风
2. 苏文分析孙长明被"发配"云梦县说明其并非核心心腹，并非不可动摇
3. 苏文提出串联郑战勇、刘德厚等人联名举报孙长明
4. 苏文揭露2021年矿难真相：孙长明违规批矿致20死13伤，却借机上位
5. 苏文透露已与当年矿难受伤者保持联系多年，"就等今天"

**出场人物**：苏文、苏国栋

主题标签：政治阴谋、绝地反击 | 基调：阴暗诡谲、步步惊心

## 第94章 三方博弈

**概要**：孙长明陷入两难——装死配合检查则自己政治生命终结，主动清洗则暴露更多问题。陆明在万家福视察，超市生意红火，外地顾客慕名而来。苏国栋一夜间苍老十岁，经深思熟虑后决定找陆明帮忙调停，试图以苏文的举报计划为筹码换取生机。

**关键事件**：
1. 孙长明深夜进行沙盘推演，面临"不清洗等死、清洗找死"的两难困境
2. 万家福单日客流突破两万八，"不满意无条件退款"政策在短视频平台走红
3. 苏国栋意识到陆明是唯一能调停各方势力的人，决定登门求助
4. 陆明目睹超市员工主动为顾客退换草莓并补偿，印证服务口碑正在发酵

**出场人物**：孙长明、陆明、赵一舟、沈璃、苏国栋

主题标签：三方角力、商业腾飞 | 基调：山雨欲来、各有盘算

## 第95章 晚了！

**概要**：苏国栋拜访陆明，先以教育系统开绿灯示好，再透露苏文串联官员举报孙长明的计划作为筹码，最后威胁若孙长明倒台陆明的投资也将失去庇护。陆明不为所动，反揭苏国栋2012年纵容苏文顶替中考学生王坤成绩的旧事，怒斥苏家桩桩件件的龌龊勾当，厉声宣布"想当革命家？晚了！"

**关键事件**：
1. 苏国栋以教育系统全部绿灯为筹码，请求陆明向孙长明求情
2. 苏国栋亮出底牌：苏文准备联名举报孙长明，握有矿难审批文件和伤残矿工联名信
3. 苏国栋威胁"孙长明倒了，你的投资也完了"
4. 陆明霸气回应"谁来当县委书记，面对这个烂摊子都会给我开绿灯"
5. 陆明揭穿苏家当年顶替学生王坤中考成绩的恶行，怒斥苏国栋"亲手毁了一个人的一生"

**出场人物**：陆明、苏国栋

主题标签：摊牌对决、正义审判 | 基调：刚猛凌厉、酣畅淋漓

## 第96章 不站队

**概要**：陆明拒绝站队任何一方，既不帮苏文也不为孙长明通风报信。但他立刻意识到苏文狗急跳墙可能对身边人下手，紧急安排方珩召集退伍战友组建安保团队，保护父母、三叔、陆鸢、沈璃、方瑜等核心人员。方珩推荐班长陈凯，陆明安排其全家来云梦县并解决后顾之忧。

**关键事件**：
1. 陆明定下"不帮、不传、不站队"的原则，两头都不碰
2. 陆明预判苏文可能对身边人下手，紧急组建安保团队
3. 方珩联系退伍战友，月薪八千起，二十人分三组全天候轮班
4. 方珩推荐八级伤残班长陈凯，陆明承诺安排住房、护工和孩子入学
5. 方珩连夜拨出七个电话，第一批八人当天出发

**出场人物**：陆明、方珩、陈凯（电话中）

主题标签：安保部署、兄弟情义 | 基调：沉着应对、情义深厚

## 第97章 倒查十年

**概要**：十六名退伍安保人员到位，陈凯一家获妥善安置，陆明安保体系正式建立。温泉小镇工地热火朝天，三百多工人施工进展顺利。陆明以翻倍日薪大规模招工，引发返乡潮。国家医保局检查组查明济世大药房仅去年一年套取医保基金高达八百万，组长高建平下令"倒查十年"。

**关键事件**：
1. 十六名退伍安保到位，陈凯获赠翡翠城住房和母亲护工、儿子学籍安排
2. 温泉小镇施工全面展开，三百多工人进场，管网开挖进度超前
3. 陆明以壮工日薪四百、技工六百的翻倍薪资大规模招工，三天报名超一千二百人
4. 国家医保局查明济世大药房去年套取医保基金八百万，涉及医院虚开处方、医保局内部人员违规开绿灯
5. 高建平下令"倒查十年"，直指完整利益链

**出场人物**：陆明、方珩、陈凯、秦业、沈璃、高建平、苏文、赵海

主题标签：返乡潮、反腐深入 | 基调：热火朝天与暗流涌动并存

## 第98章 我是闲杂人等？

**概要**：倒查十年的消息传开，方瑜从姑父处得知医院院长和医保局人员已被带走问话。陆明立刻将安保等级拉满，命令陈凯团队对核心人员住所实施二十四小时无死角盯防。方娟满怀期待去陆明家说媒，却被门口安保人员拦下，被告知"闲杂人等不可靠近"。

**关键事件**：
1. 方瑜紧急报告倒查十年定调，县医院院长、副院长、医保局结算科负责人已被带走
2. 陆明命令安保等级拉满，所有核心人员住所二十四小时无死角盯防
3. 陆明警告方瑜"这几天不要乱跑"，方瑜感受到陆明的冷静与狠辣
4. 陆明视察温泉小镇工地，目睹返乡工人干劲十足的景象
5. 方娟带礼品上门说媒，被安保人员以"闲杂人等"拒绝靠近

**出场人物**：陆明、方瑜、方珩、秦业、方娟、陈凯、郑强、刘彪

主题标签：全面戒备、乌龙说媒 | 基调：紧张备战与喜剧插曲交织

## 第99章 拉几个垫背的

**概要**：苏国栋拿出二十五根金条让苏文逃往美国，苏文拒绝。苏文三天内拨出十四通电话试图串联十九名关联官员，却遭到全面拒绝——七人关机、五人不接、两人接了挂断。白崇文更是暗示手机可能被监控劝苏文别拖人下水。苏国栋戳穿白崇文的谎言，感叹儿子太嫩。苏文最终表示走之前要拉几个垫背的。

**关键事件**：
1. 苏国栋取出二十五根金条（约一千多万）让苏文逃往美国，苏文拒绝
2. 苏文整理出十九名与苏家有利益往来的在职干部名单
3. 苏文拨打十四通电话试图串联，全部碰壁：七人关机、五人不接、两人挂断
4. 白崇文谎称手机被监控劝苏文别联系，苏国栋一眼看穿此为敷衍
5. 苏国栋痛斥苏文"你什么时候才能长大"，苏文仍执意"走之前拉几个垫背的"

**出场人物**：苏国栋、苏文、郑战勇、刘德厚、陈可为、钱志刚、白崇文

主题标签：树倒猢狲散、穷途末路 | 基调：凄凉绝望、人心凉薄

## 第100章 无情冷雨夜

**概要**：陆明预判苏文可能对自己下手，调来特种兵出身的郑强和刘彪贴身护卫。暴雨之夜，陆明下班回家途中，一辆无牌重型泥头车从施工便道蓄意撞击迈巴赫。方珩极限漂移避开致命撞击，但迈巴赫仍被撞飞旋转。陆明在剧烈碰撞中因高血压爆发、脑部缺氧而彻底昏迷。

**关键事件**：
1. 陆明预判苏文的行动目标是自己，调来侦察连退役的郑强（近身格斗）和刘彪（特种驾驶）
2. 暴雨夜陆明下班回家，迈巴赫行驶在建设路
3. 郑强发现后方有捷达盯梢，紧接着无牌泥头车从盲道疯狂冲出
4. 方珩极限漂移改变受力角度，避免直接侧撞，刘彪死死护住陆明
5. 迈巴赫被撞飞旋转两圈半撞上隔离墩，陆明因高血压爆发陷入昏迷

**出场人物**：陆明、方珩、郑强、刘彪、陆鸢、沈璃

主题标签：蓄意谋杀、生死一线 | 基调：惊心动魄、暴雨腥风
