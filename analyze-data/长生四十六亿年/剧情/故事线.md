# 故事线总览

## 主线

**主线标题**：苏念直播曝光哥哥苏长青跨越多个时代的身份

**核心矛盾**：苏长青想隐居钓鱼摆烂 vs 妹妹苏念直播曝光他不老和跨时代身份的全部秘密

**终极目标**：让苏长青藏在身后的全部历史身份（从清代青王到民国苏州王/师座、长征红军班长、抗鹰少校团长、六十年代红星厂总工、退休钓鱼佬）依次被公开，让全网在一夜间见证「这盛世如他所愿」

**包含剧情条**：
1. 直播开盘+第一桶古玩曝光（剧情1）
2. 六十年代工业脊梁线（剧情2）
3. 抗鹰少校+长津湖战友线（剧情3）
4. 红军班长+草地救命线（剧情4）
5. 淞沪师座+428封绝笔信线（剧情5）
6. 民国苏州王+徐家百年家奴线（剧情6）
7. 民国嫂嫂林婉清+爱情BE线（剧情7）
8. 苏长青回家+祭奠+家法+长生主题升华（剧情8）
9. 清代青王身份揭露（剧情9，连载未完）

## 交织方式

- **主线 = 多段身份曝光循环**：苏念在密室里翻出一个物件 → 弹幕/连线专家鉴定 → 当代证人（老兵/泰斗/巨富/养女）登场作证 → 苏长青身份升级一档 → 全网震撼 → 新物件再翻出。
- **多副线并行注入**：每一身份曝光循环嵌入 [直播事业]+[顶级权贵反应]+[反水军黑产被惩处] 三种副线节拍。
- **CP线作为情绪压舱石**：在第60-65章一次性引爆，跨越百年的BE把读者从「爽点」拉到「泪点」，再用「林慕青姑姑」笑梗反差缓解。
- **苏长青反向出场**：前 67 章他基本不在场（只在江边钓鱼+读者透过照片/文物/信件认识他），第 68 章无人机才接入实时画面，他靠"反差感"在登场瞬间获得最大爆点。
- **海外网友进场**（第90章）打破了「国内全网拥护」的舒适区，把民族情感冲突纳入收尾爆点，为下一卷做铺垫。
