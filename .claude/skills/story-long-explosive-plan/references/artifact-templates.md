## 设定/番茄爆款档案.md（核心）

**用途**：本 skill 专属档案，记录极端化算子选型 + 八元配置 + 代入感锚 + 番茄风承诺。

**何时创建**：Phase 2 极端化注入完成后。

**模板**：

```markdown
# 番茄爆款档案

## 目标平台
- 平台：番茄小说
- 目标在读量：{如：100 万}
- 目标性别向：{男频 / 女频}
- 短剧改编潜质：{高 / 中 / 低}
- 期望付费节点：{第 80 章 / 第 100 章}

## 八元配置

### 选定元素
- 主元素：{元 X：名称}
- 副元素 1：{元 Y：名称}（可选）
- 副元素 2：{元 Z：名称}（可选）

### 元素叠加策略
- 叠加组合：{如"元 1 主角发疯 + 元 2 系统反规律"}
- 主导情绪内核：{如"反内耗 + 高密度爽点"}
- 化学反应：{两个元素如何协同}

## 选定算子（按元素分别填写）

### 元 1 主角维度（如选）
- 选定式：{如：第 1.1 式 发疯文学型 / 第 1.2 式 假痴不癫型 / ...}
- 核心人格：{30 字内}
- 经典语录：{准备用的金句 3 条}
- 与代入感的协同：{如何不变无脑}

### 元 2 系统/金手指维度（如选）
- 选定法：{如：2.1 系统人格化 / 2.2 反噬式 / 2.3 反规律 / ...}
- 系统名：{如"虐渣系统"}
- 系统规则：{核心规则 3-5 条}
- 系统人格（如人格化）：{毒舌/萌/傲娇/...}

### 元 3 世界观维度（如选）
- 选定法：{如：3.1 类型错位 / 3.2 规则反常 / ...}
- 反差核心：{一句话概括}
- 世界观规则：{核心规则 5 条以内}
- 主角的"局外人"身份：{}

### 元 4 配角维度（如选）
- 选定法：{如：4.1 师父癫 / 4.2 萌宠癫 / 4.3 反派出彩 / 4.4 队友奇葩}
- 关键配角：{角色名 + 极端特征}

### 元 5 剧情反差维度（如选）
- 选定法：{如：5.1 身份反转 / 5.2 反向操作 / ...}
- 反差核心：{读者以为 A 实际 B}
- 揭露时机：{第 X 章}

### 元 6 题材融合维度（如选）
- 主题材：{}
- 副题材 1：{}
- 副题材 2：{}（如有）
- 力量体系：{统一为哪个}

### 元 7 设定脑洞维度（如选）
- 选定法：{如：7.1 规则文 / 7.2 考据流 / ...}
- 核心脑洞：{}
- 可视化方式：{}

### 元 8 暧昧氛围维度（如选）
- 选定法（可多选）：{如：8.1 环境暗示法 / 8.2 动作留白法 / 8.3 语气试探法 / 8.4 眼神交汇法 / 8.5 信息差暧昧法 / 8.6 距离张力法}
- 暧昧路径定位：{纯氛围驱动 / 极端化+暧昧混合 / 暧昧 + 题材融合}
- 核心 CP：{角色 A × 角色 B}
- 暧昧推进节奏：
  - 第 1-30 章：{初识/试探阶段}
  - 第 31-80 章：{加深/拉扯阶段}
  - 第 81-150 章：{挑明/对抗阶段}
  - 第 150 章+：{蜜月/新挑战阶段}
- 暧昧的"代价"：{}（必填——心动后的代价：被发现/被利用/被伤害等）
- 与主线关系：{爱情线 vs 事业线的权重，男频 30% vs 女频 50%）
- 与极端化的协同：{如选了极端主角，暧昧如何不弱化主角强势}

## 代入感锚（必填，详见 immersion-anchor.md）

- 主角普世痛感：{失业/被背叛/失去亲人/底层挣扎 - 选具体场景}
- 主角软肋：{亲人/挚友/底线/秘密/PTSD/信念/身份 - 选 1 项 + 具体描述}
- 主角理由链：
  - 童年：{}
  - 事件：{}

  - 行为模式：{}
- 道德爽感锚：{保护弱者/孝顺/言出必行/知恩图报 - 选 1 项 + 具体表现}
- 致命弱点：{必须能真实伤害到主角 - 具体描述}

## 反派威胁配置

- 第一反派（小反派，1-5 章）：{名字 + 1-2 个特征 + 出场章}
- 第二反派（中反派，10-30 章）：{名字 + 完整动机 + 至少赢主角 1 次的方式}

- 终极反派（全书）：{名字 + 与主角的核心矛盾}

## 番茄风承诺（前 3 章必兑现）

- 第 1 章金手指激活：{具体方式}
- 第 1 章末尾钩子：{具体内容}
- 第 2-3 章打脸点：{具体场景 + 反派}
- 第 10 章前的大爆点：{具体场景}

## 题材定位与对标书（如有）

- 主题材：{}
- 子题材：{}
- 对标书：{书名}（如有）
  - 对标书的情感弧线：{摘要}
  - 对标书的伏笔系统：{摘要}
  - 对标书的节奏控制：{摘要}

## 修订记录

- {日期} v1.0：初始版本
```
