## 死亡开局 7 公式

### 公式 1：肉体危机式

**结构**：主角正面临身体伤害 → 反派施暴 → 主角反应

**开篇要素**：具体的暴力动作 + 主角的具体反应 + 反派身份暗示

**适用题材**：玄幻/都市/重生/穿越

**模板**：
```
[动作描写]"嘭！"[动词]
[反派身份]+[暴力动作]
[主角具体反应]——[受伤具体细节]
[主角内心一句话，体现极端人设]
```

**案例**：
> "嘭！"
> 父亲一脚踢翻了沈栀手中的早餐。
> 滚烫的稀饭洒在地上，碎瓷片溅起来划过她的小腿，血珠渗出来。
> 沈栀面无表情地捡起一片瓷片，慢慢攥紧。
> "你再敢碰我妈试试？"

### 公式 2：信息冲击式

**结构**：一句"价值百万"的信息直接砸读者脸上 → 主角反应 → 暗示后续爆点

**开篇要素**：颠覆性的信息 + 主角的具体反应

**适用题材**：重生/穿越/系统流/反转流

**模板**：
```
[颠覆性信息一句话]
[主角的具体身体反应]
[这个信息为什么是颠覆性的]
[主角的决定/承诺]
```

**案例**：
> "你不是我的亲妹妹。"
> 沈栀手中的咖啡杯啪地砸在地上。
> 二十年的"家人"，二十年的偏心，二十年的委屈——
> 原来都是假的。
> 沈栀慢慢笑了："那好，从今天起，我也不当她的姐姐了。"

### 公式 3：场景反转式

**结构**：开篇看起来是 A 场景 → 一句话反转成 B 场景 → 主角的判断

**开篇要素**：误导性场景 + 反转一句话 + 主角反应

**适用题材**：悬疑/反转/穿书/重生

**模板**：
```
[误导性场景描写一两句]
[一句话反转]
[主角恍然/震惊/笑了]
[主角的真实判断/动作]
```

**案例**：
> 婚礼现场，新郎正在朗读誓词，新娘含泪点头。
> ——这是沈栀第三次看着同一个画面。
> 第一次她是新娘。第二次她是被前夫推下楼的鬼魂。
> 第三次她坐在亲戚席上，慢悠悠剥着橘子。
> 她对身边的妹妹笑了笑："放心，这次我不抢你的男人。"

### 公式 4：金手指激活式

**结构**：极端处境 → 金手指激活 → 主角的反应 → 第一次施展

**开篇要素**：危急处境 + 金手指激活时刻 + 视觉化效果

**适用题材**：系统流/重生/穿越

**模板**：
```
[主角的极端处境一两句]
[金手指激活的具体时刻 + 视觉效果]
[系统/能力的第一句话]
[主角的反应——惊喜/狡黠/狂笑]
```

**案例**：
> 沈栀蜷缩在垃圾堆里，七天没吃东西了。
> 就在她准备闭眼时，脑海中一道金光炸开。
> 【虐渣系统已激活。每打脸一次原配亲戚，账户入账 10 万。】
> 沈栀慢慢睁开眼，嘴角勾起。
> 她饿了七天的胃，突然不饿了。

### 公式 5：身份反转式

**结构**：表面身份 → 关键信息揭示真实身份 → 视角切换

**开篇要素**：误导性身份 + 真实身份 + 信息差形成

**适用题材**：身份反转/穿书/扮猪吃虎

**模板**：
```
[误导性身份场景]
[一句话揭示真实身份]
[视角切换或主角的真实想法]
[暗示后续布局]
```

**案例**：
> "废物大小姐，又来求收留？"
> 守卫推开了沈栀。
> 沈栀踉跄着倒地，狼狈不堪。
> 没人知道，她袖子里藏着的，是足以让整个沈家家破人亡的证据。
> 她低头掩饰嘴角的冷笑："我只是想见父亲一面。"

### 公式 6：道德困境式

**结构**：主角面临两难选择 → 选择的瞬间 → 极端决定

**开篇要素**：具体的两难场景 + 主角的选择 + 选择的代价

**适用题材**：现实/悬疑/谋略

**模板**：
```
[具体的两难场景]
["A 还是 B" 的明确二选一]
[主角的选择——快速、果断]
[选择的代价/后果暗示]
```

**案例**：
> 火光中，两扇门，两个人。
> 左边是她的儿子。右边是她的妹妹。
> 沈栀只能救一个。
> 她毫不犹豫地推开了——
> 右边那扇门。
> 因为她知道，妹妹手里有她儿子的解药。

### 公式 7：高潮预演式

**结构**：直接从故事高潮场景切入 → 暗示这是结局 → 回溯起点

**开篇要素**：视觉化高潮场景 + "这是怎么开始的" + 主角的回望

**适用题材**：长篇宏大叙事/谋略/玄幻

**模板**：
```
[视觉化高潮场景]
[主角的某个标志性动作]
[一句话"这一切是怎么开始的"]
[切回故事起点]
```

**案例**：
> 沈栀站在沈家祖坟前，看着那十二座新立的墓碑。
> 她笑得很轻松。
> 这一切是怎么开始的呢？
> 三年前的那个夜晚——
> 沈栀被锁在地下室里，听着楼上传来的笑声。

## 前 200 字硬性要求

无论用哪个公式，前 200 字必须满足：

### ⭐ 必有 5 项

1. **具体冲突/危机/反转**（不是抽象描述）
2. **主角的核心特质亮相**（性格关键词通过一个动作/一句话体现）
3. **某种金手指/能力/优势暗示**（不必完整展示，给出预告）
4. **明确的情绪基调**（愤怒/决绝/狡黠/狂喜——选 1）
5. **主角的"承诺"**（一句话告诉读者"我接下来要做什么"）

### ❌ 必无 5 项

1. **环境描写**（天气/风景/季节）
2. **背景介绍**（年代/地点/时代）
3. **主角简历**（年龄/职业/经历）
4. **抽象心理活动**（"心想..."不带具体场景）
5. **配角先出场**（前 200 字必须主角主导）

### 字数分配

- 200 字内必须包含：1 个动作场景 + 1-3 句对话 + 1 个主角内心承诺
- 200 字之后到 500 字：补充信息（金手指/背景/反派）
- 500 字到 1000 字：第一个小冲突/小爽点
- 1000 字到 2000 字：第一次施展能力 + 章末钩子

### 章末 200 字要求

第 1 章末尾 200 字必须有：
- 大悬念钩子（让读者必须看第 2 章）
- 暗示第 2 章的具体动作（不是抽象"接下来..."）
- 至少 1 个新人物/新事件信息释放

## 死亡开局核心范例 10 个

> **说明**：本节提供 10 个核心范例（男频 5 + 女频 5），作者按需扩展。
> 实际写作时可以根据"死亡开局 7 公式"自行组合更多变体。

### 男频范例（5 个）

#### 范例 1（玄幻/重生）
> "嘭！"
> 萧昊被弟弟一脚踢出了萧家祠堂。
> 七岁的萧云笑着冲他做鬼脸："废物哥哥，你已经不是萧家的人了。"
> 萧昊从地上爬起来，吐出一颗血淋淋的牙齿。
> 这是他第二次活到七岁。
> 上辈子的他被弟弟踢出家门后流落街头死了。
> 这辈子——他要让整个萧家给他陪葬。

#### 范例 2（都市/系统）
> 林峰被女朋友的奔驰车撞飞时，脑海中一道金光炸开。
> 【虐渣系统已激活。每被前任女友伤害一次，奖励 100 万。】
> 林峰躺在血泊里，慢慢笑了。
> 远处，前女友的尖叫声传来：
> "天啊！我撞死人了！"
> 系统提示音再响：【账户入账 100 万。】
> 林峰：太好了，再撞我一次。

#### 范例 3（玄幻/扮猪吃虎）
> 武道大会上。
> "废物，跪下！"
> 大师兄一脚踩在江尘的脸上。
> 江尘趴在地上，看着头顶的天空，露出一个浅浅的微笑。
> 他想起三天前父亲临终的嘱托："尘儿，记得忍。"
> 江尘忍了。
> 但他没说要忍到什么时候。
> 现在嘛——
> 他的右手，悄悄按上了腰间的剑柄。

#### 范例 4（都市/赘婿）
> 苏家家宴，赘婿陈凡被罚跪在祠堂前。
> "三天没吃饭，跪到天明，让你知道什么是规矩。"
> 老丈人抽了陈凡一耳光。
> 陈凡擦掉嘴角的血，慢慢站起来。
> "够了。"
> 老丈人勃然大怒："你说什么？！"
> 陈凡笑了："我说，够了。"
> 他抬手，按下了手机里早已编辑好的一条短信。
> 三秒后，整个苏家颤抖了——
> 苏家所有的银行账户，被冻结了。

#### 范例 5（玄幻/无敌流）
> 飞云宗大殿。
> "处死！"宗主一声令下。
> 千名弟子抽剑，齐刷刷指向叶辰。
> 叶辰一身白衣，站在大殿中央，连眼睛都没眨。
> "处死我？"
> 他笑了。
> "可以试试。"
> 千名弟子动了。
> 下一秒——
> 大殿外天空炸裂，一头万丈巨龙俯冲而下。
> 叶辰头都没抬："小事。"

## 短剧化梗点设计

番茄短剧改编的核心是「可剪辑性」。爆款短剧的素材必须能拆解为 30-60 秒的片段。

### 可视化冲突场景清单

| 场景类型 | 男频版本 | 女频版本 |
|---------|---------|---------|
| 当众打脸 | 武道大会单挑/装逼 | 婚礼现场摊牌 |
| 手撕绿茶 | 撕开伪君子面具 | 撕开白莲花伪装 |
| 当街救美 | 救美主线 | 救男主线 |
| 一句封神 | "我服了"/"卧槽" | "姐姐好厉害"/"我的天" |
| 视觉冲击 | 金光/异象/天罚 | 满堂跪/全场鸦雀无声 |
| 反转瞬间 | 暴露身份 | 摘掉伪装 |
| 装逼时刻 | 报家门震慑全场 | 亮身份让所有人跪 |
| 救命瞬间 | 出手救人 | 救命扭转 |

### 梗点设计 7 原则

1. **可短视频化**：每个梗点能独立成 30-60 秒短片
2. **视觉冲击**：必须有可视化的画面（不只是对话）
3. **金句驱动**：每个梗点必须有 1 个可传播金句
4. **情绪锚点**：让读者会心一笑/拍案叫好/瞳孔地震
5. **二创友好**：能被读者剪辑/二次创作
6. **平台适配**：适合抖音/快手/小红书的传播
7. **梗点密度**：每 5-10 章至少 1 个可改编梗点

### 经典梗点模板

#### 模板 1：当众打脸梗

```
[反派嚣张挑衅]→[主角看似弱势]→[主角一句话/一个动作]→[反派表情瞬间崩塌]→[围观者震惊]→[主角淡定离开]
```

#### 模板 2：手撕绿茶梗

```
[绿茶演戏假惺惺]→[主角不动声色]→[绿茶以为得逞]→[主角拿出证据/直接打断]→[绿茶崩溃]→[围观者真相大白]
```

#### 模板 3：身份揭露梗

```
[反派欺压]→[反派暴露真实嘴脸]→[主角露出杀机]→[主角亮明身份]→[反派吓尿]→[主角动手]
```

#### 模板 4：金句反击梗

```
[反派一句嚣张话]→[主角神色不变]→[主角回一句金句]→[反派愣住]→[围观者:"卧槽"]→[主角继续走自己的路]
```

## 番茄风章首钩子 7 式

### 1. 动作开局式

直接进入动作场景，无铺垫无介绍。

**模板**：
```
"[拟声词]！"
[具体动作描写一句]
[主角反应一句]
```

**例**：
> "啪！"
> 一巴掌打在沈栀脸上。
> 沈栀慢慢转过头，露出了三年来第一个笑容。

### 2. 对话开局式

用一句具有冲击力的对话开场。

**模板**：
```
"[震撼的一句话]"
[谁说的 - 具体身份]
[听者的反应]
```

**例**：
> "你不是我亲生的。"
> 母亲突然开口，手中的茶杯停在半空。
> 沈栀手中的勺子掉在桌上。

### 3. 信息开局式

抛出一个颠覆性信息直接砸读者脸上。

**模板**：
```
[颠覆性信息一句]
[主角的具体反应]
[这意味着什么]
```

**例**：
> 父亲死了。
> 沈栀放下手中的电话，没有哭。
> 她笑了——因为她终于可以拿到那一千亿的遗产了。

### 4. 危机开局式

主角正面临具体危机（生命/财产/名誉/感情）。

**模板**：
```
[具体危机场景]
[主角的位置 + 状态]
[危机的迫近]
```

**例**：
> 火光冲天，沈栀被困在地下室。
> 头顶是凶手的脚步声。
> 还有 5 分钟，她就要被烧死。

### 5. 反转开局式

开篇看起来是 A 实际是 B。

**模板**：
```
[误导性 A 场景]
[一句话反转]
[主角的真实反应]
```

（参考公式 3 场景反转式）

### 6. 视角开局式

用非常规视角（反派/物品/动物）切入。

**模板**：
```
[非常规视角的描述]
[这个视角的特殊性]
[与主角的关系]
```

**例**：
> 我是一把剑。
> 一把被沈栀握在手里的剑。
> 三百年了，我看过沈栀爱过的所有人都死在我手里。
> 今天，第十二个。

### 7. 倒计时开局式

明确的时间限制立刻形成紧迫感。

**模板**：
```
[倒计时单位]
[即将发生的事]
[主角的处境]
```

**例**：
> 还有 24 小时，沈栀就要嫁给那个杀她全家的人。
> 而她，刚刚收到了来自前世的死亡通知书。

## 番茄风章尾钩子 13 式

### 1. 断句钩

一句话拆成两章。本章末尾是上半句，下一章开头是下半句。

**例**：
- 章末："沈栀冷笑：'你以为我重生只是为了报仇？'"
- 下章首："'我重生是为了——'"

### 2. 信息钩

章末释放一个颠覆性新信息。

**例**：
> 沈栀握着那张 DNA 报告，慢慢合上眼。
> 原来——她不是沈家的女儿。
> 她，是萧家的失踪二十年的大小姐。

### 3. 危机钩

章末出现新的危机/威胁。

**例**：
> 门被踹开。
> 沈栀转头，看见门口站着的男人。
> 那是她"死了三年"的前夫。

### 4. 身份钩

章末暗示某人的真实身份。

**例**：
> 男人摘下面具，露出了那张沈栀曾经无比熟悉的脸。
> "好久不见，妹妹。"

### 5. 时间钩

章末用倒计时创造紧迫感。

**例**：
> 沈栀看着手机上的倒计时：还有 24 小时。
> 24 小时后，整个沈家——
> 将不复存在。

### 6. 承诺钩

章末主角立下狠话。

**例**：
> 沈栀站在沈家门前，露出了三年来第一个微笑。
> "沈家，我回来了。"
> "从今天起，你们每个人——都要为我付出代价。"

### 7. 反转钩

章末发生彻底颠覆前面情节的反转。

**例**：
> 沈栀以为自己赢了。
> 直到她推开男主的办公室门——
> 看见里面坐着的，是她以为已经死了的母亲。

### 8. 悬念钩

章末抛出一个无法解释的现象。

**例**：
> 沈栀醒来时，发现自己躺在一具尸体旁边。
> 那具尸体的脸——
> 和她长得一模一样。

### 9. 道具钩

章末出现一个神秘道具。

**例**：
> 沈栀的手机突然震动。
> 屏幕上显示一条短信：
> "你下一步要做什么，我都知道。"
> 发件人显示——沈栀自己。

### 10. 视角钩

章末切换到另一个视角，看主角不知道的事。

**例**：
> 沈栀离开后。
> 男主从暗处走出来，捡起沈栀掉落的钢笔。
> 他笑了笑："小姑娘，你以为你今天赢了？"
> "其实——是我让你赢的。"

### 11. 误会钩

章末出现读者知道但主角不知道的真相。

**例**：
> 沈栀以为自己救下了陌生人。
> 她不知道——
> 她救下的男人，正是她下一个要复仇的目标。

### 12. 决定钩

章末主角做出一个改变命运的决定。

**例**：
> 沈栀放下手中的合同，闭上眼睛。
> 三秒后，她睁开眼，眼神冰冷。
> "好。"
> "我答应你的条件——嫁给贺廷渊。"

### 13. 神秘人钩

章末出现一个神秘新人物。

**例**：
> 沈栀走出沈家大门。
> 路边停着一辆黑色轿车。
> 后座的窗户摇下来，露出一张让沈栀瞳孔地震的脸。
> 那是——前世临死前看到的最后一个人。

## 前 3 章打脸节奏图

### 第 1 章节奏（标准 2500-3000 字）

```
0-200 字：死亡开局（用 7 公式之一）
200-500 字：补充主角背景 + 金手指/能力暗示
500-1000 字：第一次小冲突 + 主角性格亮相
1000-1500 字：第一个小打脸（让反派感到不对劲）
1500-2000 字：主角承诺 + 暗示后续大爆点
2000-2500 字：第二个小冲突 + 信息释放
2500-3000 字：章末大钩子（用钩子 13 式之一）
```

### 第 2 章节奏（标准 2500-3000 字）

```
0-200 字：承接第 1 章悬念 + 新场景
200-800 字：金手指/能力正式激活
800-1500 字：第一次大打脸（反派被狠狠教训）
1500-2200 字：装逼时刻（围观者反应）
2200-2700 字：新信息释放 + 新反派暗示
2700-3000 字：章末大钩子
```

### 第 3 章节奏（标准 2500-3000 字）

```
0-200 字：承接第 2 章悬念
200-1000 字：第一个大爆点（让读者瞳孔地震）
1000-1800 字：情绪锚点钉死（让读者知道未来能爽什么）
1800-2500 字：暗示第一个副本 + 中反派登场
2500-3000 字：章末大钩子（吸引读者追第 4 章）
```

### 关键节点

- **第 1 章 200 字内**：死亡开局
- **第 1 章 1000 字内**：第一次小打脸
- **第 2 章 1500 字内**：第一次大打脸
- **第 3 章 1500 字内**：第一个大爆点
- **第 3 章末尾**：情绪锚点钉死（让读者完整知道未来能爽什么）

## 第 10 章新副本设计

### 副本切换的意义

番茄读者在第 10 章前需要看到「副本切换」信号——告诉读者"故事在进展"，不是原地踏步。

### 副本切换 3 模式

#### 模式 1：地图切换
主角从 A 场景换到 B 场景（学校→职场、家庭→外面世界）

#### 模式 2：对手升级
小反派被解决，中反派出场

#### 模式 3：目标升级
完成阶段性目标，开启新目标

### 副本闭环的标志

第 10 章前需要完成至少 1 个完整的"副本闭环"：

```
主角立目标 → 主角遇阻 → 主角突破 → 主角达成 → 收获 + 升级
```

### 第 10 章必备元素

- [ ] 副本切换信号（地图/对手/目标至少一项）
- [ ] 大爆点（章节末有视觉冲击场面）
- [ ] 主角的明显成长（实力/资源/认知）
- [ ] 下一阶段的悬念预告
- [ ] 一个关键人物的引入或退出

### 第 10 章范例（结构）

```
0-500 字：本章节奏开始 + 上一副本残局收尾
500-1500 字：副本完美收尾（最后一波打脸）
1500-2500 字：收获时刻（升级感最强烈的瞬间）
2500-3500 字：新副本预告 + 新反派出场
3500-4000 字：章末大钩子（让读者必须看第 11 章）
```

## 最后提醒

### 死亡开局不是噱头，是底线

不要试图"创新"突破死亡开局。番茄的算法基于数据，前 200 字跳出率是硬性指标。

### 钩子要"具体"，不要"抽象"

❌ "他想起了过去种种"
✅ "他想起了那年冬天，父亲倒在血泊中的样子"

### 章尾钩子是"承诺"，不是"骗局"

钩子设的悬念，下一章必须真有内容兑现。不要"标题党"骗读者翻页。

### 范例不是抄袭模板，是公式参考

10 个核心范例 + 扩展指引提供的是"开局公式"和"语感"，不是抄袭素材。学结构和节奏，不要抄具体内容。
