## 男女频识别 3 信号

不要让模型靠"猜"决定性别向。如果信号矛盾，**强制询问用户**。

### 信号 1：对标书的性别向

如有对标书，**优先看对标书的性别向**：

| 对标书 | 性别向 |
|--------|--------|
| 《斩神》《十日终焉》《诡秘之主》 | 男频 |
| 《游戏入侵》《攀高枝》《长矛老师》《癫》 | 女频 |
| 《冒姓琅琊》 | 跨频（古言可男可女） |

判断依据：
- 主角性别 + 视角
- 爽点类型（男频偏暴力/装逼/收女；女频偏复仇/事业/打脸）
- 章评/弹幕的读者性别比

### 信号 2：题材标签

不同题材天然偏男频或女频：

**男频强偏向**：
- 玄幻、仙侠、武侠
- 都市高武
- 系统流（无女主向）
- 历史军事

**女频强偏向**：
- 言情、霸总
- 古言、宫斗、宅斗
- 穿书、年代文
- 重生（女主复仇/事业）

**跨频中性**（看主角性别）：
- 都市、悬疑、推理
- 末世、星际
- 现实、职场
- 脑洞文

### 信号 3：核心爽点偏好

| 爽点偏好 | 男频 | 女频 |
|---------|------|------|
| 装逼打脸 | ✓（核心爽点） | ✓（次要爽点） |
| 收女后宫 | ✓ | × |
| 复仇 | ✓ | ✓（核心爽点） |
| 事业升级 | ✓ | ✓ |
| 智斗 | ✓ | ✓ |
| 修罗场 | × | ✓ |
| 撕白莲花 | × | ✓ |
| 怼婆婆 | × | ✓ |
| 救美 | ✓ | ✓（救自己人） |
| 道德爽感 | ✓ | ✓ |

### 信号矛盾时的处理

如果 3 个信号有冲突（如对标书是男频但题材是女频），**强制询问用户**：

```
我注意到性别向有冲突：
- 对标书 {书名} 是 {男频/女频}
- 题材 {类型} 偏 {男频/女频}
- 你提到的爽点 {如复仇/装逼} 偏 {男频/女频}

请明确：本书的目标性别向是？
1. 男频
2. 女频
3. 跨频（同时吸引男女读者）
```

## 题材融合矩阵（性别向决策视角）

> **完整融合矩阵参见 [viral-elements-toolkit.md](viral-elements-toolkit.md) 「元 6：题材融合维度」**（含经典爆款 / 现实向 / 文化向 / 强设定四大分类）。
> 本节专注于"性别向 × 题材融合"的决策视角。

### 番茄爆款的融合公式

```
主题材 = 提供主线驱动力（占比 60%+）
副题材 = 提供日常爽点（占比 30%）
副副题材 = 提供反差和差异化（占比 10%）
```

### 男频高潜力融合速查（爽点公式）

| 融合 | 爽点公式 | 标志作 |
|------|---------|--------|
| 玄幻 + 系统 + 重生 | 升级 + 信息差 | 主流玄幻 |
| 都市 + 重生 + 商战 | 信息差 + 商战装逼 | 《重生鉴宝》系 |
| 修仙 + 都市 + 职场 | 修仙规则 + 职场代入 | 修仙 2.0 |
| 玄幻 + 扮猪吃虎 + 都市 | 隐藏身份 + 装逼亮牌 | 都市装逼流 |
| 历史 + 穿越 + 考据 | 考据装逼 + 历史智斗 | 《冒姓琅琊》 |
| 苟派 + 无限流 + 推理 | 低调升级 + 群像反转 | 《诡舍》 |

### 女频高潜力融合速查（爽点公式）

| 融合 | 爽点公式 | 标志作 |
|------|---------|--------|
| 言情 + 重生 + 复仇 | 复仇 + 火葬场 | 《攀高枝》系 |
| 穿书 + 娱乐圈 + 发疯 | 反内耗 + 撕白莲花 | 《癫》《长矛老师》 |
| 重生 + 商战 + 复仇 | 无 CP + 事业 | 《游戏入侵》 |
| 年代 + 真假千金 + 先婚后爱 | 年代 + 反转 | 《我在八零当后妈》 |
| 女强男弱 + 商战 + 暧昧 | 女王气场 + 暧昧拉扯 | 《知京冬风雪》 |
| 末世 + 经营 + 治愈 | 治愈 + 经营升级 | "末世奶茶店"类 |

### 融合的 4 大原则

1. **主次分明**：1 个主题材 + 1-2 个副题材（详见 viral-elements-toolkit.md「八元配置规则」）
2. **化学反应**：副题材必须能为主题材带来新看点（不是并列）
3. **力量体系统一**：跨题材的力量体系必须明确用哪个（如修仙+都市必须明确）
4. **平台适配**：番茄爱融合，起点偏单题材，晋江偏感情线，按平台选

### 步骤 1：根据性别向 + 题材确定"主元素"推荐

**强烈推荐主元素**：

#### 男频扮猪吃虎/伪装流

**强烈推荐主元素**：

## 对标书选择原则

### 番茄爆款的对标书选择"三宜三不选"

**三宜（应该选）**：

1. **平台匹配**：对标书最好就是番茄上的（其他平台爆款不一定适合番茄）
2. **题材匹配**：本书的主题材与对标书一致
3. **方法论可学**：对标书的成功有清晰的方法论（不是"我也不知道为什么火了"）

**三不选（避开）**：

1. **千万级大神书**：笔力碾压，套路近乎无形，学不会（如《斗破苍穹》《诡秘之主》对新人来说太高）
2. **刚火的题材跟风书**：流量已被火书吸光，跟风必死
3. **个人风格极强的好书**：独特世界观和风格无法复制

### 推荐的对标书选择层级

| 对标层级 | 标准 | 例子 |
|---------|------|------|
| 万订爆款（最佳） | 30-100 万在读 / 500-3000 均订 | 《攀高枝》《长矛老师》 |
| 精品至万订之间 | 完结口碑+短剧改编 | 《我在八零当后妈》 |
| 千万级大神（不推荐新人） | 文学性 + 套路无形 | 《诡秘之主》《斗破苍穹》 |

### 对标书的"三同"原则

选对标书时，理想的对标书与本书要满足：
- 同平台（都在番茄）
- 同题材（主题材一致）
- 同性别向（都是男频或都是女频）

如果三同都符合 → 强对标
如果两同符合 → 中对标
如果一同符合 → 弱对标（仅作参考）

## 适配冲突解决

### 冲突 1：性别向冲突

**症状**：对标书与目标题材性别向不一致。

**解决**：
- 优先以"目标题材"为准
- 用对标书的"通用方法论"（钩子/节奏/伏笔）
- 不用对标书的"性别专属表达"

### 冲突 2：题材冲突

**症状**：对标书是 A 题材，你想写 B 题材。

**解决**：
- 不强行用 A 的方法套 B 的题材
- 找 B 题材的对标书
- 用对标书的"方法论"而不是"题材桥段"

### 冲突 3：元素冲突

**症状**：选定的元素之间互相打架。

**解决**：
- 重新审视八元的"叠加原则"
- 主次必须分明
- 不要 4+ 元素叠加

### 冲突 4：代入感冲突

**症状**：极端化让代入感失效。

**解决**：
- 优先保代入感（参考 [immersion-anchor.md](immersion-anchor.md)）
- 极端化的"边界"必须明确

## 关键提醒

### 性别向不是绝对的

虽然番茄上男女频分类清晰，但"跨频爆款"也存在：
- 《诡秘之主》：男频写法 + 女频读者也爱
- 《冒姓琅琊》：考据流 + 男女通吃

如果你能写出"跨频爆款"，市场更大。

### 元素是工具，不是目标

八元的选型是"工具选择"，不要为了"用更多元素"而硬塞。**主元素 1 个 + 副元素 1-2 个**就足够。

### 对标书是参考，不是抄袭

对标书提供"方法论"和"节奏感"，但**严禁照搬具体角色、世界观、桥段**。

参考 `../story-long-write/references/artifact-protocols.md` 第 51-52 行"禁止照搬"原则。
