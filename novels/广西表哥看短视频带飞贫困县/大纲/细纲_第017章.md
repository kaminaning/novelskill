# 第 17 章 细纲：溶洞地质难题 / 老工匠 + 系统蓝图

## 核心事件

时间 2026 年 4 月 21 日（周一）至 23 日（周三）。村道副本启动第二天，韦东组织的 17 人施工队（覃国伟带回来的弄崖村 + 苗排村青壮年 + 县城雇来的 3 个挖机师傅）从弄崖村北口 1.3 公里处开挖。**周一上午 11:17 出现重大意外**——挖机挖到 1.8 米深时陷入了一个**未标注溶洞**（喀斯特峰林峰丛地貌 + 石灰岩长期溶蚀的结果），溶洞直径约 4 米 × 3 米、深约 6 米、底部还在持续渗水。如果按常规填土法继续修，半年内可能塌陷；如果绕路则要多走 800 米、80 万扶贫款不够；如果走"溶洞顶部加桥"工艺需要石灰岩懂行的喀斯特石匠。韦东蹲在溶洞边沿 8 分钟没说话，然后让阿贵去村南把 78 岁覃成福请来。覃成福进入溶洞观察 17 分钟，出来时拄着拐杖说了一句话："**18 年没碰石头，手忘了。但石头没忘——这个洞是 1957 年柳钢勘探队碰到过的，我爹给柳钢供过这边的石灰，留过一张图。**" 韦东："**那张图在哪？**" 覃成福："**1998 年我爹去打工前锁在了吊脚楼的旧柜子里。**" 韦东突然想起第 6 章卷纲伏笔——**M02 弄崖村吊脚楼地下的旧机器（爷爷辈柳钢工人留下的）**——那个吊脚楼正是覃成福家的老屋。当晚韦东 + 覃成福 + 外公一起去那个吊脚楼，找出了**1957 年的喀斯特勘探手绘地质图 + 一把锈得几乎认不出的石匠凿**。系统在韦东打开手绘图时弹出："**蓝图匹配度 73% / 可补全。消耗 30 DP 补全完整工艺。**" 韦东 all in。蓝图自动写入记忆——他突然知道了"溶洞顶部加石灰岩桥拱 + 钢筋混凝土补强 + 排水暗渠"的全套工艺。覃成福手抖着握住 18 年没动过的石匠凿："**后生仔——你晓不晓得，我爹生前总说，弄崖村这地下，是龙城的根。**" 万人榜 #6900 → #6850。

## 章首钩子

**意外型 + 危机型** —— 周一 11:17 挖机突然下陷，整个铲斗从地面消失在一个 4×3 米的黑洞里：

> "**操他——这下面是个洞？**"
> "**多大？**"
> "**深 6 米。底部还在渗水。**"

## 章节结构

1. **施工首日 + 溶洞意外（500 字）**：周一上午 06:30 施工队进场。17 人，3 台挖机，弄崖村北口 1.3 公里处开挖。覃国伟（主角表哥 / 32 岁返乡失败案例 / 卷纲第二幕共情角色）带队 + 阿贵作为主角的运营搭档负责后勤。**11:17 第一台挖机的铲斗陷入地面**——挖机师傅老周尖叫一声："**操他——这下面是个洞？**" 工人围上来看——4 米 × 3 米 × 6 米深的溶洞，底部还在持续渗水。韦东蹲到洞边沿，目测 8 分钟没说话。覃国伟："**东哥，这咋整？填？绕？**" 韦东："**算了卵，先停工。让阿贵去村南把覃成福请来。**"
2. **覃成福进溶洞勘察（700 字）**：下午 1 点 17 分覃成福（78 岁 / 喀斯特石匠世家 / 18 年没干这活）拄着拐杖到现场。他穿一件旧蓝色解放装 + 一双布鞋。下溶洞前他蹲在洞边沿摸了摸石头——**石灰岩 + 风化层 + 渗水路径**——三个动作做完后让人用绳子放他下去。**他在溶洞里待了 17 分钟**。出来时拄着拐杖，先嗦了一口阿贵递给他的螺蛳粉汤（这一段细节延续了"嗦粉式思考"），然后说了一句话："**18 年没碰石头，手忘了。但石头没忘——这个洞是 1957 年柳钢勘探队碰到过的，我爹给柳钢供过这边的石灰，留过一张图。**" 韦东蹲下来："**覃叔，那张图在哪？**" 覃成福："**1998 年我爹去打工前锁在了吊脚楼的旧柜子里。**"
3. **找 1957 年的勘探图 + M02 伏笔回收预热（800 字）**：当晚 7 点韦东 + 覃成福 + 外公覃老六一起去覃成福家的吊脚楼。这个吊脚楼正是第 6 章村委会大白纸方案时主角"故意让韦德才看见"的那个老屋——M02 伏笔（弄崖村吊脚楼地下的旧机器）的实体。覃成福拄着拐杖打开吊脚楼地下室的木门——里面堆着旧柜子 + 一台 1980 年代的柳钢小型石灰窑（被山猪拱过但还能认）+ 一摞被苗绣布盖着的工具。覃成福颤抖着翻开第三个柜子，**取出一张折了八层的牛皮纸 + 一把锈得几乎认不出的石匠凿 + 一本 1957 年柳钢工人手册**。**那张手绘地质图上密密麻麻标注着弄崖村底下 17 处溶洞 + 石灰岩走向 + 地下水路径**——比县地质局的资料还详细。覃成福："**我爹叫覃永福，1957 年柳钢勘探队的临时工，懂壮苗汉三种话，能跟苗山的人交涉。**" 韦东看着那张图，眼眶红了第三次：**弄崖村失踪的男人，又多了一代**。
4. **系统蓝图补全 + 老工匠出山（800 字）**：韦东把那张 1957 年手绘地质图拍照——系统弹出红字"**电子化储存触发——3 秒后蒸发。**" 韦东抠了抠后脑勺："**挨卵啦——这鬼系统。**" 改为口头记忆 + 手抄。系统读取手绘图后弹出："**蓝图匹配度 73% / 可补全。消耗 30 DP 补全完整工艺。**" 韦东 all in。蓝图自动写入记忆——**他突然知道了"溶洞顶部加石灰岩桥拱 + 钢筋混凝土补强 + 排水暗渠"的全套工艺，包括每一道工序的工时 + 材料用量 + 关键节点**。他把蓝图工艺一字一句讲给覃成福听。覃成福听到第三句时拿起 18 年没动过的石匠凿，**手抖着握了三下握紧了**："**后生仔——你晓不晓得，我爹生前总说，弄崖村这地下，是龙城的根。**"（"龙城"=柳州雅称）外公在旁边补了一句："**搞得动就搞，搞不动就回家嗦粉。**" 当晚 11 点 17 人施工队 + 覃成福老工匠 + 系统蓝图三路汇合。次日（周二）开始溶洞顶加桥拱施工。万人榜 #6900 → #6850。

（合计约 2800 字）

## 爽点

**意外危机 + 老工匠出山 + 系统蓝图协同 + 工业血脉传承** —— 这一章的爽点不在数据而在"层级化解决方案"的展开：意外（溶洞）→ 人（78 岁老工匠）→ 物（1957 年柳钢手绘地质图）→ 系统（30 DP 补全蓝图）→ 协同（老工匠手 + 蓝图技术 + 主角组织力）。这种"传统手艺 + 现代系统"的协同是本书与对标书的核心差异化——对标书主角靠"行政指令 + 政策"解决问题，本书主角靠"传统手艺 + 系统蓝图 + 工业血脉传承"。同时 1957 年柳钢小型石灰窑 + 1957 年勘探队临时工覃永福 + 1998 年覃成福他爹去打工没回来三层时间锚，把"工业柳州血脉"从主角父亲一代往上推到了爷爷一代——读者会感受到"弄崖村不是凭空的穷山沟，是 70 年柳钢工业血脉的末梢"。这正是柳州元素强制清单第 4 节"工业柳州 / 柳钢 / 柳工 / 五菱重工业基因"的核心爽点。
