# 第 19 章 细纲：解锁人才树 T1 / 农技培训站开课

## 核心事件

时间 2026 年 4 月 27 日（周一）。第 18 章村道副本完成后村道点亮带来 + 300 DP 大奖励，韦东 DP 余额来到 **518 DP**（首次进入三位数高位）。系统弹出"**评论办公副本 4 激活：弄崖村人才培训 / 农技培训站**"——这是新一周副本起点。韦东周日下午把大白纸方案改过一遍，决定 all in 解锁**人才树 T1 / 农技培训站**（消耗 100 DP），同时启动"返乡青年招募计划"——卷纲规定的第二幕共情角色覃国伟（32 岁 / 表哥 / 2017 年茶园失败返回深圳 / 现已回弄崖村做村道副本运营搭档）作为第一个返乡青年挂牌入伙。周一上午 09:00 农技培训站在弄崖村吊脚楼旁的杉木林边正式挂牌——师资来自三方：系统蓝图给的"科学种紫苏 / 香糯 / 笋干 / 油茶树"技术包 + 主角对接的县农业农村局张秀芳副局长（卷纲规定第 20 章登场，本章先打电话铺设）+ 苗山镇 / 侗江寨过来的两位本地老农技手。首批学员 22 人——12 人弄崖村 / 6 人苗排村 / 4 人侗江寨——学员中包括一位 17 岁辍学回乡的女孩龙阿萍（苗排村 / 母亲 2 年前去广东打工没回 / 龙阿婆的外孙女）。开课首日韦东讲了 17 分钟，覃国伟讲了 23 分钟"我那个茶园是怎么亏掉的"——这是反面教材式分享。下午 14:00 直播培训站开课全程，在线 15 万人，弹幕"广西终于有自己的农技培训站了""我也想回去""我妈在广东打 8 年工"刷屏。M03 评论区留守儿童家长留言（第 14 章伏笔）回声变强——这一天评论区"我也想回去"的留言新增 1873 条。系统弹出："**评论办公副本 4 推进：1/7。**" 万人榜 #6500 → #6480。

## 章节结构

1. **DP 战略重排 + 解锁人才树 T1（500 字）**：周一清晨 06:30 韦东蹲在母亲油茶店门口把第 18 章新增的 300 DP 战略账目重排——**当前 DP 518 / 已解锁科技树：产业树 T1（11-12 章）+ 文化树 T1（14-15 章）+ 基建树 T1（18 章）/ 待解锁：人才树 T1（100 DP）/ 民生树 T1（80 DP）**。母亲在屋里打油茶问"你揍马咯"。韦东："**阿妈，**借个事好搞**——这次解锁人才。**"他点击系统按钮："**解锁人才树 T1 / 农技培训站 / 消耗 100 DP。**" 屏幕一闪——蓝图自动写入记忆：**科学种紫苏（亩产 3200 斤 vs 弄崖村当前 1800 斤）/ 香糯（亩产 1100 斤 vs 当前 750 斤）/ 笋干加工（出干率 18% vs 当前 9%）+ 油茶树（结实期从 7 年缩到 5 年）**——四个本地农产品的科学种植/加工全套技术包。
2. **农技培训站挂牌 + 师资三方（700 字）**：上午 09:00 弄崖村吊脚楼旁的杉木林边正式挂牌——"**弄崖村农技培训站**"。匾额用杉木手刻，是覃成福老工匠（第 17-18 章石匠出身）昨天晚上熬夜刻的。师资三方：1）韦东（系统蓝图技术包负责人 / 不挂头衔但实际主讲）；2）覃国伟（32 岁 / 表哥 / 现任运营 + 学员双重身份）；3）主角对接的两位本地老农技手——苗排村 67 岁覃永田 + 侗江寨 71 岁吴守德。同时韦东电话联系县农业农村局副局长**张秀芳**（卷纲规定第 20 章登场，本章电话铺设）——张秀芳 49 岁、温和型干部、对扶贫务实派、听完培训站方案后说"**周振华副县长前两天跟我提过你。下周我过来看看。**"
3. **首批学员 + 龙阿萍 + 反面教材分享（800 字）**：22 人首批学员：12 人弄崖村 / 6 人苗排村 / 4 人侗江寨。年龄从 16 到 64 岁，男女各半。**学员中包括一位 17 岁辍学回乡的女孩龙阿萍**——苗排村人 / 母亲 2 年前去广东打工没回 / 父亲在柳州市区打工偶尔寄钱 / 她跟外婆龙阿婆（71 岁苗绣老艺人 = 主角外婆）相依为命 / 初三辍学 / 想学农技养活自己 + 外婆。龙阿萍坐在第二排，眼眶红红的没说话。开课首日韦东讲了 17 分钟科学种紫苏的技术包——亩产 3200 斤 + 紫苏深加工方向。然后**覃国伟讲了 23 分钟"我 2017 年那个茶园是怎么亏掉的"**——反面教材式分享。他讲到第 18 分钟时说出第 18 章铺设过的金句的完整版："**我也想留下来啊——但留下来真的能活吗？我 2017 年回来时跟你们一样，以为我有大学文凭就行。结果三年亏了 48 万，连给我妈过个 60 大寿的钱都没有。这就是为什么我跟着东哥再回来——**这次不一样**，这次有系统、有路、有培训站。**" 弹幕全网刷屏"求一个返乡机会"——M03 伏笔（评论区留守儿童家长留言）的回声在本章变强。
4. **培训站开课直播 + 系统副本推进（800 字）**：下午 14:00 直播培训站开课全程。直播间在线峰值 15 万人，弹幕"广西终于有自己的农技培训站了""我也想回去""我妈在广东打 8 年工"刷屏。当天评论区"我也想回去"的留言新增 1873 条（卷纲伏笔表 M03 加层级）。系统弹出："**评论办公副本 4 推进：1/7。**" 韦东在大白纸上添了一栏新数据："**弄崖村返乡青年潜在数 1873 人。**" 阿贵蹲在旁边："**东哥，这要是真有 1800 个人回来——咱弄崖村住不下啊。**" 韦东："**几个表弟——**借个事好搞**。住不下，咱就**修房子 + 修工厂 + 修学校**。一步一步搞，不慌。**" 当天晚上 17:30 张秀芳副局长电话回访——她说下周三过来培训站考察。万人榜 #6500 → #6480。卷尾韦东蹲在培训站门口望着 22 个学员排着队签到，龙阿萍是最后一个走进来的，签到时小声说了一句："**东哥——我能帮我外婆把苗绣卖到全国吗？**"

## 爽点

**人才系统启动 + 返乡浪潮预热 + 共情角色推进 + 反面教材式人设** —— 这一章的爽点是"层级化扩展"——主角从前面的"产业 / 文化 / 基建"三层科技树进入第四层"人才"，并且开始正面规划"让 1873 个想回家的人回来"。这是卷纲第二阶段（31-100 章）"融城摘掉贫困县帽子 + 万人榜冲入前 5000"的关键铺垫。共情角色阶梯第二层（覃国伟 / 32 岁返乡失败青年）正式上线全网级金句"我也想留下来啊——但留下来真的能活吗"——这是对标书"杨德贵那句活了 62 年头一回有当官的听我说话"在本书的对应锚点。同时 17 岁辍学女孩龙阿萍（苗绣老艺人外孙女 / 母亲在广东打工没回）作为本卷"留守儿童 + 留守青年"双重身份共情角色补充，把第 14 章 M03 伏笔（评论区留守儿童家长留言）从抽象"东莞陈女士"具象化到弄崖村本地"龙阿萍"。读者会感受到"主角不只是修路 / 修产业，他在修整一代人的命运"。这是本书与对标书最大的格局差异——主角的目标不是"打脸"或"建制度"，而是"让一代人回家"。
