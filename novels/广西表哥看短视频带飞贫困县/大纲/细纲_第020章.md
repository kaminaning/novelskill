# 第 20 章 细纲：培训站开课 / 苏家盯上信息差 / 苏明远派周末团建去外县（商标截胡前奏）

## 核心事件

时间 2026 年 4 月 28 日（周二）。培训站开课第二天，韦东带着 22 名学员到田间地头实操——上午到弄崖村紫苏地学科学种植法、下午到苗排村油茶林学结实期管理、傍晚到侗江寨笋干棚学发酵工艺。直播一整天累计在线峰值 18 万人。本章一个核心场景是张秀芳副局长（49 岁 / 县农业农村局副局长 / 卷纲规定第 20 章正面登场）从县城来到弄崖村培训站考察——比预定的下周三提前了 6 天——这种"提前到访"暗示县里对主角的关注度在升级。**张秀芳在田间地头跟主角谈了 3 个事情**：1）县农业农村局可以挂牌做"弄崖村培训站省级农技推广点"；2）但希望主角的产业能不只是停在弄崖村一个点，要做"融城北部山区五乡联动"；3）她暗示"周末苏家有动作，你自己注意"。韦东蹲在油茶林边嗦了口螺蛳粉汤："**张姐，您放心——我醒水的咯。**" **本章的真正爆点在弄崖村以外**——同一天下午苏明远办公室。苏明远（48 岁 / 远辉实业集团董事长 / 主反派）召集弟弟苏明亮 + 妹妹苏婉婷 + 小舅子黄大发开会，把过去 18 天韦东的直播数据 / 销售数据 / 万人榜变化打印成 87 页厚的资料摆在桌上。"**这小子三周从全国 #9527 跑到 #6500——前进 3000 名。**" 苏明远轻轻敲了三下桌面（标志性动作）："**周末远辉实业团建。地点：南宁。事项：跟商标代办公司谈合作。把'融城非遗体验馆 / 弄崖村螺蛳粉 / 苗山苗绣 / 三江油茶' 4 个品类的商标注册了。注册主体写远辉实业旗下'融城远辉文旅 有限公司'。**" 这是 M04 伏笔（苏明远派周末团建去外县）的正式落地——为第 28 章商标截胡铺设。苏宇航（苏明远 18 岁儿子 / 主角抖音粉丝 / F02 伏笔）在隔壁书房里听见了——他偷偷打开抖音翻到主角昨天的视频。系统在韦东这边弹出："**评论办公副本 4 推进：3/7。**" 万人榜 #6480 → #6470。

## 章首钩子

**反派暗影型 + 反派轻视段** —— 苏明远把 87 页韦东直播数据资料摆在会议桌上，弟弟苏明亮翻了三页就笑出声：

> "哥，一个被裁的程序员，搞几锅螺蛳粉而已。"
> 苏明远没笑。他敲了三下桌面。
> "**这小子三周前进 3000 名。我们家用了 22 年，才进了 1000 名。**"

## 章节结构

1. **田间地头实操 + 22 名学员第一天落地（500 字）**：周二上午 09:00 韦东带 22 名学员到弄崖村紫苏地。**亩产 3200 vs 1800 的核心差异**——韦东讲到第 17 分钟时蹲下来扒开土："**几个表弟、几个表妹——莫搞这些有的没的——这棵紫苏的根扎深 22 厘米，咱们传统种法扎深 11 厘米。多扎一倍，亩产多一倍。**" 龙阿萍（17 岁 / 主角姨表妹 / 第 19 章学员）蹲在韦东旁边一笔一画记笔记，写得比所有学员都认真。中午在田垄上吃便当——母亲做的紫苏炒田螺 + 自带烧炙 + 三江油茶。下午到苗排村油茶林、傍晚到侗江寨笋干棚。直播一整天累计在线峰值 18 万人，弹幕"广西农技培训站第一天""我也想报名"刷屏。
2. **张秀芳副局长突然到访（700 字）**：傍晚 18:23 张秀芳副局长在侗江寨笋干棚找到主角——她比原定的下周三提前了 6 天到访，开的是一辆县农业农村局的旧捷达。张秀芳 49 岁，温和型干部，穿一件米色风衣 + 普通牛仔裤——不是周振华那种领导范，是真下乡的干部气质。她蹲在笋干棚的杉木门槛上跟主角聊了 23 分钟，谈了三个事情：1）**县农业农村局挂牌做"弄崖村培训站省级农技推广点"**——这是给主角的政治资源加层；2）**希望主角做"融城北部山区五乡联动"**——把弄崖乡 + 苗山镇 + 侗寨乡 + 北河镇 + 弄潭乡的产业串起来；3）**暗示苏家周末有动作**："**卯崽，借个事你晓得——这周末，融城商会几个老板要去南宁。你自己注意点。**" 韦东蹲在门槛上嗦了一口母亲灌的螺蛳粉汤："**张姐，您放心——我醒水的咯。**" 张秀芳临走前看了一眼龙阿萍："**这丫头我认识她妈——她妈 2 年前在番禺一个电子厂落了一段时间，后来没消息了。**" 韦东握紧手机——又是一个"出去打工没回来的母亲"。
3. **苏明远办公室开会 + 商标截胡布局（900 字）**：同一天下午 16:30 融城镇苏氏远辉实业总部 18 楼董事长办公室。苏明远（48 岁 / 三件套西装 / 浪琴名匠手表 / 南珠手串 = F01 伏笔标志性细节）召集弟弟苏明亮（46 岁 / 远辉物流总经理）+ 妹妹苏婉婷（43 岁 / 远辉商超总经理）+ 小舅子黄大发（55 岁 / 融江物流老板）开会。会议桌上摆着 87 页韦东直播数据资料。**苏明远把手指点到第 47 页**——"**4 月 19 日苗山镇拉鼓节直播 1.8 亿播放。4 月 26 日修通村道直播 5800 万播放。三周从 #9527 到 #6500，前进 3027 名。**" 黄大发："**姐夫，这小子要不要直接掐？**" 苏明远轻轻敲了三下桌面："**直接掐——动作太脏。**" 他抬头："**周末远辉实业团建，地点南宁。事项：跟商标代办公司谈合作。**" 然后亮出商标布局清单：**1）融城非遗体验馆 商标 / 2）弄崖村螺蛳粉 商标 / 3）苗山苗绣 商标 / 4）三江油茶 商标——4 个品类，全部注册主体写远辉旗下"融城远辉文旅 有限公司"。** 苏明亮："**这小子要这些商标干啥用？他自己还没做到。**" 苏明远："**所以现在抢——他还没醒水。他做大了，咱再有这些商标，价值翻 50 倍。**" 这是 M04 伏笔的正式落地——为第 28 章商标截胡铺设。
4. **苏宇航在书房 + 父子线伏笔加层（700 字）**：苏明远开完会回家时已经晚上 22:00。**苏宇航**（18 岁 / 苏明远儿子 / 高三 / 主角抖音粉丝 / F02 伏笔）在隔壁书房里听见了爸爸刚才在客厅打电话叮嘱"南宁那边商标代办的事，周一前必须搞定 4 个商标的初步申请"。苏宇航低头看自己手机——他刚刚把主角今天的"农技培训站第二天田间实操"直播切片收藏到自己的私密文件夹。他点开抖音给主角发了一条私信草稿——但写了三句又删了三句，最后只发了一个"广西表哥加油"的表情包。韦东那边没回——他还在弄崖村培训站宿舍跟覃国伟规划"五乡联动"的方案。系统在韦东这边弹出："**评论办公副本 4 推进：3/7。**" 万人榜 #6480 → #6470。卷尾韦东蹲在培训站宿舍门口风雨桥头，对着覃国伟说："**几个表弟——张姐刚说苏家周末有动作。咱挨卵啦——莫慌，但也莫睡。**" 覃国伟："**东哥，咱搞什么动作？**" 韦东："**让他们搞——你表哥我先把这培训站搞稳。蛋糕做大十倍，他要那点小蛋糕没用。**" 但他心里清楚——周末南宁的事情绝对不是小事。

## 爽点

**反派正式布局 + 共情角色加层 + 体制内盟友推进 + 卷外勾连** —— 这一章是承转章，承前面 11-19 章主角四连胜（直播爆款 / 政商谈判 / 拉鼓节直播 / 修通村道 / 农技培训站），转入第 21-30 章"危机暗涌 + 商标截胡 + 鸿门宴"的下半卷大节奏。爽点四层叠加：一是反派阵营立体化推进——苏明远（家族掌门人 / 精明隐忍） + 苏明亮（物流弟弟 / 轻视主角的典型轻视段） + 黄大发（小舅子 / 暴力倾向） + 苏婉婷（商超妹妹 / 沉默听从）+ 苏宇航（儿子 / 主角粉丝 = 反派内部缺口）的五人格局首次完整呈现；二是反派轻视段提前埋打脸——苏明亮"一个被裁的程序员，搞几锅螺蛳粉而已"会在第 28-30 章商标截胡逆袭时被打脸；三是张秀芳副局长正式登场——县级体制内第二位盟友（继周国强 + 周振华之后）+ 主动暗示苏家动作 = 体制内"实地主义"派开始向主角靠拢；四是 M04 伏笔落地 + F01 南珠手串 + F02 苏宇航父子线 = 反派阵营伏笔密度本卷新高。读者会感受到"主角第一次面对的不是单一反派，是一个家族 + 一个商业帝国 + 一个市级保护伞链条"的格局升级——这是对标书"钱富贵 5 家公司 + 三圈外围"在本书的对应锚点。
