# 《广西表哥看短视频带飞贫困县》全书大纲

> 总目标：600 章 / 6 卷 / 100 万 + 在读量

## 一、一句话梗概

被裁回乡的广西表哥韦东，绑定县城科技树系统 + 全国万人挑战赛，靠手机短视频和一锅螺蛳粉，把全国 GDP 后 10% 的融城从万人榜 #9527 一路带飞到 #1，把"看不到希望的桂中小县城"建成全国百强县。

## 五、主线、辅线、情感线

### 主线：科技树 + 万人榜爬升

- 第一卷 #9527 → #4500（万人榜垫底逆袭区）
- 第二卷 #4500 → #800（融城起飞期）
- 第三卷 #800 → #100（全省样本期）
- 第四卷 #100 → #30（全国关注期）
- 第五卷 #30 → #5（全国前列期）
- 第六卷 #5 → #1（登顶期）

### 辅线 1：评论办公副本

- 每 5-10 章一个完整的评论办公副本
- 第一卷副本（约 10-12 个）：油茶申遗 / 修村道 / 防溶洞 / 给老人取暖 / 留守儿童作业辅导 / 拉鼓节恢复 / 苗绣销售 / ...
- 各副本完成后：DP 大额奖励 + 民生改善 + 短视频破爆款 + 万人榜小跃迁

### 辅线 2：苏明远对决

- 1-25 章：苏明远侧面观察
- 25 章：正式接触
- 28-30 章：商标截胡（主角首败）
- 50 章：鸿门宴
- 80-150 章：全面对抗
- 150-200 章：决战 + 倒台

### 辅线 3：万人榜跨县联盟

- 第 22 章：第一次接触黄文秀（梧州蒙山）
- 第 33 章：跨县合作框架
- 第 80 章：三方联合（韦东 + 黄文秀 + 周林）
- 第 200 章：50 县联盟雏形
- 第 400 章：300 县联盟正式
- 第 600 章：全国 10000 县互助网

### 情感线（轻度）：韦东 × 覃小慧

- 1-30 章：青梅竹马重逢 + 互怼期 + 创业搭档
- 31-80 章：暧昧期 + 互相欣赏
- 80-150 章：加深期 + 暧昧深化
- 150-200 章：拉扯期 + 误会 / 苏家挑拨
- 200-300 章：挑明期 + 表白 + 正式在一起
- 300+：共建期

### 长线 ：父亲失踪真相

- 第 1 章：父亲失踪 18 年 + 衣冠冢（钩子）
- 第 47 章：发现父亲日记（碎片信息）
- 第 120 章：东莞同事赵建华出场
- 第 180 章：父亲生前最后工作工厂线索
- 第 350 章：父亲是某资本牺牲品的真相揭露
- 第 400 章：父亲落工原因（与系统幕后设计方有间接关联）

### 终极线：系统真相

- 第 1 章：系统激活（无说明）
- 第 23 章：发现"系统其实没真正惩罚机制"——保密规则是伪规则
- 第 200 章：系统人格化片段（轻度）
- 第 400 章：系统幕后设计方现身 + 万人榜实验真相
- 第 600 章：算法 vs 主角的最终对决 + 制度化收尾
