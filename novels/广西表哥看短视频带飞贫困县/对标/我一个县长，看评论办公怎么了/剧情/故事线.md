# 全书故事线

## 主线一句话

2024 年街道办科员李铮猝死穿越到 2017 年甘省凉水县（全国倒数第一贫困县）当代理县长，用短视频"评论区办公"模式半年扭转县政府治理瘫痪、扳倒地头蛇钱富贵和市副市长周永刚，把凉水县从全省倒数第一打造成省级示范点+国家课题样本。

## 三幕结构

### 第一幕：开局立威（第 1-20 章）

**核心冲突**：新县长 vs 凉水县官场惰政惯性

**节奏**：穿越上任 → 短视频立旗 → 第一条评论破局 → 撤职第一个局长 → 砍接待费买校车

| 章号 | 关键事件 | 标志性时刻 |
|------|----------|------------|
| 1 | 李铮穿越，看县城烂状 | "既然穿越了，那就干一票大的" |
| 2 | 注册"不忘初心 共创未来"账号 | 老杨头深夜留言 |
| 3-5 | 第一案：杨家沟修路 | 杨德贵田埂喊"十年了我写了八封信" |
| 6 | 赵德明茶凉两次敲打 | "凉水县的班子是什么样子？" 反将一军 |
| 7-8 | 大白纸+李家坪水污染查源 | 137 米外垃圾堆放场 |
| 9-13 | 第二案：城管局长陈志远 + 修路偷工减料**撤刘建国** | "凉水县炸了"——全县干部大会 |
| 14-16 | 钱富贵反扑：市纪委匿名举报+高建华调查 | 杨德贵拉调查组组长胳膊"别把李县长调走" |
| 17-20 | 校车案：砍三公经费 117 万买校车 | 张建军看车窗里儿子流泪+省下接待费视频全国热搜 |

**幕末状态**：粉丝 10 万+；3 个局长动了（刘建国免、陈志远斗、田志刚露面）；赵德明从"让他折腾"到"心里有数就好"

**节奏**：城管腐败查到底 → 教育系统案 → 修路 → 招商 → 医疗系统案 → 钱富贵反扑 → 副市长倒台

| 24-25 | 开发区 200 亩地+鸿门宴 | "不识抬举" 钱富贵翻脸 |
| 26-30 | **教育系统案**：粪坑小学+教育局长哭+全面改薄抢跑 | 马永强从警觉派彻底转化 |
| 31-34 | 教师工资挪用 180 万案 | 周敏"我们不走了" 出圈 |
| 35-36 | 甘省日报头版整版报道 | "老百姓不是不讲道理是没人讲" |
| 37-40 | **陈志远水军案 → 第二个局长下课** | 钱富贵砸青花瓷笔筒 |
| 41-42 | 秦远征 2000 万投资+三月攻坚 | "钱花在刀刃上，不花在面子上" |
| 43-44 | 钱富贵卡建材 → **李铮上网破垄断** | "你垄断？我上网" |
| 45-46 | 威胁信+村民自发帮修路 | "他们不是在帮政府修路，是在修自己的路" |
| 47-50 | 田志刚投案+秦远征签约+省政府考察 | "凉水县的春天来了" + 郑明远红头文件 |
| 51-55 | 郑明远实地+省电视台+医共体抢跑 | 陶春明"23 年盼到了" |

**幕末状态**：粉丝 200 万+；钱富贵被押；周永刚副市长留置；省政府文件下达

### 第三幕：丰收 + 越冬 + 制度化（第 71-98 章）

**核心冲突**：新县长 vs "人走政息"风险（自我超越）

**节奏**：制度化建设 → 智慧农业 → 电商爆款 → 取暖攻坚 → 除夕值班 → 国家课题样本

| 章节段 | 主线案件 | 转折点 |
|--------|----------|--------|
| 71-72 | 钱富贵供出周永刚副市长+周永刚倒 | 高建华"放心干 省里看在眼里" |
| 73-75 | 干部调整：方志明转正+何大勇副县级后备+赵德明合体上镜 | "倒数第一变中游了" |
| 76-77 | 四大央媒到+《凉水县网络民声办理工作制度》一号文 | 方记者"可持续性"问题 + 制度回答 |
| 78-80 | 方维智慧农业试点+独自走县城 | "你们自己也变了" |
| 81-83 | 省政府 117 号文+经验报告"实事求是"+丰年县长被打脸 | "我们做的事比你想象的更重要" |
| 84-85 | 王德旺手写信"别变"+全省推广会议 | "**别变**"两个字+一部手机镇住全场 |
| 86-87 | 新门诊楼动工+产科恢复首例分娩 | 赵军抱住陶春明腿"谢谢你谢谢你" |
| 88-89 | 老杨"我服气了"+凉水好物 3142 单 | "翻了一倍" |
| 90 | 八个月对比视频（全省第 19 位） | "别的县长在开会，李县长在干活" |
| 91-94 | 张国平国家课题组调研+取暖攻坚战 | "要是早一年有这个东西，我爸就不用死" |
| 95-97 | 除夕值班+大年初一管道爆裂 | "我没地方回"+ 透明认错 |
| 98 | 省政府 500 万试点+网络民声办成立 | "**从明天开始，你不是我的秘书了**" |

**结局状态**：凉水县全省第 19 位+国家课题样本（5 个之一/西北唯一）；李铮从"一个人盯手机"→"系统化平台+网络民声办"+从依赖个人→依赖制度
