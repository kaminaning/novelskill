# 第5章 母亲腰椎复发与高中同学

"东仔……阿妈起不来。"

2026年3月30日，周一，凌晨4点47分。

韦东被手机铃声叫醒，屏幕显示"阿妈"两个字，铃声是2018年默认的老 iPhone 铃。

他接起电话，听到母亲断断续续的喘气声。

那一句话从话筒里漏出来，像漏了气的旧棉布。

韦东瞬间清醒，套上拖鞋飞奔下楼，木梯吱呀吱呀响了一路。

母亲瘫坐在厨房地砖上，旁边是打翻的酸笋坛子，酸笋汁淋了一地，木板墙上溅着几道红色辣油。

她面色苍白，冷汗顺着鬓角往下淌，嘴唇咬得发白。

韦东冲过去把母亲扶起来——那一瞬间他才发现母亲瘦得只剩骨头架子，大概88斤。

母亲腰椎间盘突出复发不是第一次，但每次都比上次更狠。

两人都没说话，韦东拨阿贵电话："狗肉，借你五菱拉我阿妈去县医院。"

阿贵3分钟开车到，五菱小卡是那种拉货的，副驾驶位塞着两袋化肥，他把化肥扔到后车斗。

韦东抱母亲上副驾驶，自己挤在中间。

凌晨5点的盘山公路没车，五菱小卡50时速翻47个弯。

车头大灯把山路照成一截一截，每一截转弯尽头都是漆黑的山影。

车里没有暖气，韦东把母亲那件深蓝色苗绣外套盖在她身上，自己穿着短袖在4度的山风里发抖。

母亲咬牙说："东仔，莫慌，阿妈以前也这样，扛过去就好。"

韦东盯着挡风玻璃外掠过的盘山公路转角石灯，喉咙堵着没说话。

软肋开始疼了。

手机震了一下，蓝字弹出——【⚠️ 检测到主角情绪波动剧烈。提示：DP 转化率本周下降 15%。】

韦东在心里骂了一句"这个破系统"，把手机扣进口袋。

阿贵在驾驶座没吭声，方向盘握得很紧，每一个弯都比平时早半秒打。

母亲在副驾迷迷糊糊靠着车窗，嘴里念叨："东仔不哭，阿妈以前也这样。"

韦东没哭，但喉咙里像塞了块烧红的炭。

车过弯第19个的时候，远处的天色开始泛白，一小条灰白的光带挂在峰丛上方。

韦东看了一眼母亲，又看了一眼阿贵——这两个人，是他回弄崖村三天内最实在的依靠。

早上7点，融城县医院骨科门诊。

医院大楼是2015年盖的6层水泥楼，外墙瓷砖已经掉了几块，门口停着两辆救护车。

骨科在三楼，电梯堵着，韦东背着母亲爬楼梯，阿贵跟在后面拎着一个临时打包的洗漱包。

挂号窗口前，韦东蹲着算钱包：CT拍片，全套检查，住院押金，合计8550块。

手机银行余额23997块——扣掉给阿贵转的油费200，再减回乡这一周吃饭用掉的800。

N+1那3万1247块5毛6，还剩不到2万4。

他正想刷信用卡，身后传来一个声音："东哥？是你不？"

韦东转身——一个穿深蓝夹克的男人拎着两个保温饭盒站在那里。

他愣了三秒才认出来：周国强，融城一中高三8班的化学课代表，当年帮过韦东无数次抄化学作业。十年没见，两人都长开了。

周国强先开口："我阿爸住骨科隔壁病房，腰椎手术第三天。你呢？"

"我阿妈，刚送来，腰椎间盘突出复发。"

周国强二话不说，直接刷自己的医保卡垫了一万押金。

韦东要推："借个事我自己来。"

周国强按住他："东哥，你阿妈是覃姨，我高中六年每个礼拜都嗦她油茶店的螺蛳粉，这一万押金我刷得心安理得。"

韦东红了眼眶，两人在走廊上抱了一下。

"周国强，你还是当年那个借我化学卷子的瓜娃子。"

周国强笑："东哥，你还是当年那个把校长气哭的瓜娃子。"

走廊那头的电子屏跳到7点27分，扫地阿姨拖着推车从他们身边过去，水迹拖了一条长长的湿带。

韦东问："你阿爸要紧吗？"

"老毛病，腰椎L4-L5突出，做了一个小手术，第三天能下床走路了。"周国强叹气，"我妈在家炖排骨，我中午回去拿，下午再过来。你阿妈这边，我帮你叫骨科最好的张主任，他下午两点上班。"

"借个事，麻烦你了。"

"莫客气，先把人安顿好。"

中午12点，母亲已经办好住院，主治医生说"卧床7天 + 牵引治疗14天 + 后续看膝盖手术安排"，目前生命体征稳定。

韦东把母亲安顿在305病房——三人间，靠窗的位置，另外两张床住着两个老人，一个是断了胫骨的果农，一个是车祸的中年人。

母亲打了一针止痛药，已经睡了。

韦东轻轻把那件苗绣外套叠好放在床头柜上，转身出门。

走廊上飘着一股消毒水加饭菜的混合味，间隔3米的天花板上挂着白色LED灯，光把病人家属们的脸照得有点惨白。

周国强在走廊尽头等他："东哥，嗦碗粉？县医院对面那家老韦家。"

"嗯。"

县医院对面，老韦家螺蛳粉。

店是个6平米的小铺面，玻璃门上贴着一张2019年的"诚信经营"标志，已经发黄。

老板老韦头60岁，光头，认识周国强的父亲，看见两人进来咧嘴一笑：

"周科长来咯。"

"叫副局长。"周国强笑骂，"老韦头，两碗螺蛳粉，加蛋，加酸豆角，再来两瓶冰峰。"

老韦头从冰柜里掏出两瓶——其实是柳州本地的青岛崂山，但周国强坚持要叫"冰峰"，说是高中习惯。

两碗粉端上来，热气把酸笋味推到桌沿。

韦东先尝了一口汤，眯了一下眼："老韦头这汤还是当年那个味。"

"那当然，"老韦头从灶台后面探出头来，"你阿爸2007年带你来嗦的时候，我就是这个配方。"

韦东顿了一下，周国强没接话，喝了一口汽水。

两碗螺蛳粉加两个煎蛋加两瓶汽水，共36块，周国强抢着付。

两人在小桌上嗦粉。

周国强先开口："东哥，昨晚那个'我妈在弄崖村打了18年油茶'是不是你？"

韦东点头。

周国强放下筷子，看着他："东哥，我等你这个回乡仔等了八年了。"

韦东抬眼。

周国强继续说："我2018年从柳州外贸大学毕业回融城考公，现在文旅局副局长，管的就是非遗这一摊。融城下辖142个村，78个空心村，32万户籍人口跑走了14万。每年坡会能拉来4万游客，挣不了几个钱。"

"我跟县长打报告打了八年——要发展苗侗文旅，要做螺蛳粉作坊，要拍短视频带流量。上头一句话，'融城不是桂林，搞那些没用'。"

"东哥，你昨晚那18秒，把我憋了八年的话说了。"

韦东嗦完最后一口粉，抠后脑勺："几卵难搞，你表哥我刚回来，还啥都没搞，你倒先给我戴高帽。"

周国强笑："东哥，你不是来戴高帽的，你是来踩坑的。我能帮你的就两件——批文加快、政策给你说清楚。剩下的你自己干。"

韦东掏出绿色"评论办公本"，在第二页工工整整写下三行字。

"1. 问题：母亲腰椎复发 + 膝盖手术钱缺口2万4。"

"2. 交办：第一笔短视频变现必须在30天内到账。"

"3. 时限：2026年4月30日。"

写完，手机屏幕亮了一下，蓝字弹出：

【检测到新交办：30天内变现2万4。】

【建议路径：解锁T1螺蛳粉作坊蓝图（50 DP，已具备）→ 开作坊熬粉直播 → 评论办公副本001完成（+500 DP / 民生改善 +1）→ 排名跨进#9000。】

【⚠️ 同步提示：你30天后第一次季度淘汰还有59天。】

韦东合上本子。

周国强看见那三行字，只问了一句："东哥，你2万4缺口，凭什么30天到账？"

韦东抠后脑勺，笑了笑："你表哥我赌一把。"
