# 第1章 3万1247块5毛6

3万1247块5毛6。

这是韦东在深圳那家互联网中厂工作6年1个月7天，公司给他的全部交代。

2026年3月26日，周四，下午3点47分，南山区科兴科学园B座9楼会议室。

磨砂玻璃门被推开的时候，韦东正用红米手机搜"螺蛳粉哪家好吃"。

"韦东。"

HR王婷站在门口，手里夹着一份文件袋，封面三个字："优化方案"。

韦东抬头应了一声，隔壁PMO区的快递扫码声穿过隔音差的隔断——"嘀，嘀，嘀"。

王婷坐下，把文件袋推过来："公司这次优化……"

韦东手机震了一下，屏幕亮起来，是建设银行的短信。

【您尾号8842账户3月26日15:47收入31247.56元，来自：深圳XX科技有限公司。】

韦东看了一眼，抬头说："哦。那我把工位收了？"

王婷愣住了，张了张嘴："……你不问问原因吗？"

韦东抠了抠后脑勺："问了能不裁吗？"

她沉默了一下，没接话。

"那就不问了。"

韦东站起来："姐，麻烦你把离职证明给我，下周一去办失业金。"

王婷低头翻文件，眼神有点复杂："你……反应挺快。"

"挨卵啦。"

韦东从嗓子里冒出来这一句，王婷没听懂，他也懒得解释。

她拿出离职证明递给他，韦东签了字，又签了竞业协议，把笔放回桌上，整个流程不到八分钟。

她送韦东到会议室门口，犹豫了一下："韦东，你要不要……喝杯水再走？"

"不用了姐，我饿了，"韦东笑了一下，"想去楼下嗦碗粉。"

下楼的时候电梯里只有他一个人，他打开备忘录算了一道数。

房租4200，信用卡8000，还给阿贵的5000，母亲膝盖手术差的24000，弄崖村油茶店再亏3个月3000。

加起来44200。韦东手里的钱31247，差13000。

韦东笑了一下，对着电梯里的镜子说："挨卵啦，这数算不平。"

电梯到一楼门开，正午后的太阳从B座大堂正门斜着照进来，把保安老李的影子拉得很长。

老李抬头看见韦东："小韦，今天下班这么早？"

"嗯，以后可能更早。"

韦东笑了笑就走过去没解释，背包带在肩膀上滑了一下，他顺手拉了拉。

南山的太阳照得他有点晃眼，他站在大堂门口眯了一会，朝楼下那家螺蛳粉店去了。

那家店叫"柳州姐姐螺蛳粉"，开了三年，老板娘是融安人。

韦东嗦了一碗加蛋的，18块，比柳州贵一倍。

老板娘没认出他是融城人，只问："小帅哥要不要加紫苏？"

"要。"韦东说，"双份紫苏。"

她笑了："你是广西的？"

韦东嗯了一声，低头吃粉。

当晚回到城中村出租屋，南山区西丽，月租4200，10平米，飘窗外是另一栋城中村的窗户。

退租电话打了两通，房东说押金月底退。

韦东打包了一个旧背包：半瓶老干妈、半盒过期的螺蛳粉酱料包、母亲两年前织的苗绣围脖、一个充电宝、一支牙刷，和一本2020年的工作笔记。

笔记扉页写着"恭喜入职"。

他翻到最后一页，那里写着他入职时给自己定的目标——"5年内在深圳买房"。

韦东笑了一下，撕掉那一页揉成一团扔进垃圾桶。

剩下的页留着，万一以后回头看，也算追过点什么。

凌晨4点罗湖站，K157次硬座，深圳到柳州，46个小时，168块。

候车厅里挤满返乡农民工，蛇皮袋堆成小山，韦东找到自己的位置：23车厢68号。

邻座是一个穿迷彩外套的大哥，50岁上下，桂柳口音。

大哥打量了他一眼，开口问："卯崽，克哪里？"

"克柳州融城，弄崖村。"

大哥乐了："你揍马的？"

韦东抠后脑勺："被裁的。"

大哥沉默了一会，从蛇皮袋里掏出半瓶米酒递过来："喝一口，莫推辞。"

韦东喝了一口，辣得眼睛发酸。

大哥又给自己倒了一口，盖好瓶塞塞回袋子里，闭眼靠在座椅上。

车厢里挤得很，过道上坐着抱孩子的妇女，行李架上塞满编织袋。

空气里混着方便面、汗味、和不知道谁带的卤鸡腿的味道。

韦东把背包抱在腿上，闭上眼。

火车出广东边境时是凌晨6点零8分，韦东盯着车窗外的盘山隧道。

隧道一个接一个，灯光一明一灭。

那一瞬间脑子里闪过2008年5月，韦东10岁。

父亲韦海军在弄崖村堂屋里收拾行李，他要去东莞。

母亲蹲在灶台前烧火，背对着父子俩。

父亲临走前蹲下来，拍了拍韦东的头："东儿，阿爸过两天就回。"

韦东应了一声，父亲背着蛇皮袋走出门，走到院子门口，回头看了他一眼，笑了一下。

那是韦东最后一次见到父亲。

车窗玻璃上映出韦东自己的脸，他28岁，跟父亲走的时候差不多大。

只不过父亲是出去，韦东是回来。

柳州站下午3点，韦东在站前广场买了一碗螺蛳粉，8块。

酸笋的气味窜进鼻腔，那一刻他真的回来了。

他蹲在路边吃完，抹了抹嘴，扛着背包去客运站。

客运站墙上贴着"融城方向 / 每30分钟一班"的红字告示，已经褪色到几乎看不清。

转城乡客车到融城县城用了两个钟头，半路停了一次让人上车。

下车时口袋里的手机震了一下，是阿贵发来的微信："卯崽，到县城了不？我表叔在客运站接你。"

韦东回过去："到了。"

阿贵秒回："五菱蓝色那台。"

县城客运站，表叔在停车场边上等他，五菱不在，是一台二手桑塔纳——阿贵记错了。

表叔没多问，只递给韦东一根红双喜："东仔，回来咯。"

韦东接过烟应了一声。

"莫想那么多。"表叔自己点了一根，"你阿妈昨天就跟人讲你今天到了，杀了一只老母鸡。"

韦东点点头没接话，抽了两口又把烟掐了，山路开始绕，烟味让人晕。

弄崖村离融城县城28公里，47个弯。

桑塔纳是2003年的车，发动机冒着白烟，车里的收音机串着调，断断续续播放着柳州交通台："……新时代柳州人——五菱，人民的五菱……"

韦东把头靠在车窗上，外面的山是青色的，喀斯特峰丛一座挨一座。

第23个弯外，融江发着白光，像一条银带嵌在峰丛之间。

韦东闭着眼睛，盘算回家先嗦哪家粉店，是覃记，还是村口阿婆那家。

手机突然黑屏，紧接着又亮起来，一行蓝色的字浮在屏幕上：

【检测到宿主满足条件：户籍县城GDP全省后10% / 宿主年龄35岁以下 / 个人GPA大于0 / 返乡创业意愿值 ≥ 80%】

【「县城逆袭科技树·全国挑战赛」绑定中…… 绑定成功。】

【当前宿主：广西柳州融城县弄崖村韦东。】

【全国挑战赛初始排名：#9527 / 10000。】

【⚠️ 季度末末位100名淘汰。下次结算：90天后。】

韦东瞪着屏幕，愣了三秒。

第一反应：诈骗。

他把屏幕息了，又点开，蓝字还在。手指划过去，划不掉。

底下又冒出一段长字，密密麻麻：

【你的金手指：「县城综合发展科技树」+「全国万人排行榜」+「评论办公副本」三合一。】

【排行榜算法：县城GDP增速50% + 民生指标改善20% + 科技树节点数15% + 短视频流量与评论15%。每周更新一次。】

【升级燃料：DP（开发点数）。当前余额：0 DP。】

【DP获取方式：短视频评论 / 直播观众 / 涨粉 / 真实民生诉求（5 DP/条，触发评论办公副本）。】

【最近可解锁：T1 螺蛳粉作坊蓝图——需50 DP。】

【⚠️ 全国9999个其他绑定者已经在跑。你#9527 / 10000，落后第一名9526名。】

韦东盯着那一长串看完，又划了一下屏幕。

字还在。

表叔扭头瞥了一眼："东仔，你手机坏了？"

"嗯，可能。"韦东把手机扣进衣兜。

表叔咂嘴："深圳的破手机，跑山路就花屏。"韦东没解释。

桑塔纳又过了一个弯，广播里在放《新时代柳州人》的尾奏，收音机沙沙响。

韦东把那段话在脑子里捋了一遍：

短视频的评论能换DP，DP能解锁蓝图，蓝图能在融城开作坊。

作坊能卖钱，能往排名前面爬，能避开末位100淘汰。

——也就是说，他要从一个被裁的程序员，变成一个拍短视频的农村作坊老板。

#9527。九千五百二十七，一万人里第九千五百二十七，后面九十天要爬过四百二十七人才不被踢。

挨卵啦。

车头一转，弄崖村的村口到了，褪色的牌坊立在路边。

五个红字早就掉了漆——【全国GDP后10%县——融城欢迎您】，褪到只剩下一层暗红的轮廓。

韦东盯着那块牌坊，抠了抠后脑勺，然后笑了。

"挨卵啦。你表哥我都被裁了，还能再坏到哪去。既然系统都看上我了——那就跟你们这一万个狗肉，赛一下咯。"

桑塔纳轰隆一声开过牌坊，进了村。

韦东手机屏幕上，红色的"#9527"还在亮着，闪了一下，又闪了一下。
