# 第3章 穿西装的来油茶店打听

"83万。挨卵啦，你表哥我火了？"

2026年3月28日，周六，早上7点30分，韦东坐在吊脚楼木板床的床沿，手机捧在膝盖上。

屏幕亮着，播放量83万，点赞3万，评论3247。

评论区第一条置顶的，还是凌晨那条"东莞螺丝厂阿强"。

韦东盯着那三行字看了二十分钟，没回复，也没回别的评论。

"兄弟，你阿爸是不是2008年5月从你们柳州走的？"

"是不是叫韦海军？"

"——是不是真的？"

韦东冲下楼，母亲在灶台前打油茶，今天她坐着打——昨天那一下腰让她不敢再蹲。

"阿妈。"

"嗯？"

"阿爸2008年留下的东西，还在吗？"

母亲打油茶的手顿了一下，没回头。

"在。床底下那个铁皮饼干盒。"

韦东嗯了一声，转身上楼，母亲在背后说："东仔，你做嘛突然问这个？"

韦东顿了一下："没事，就是看看。"

母亲没再问，她把油茶倒进碗里，姜的味道从灶台飘上来，混着木柴烟。

吊脚楼二楼的木板地震得轻轻一颤——是母亲叹气时撑桌子的手，韦东假装没听见。

韦东跪在母亲床边，伸手到床底摸，摸了三下才碰到那个铁皮盒，拖出来，吹掉一层灰。

盒子是1996年的苏打饼干罐，蓝色铁皮上画着两个穿白裙的小女孩。

盒盖边缘有一道压痕——18年前韦东自己用指甲掐的。

他打开。

里面三样东西：一张柳钢工作证，照片上的男人32岁，平头，穿蓝色工装，名字栏写着"韦海军"，下面一行小字"1972年生／车间编号K0287／热轧分厂二车间"。

一张2008年4月17日邮戳的信，信封发黄，墨水迹有点晕。

一块停了的上海牌手表，表盘上的时间停在凌晨3点47分。

韦东先打开信，信纸是粗糙的小学生作业本，蓝色横线已经褪色。

字是父亲的字，方方正正，写得很慢——

"秀英：

我到东莞了，在长安镇螺丝厂上班，宿舍是八人间，比柳钢宿舍小，但管饭。

工厂叫"长安兴隆五金"，老板姓林。

等我赚到三个月工资就回，估计七月底。

东儿读书的钱我每月寄回，莫担心。

弄崖村那块自留地的早稻该割了，喊你哥来帮你。

海军，2008年4月17日。"

短短一封信，韦东读了三遍，把"东莞螺丝厂"五个字读了三遍，又把"长安兴隆五金"六个字读了三遍。

然后从口袋里掏出铅笔，在信封背面用很轻的笔触写下七个字：

"东莞螺丝厂阿强。"

写完他把信折好放回盒子，铁皮盖一合，发出"咔哒"一声。韦东没告诉母亲，把盒子推回床底，又抓了把灰盖上去。

下楼的时候母亲正在切红薯——午饭的菜，她回头看儿子："吃饭了。"

"嗯。"

中午1点47分，覃记油茶店门口，韦东蹲在门槛上嗦螺蛳粉。

母亲在店里擦桌子，一边擦一边随口说："东仔，今早有个穿西装的来店里打听你的事，问得几仔细。"

韦东嘴里的粉条停了一下。

"问你在深圳是做什么的、有没有钱、有没有结婚、几时回来的。我没多讲，说你回来住几天。"

韦东抬头："穿什么西装？"

母亲擦完一张桌子，想了想："深灰色的，手腕上戴串红色手串。"

"几个珠子？""我没数，挺粗的，颜色像晒过的红枣。"

韦东没说话，低头把那口粉嗦完，又点开手机搜"融城远辉实业"。

百度首页第一条出来一张董事长照片——

苏明远，48岁，深灰色西装，右手腕戴着一串南珠手串，色泽偏红，是北海老南珠。

韦东把照片放大，又缩小，又放大。

母亲在背后问："东仔？怎么了？"

"没事，"韦东把手机扣过去，"阿妈，你以后店里来什么人都莫多讲我的事。"

"嗯。"

母亲没追问，转身继续擦桌子。

她的右手扶着桌沿，左手在擦——右手是不能用力的，韦东假装没看见。

手机震了一下，屏幕跳出一长串蓝字：

【检测到评论 3247 条 / 识别到真实民生诉求 1 条。】

【触发「评论办公」副本·001：母亲油茶店申报融水苗族非遗。】

【触发条件：评论区里6条留言反复提"覃姨的油茶应该申非遗"，集中度达阈值。】

【⚠️ 必须7天内回应 + 30天内推动解决。】

【完成奖励：DP +500 / 民生改善 +1 / 排名加0.3分。】

【未完成惩罚：DP -100 / 评论办公本黑标记 / 关注度衰减15%。】

【倒计时：30天0时0分。】

紧接着又弹一行：

【当前万人榜·本周变动】

【#9527 → #9341，前进186名。】

【上升原因：短视频流量带动「短视频与评论」分项 +12.4分。】

【与你的省内对手（广西30名绑定者）排位：第27 / 30。】

【与你的桂中区域对手（前10）排位：第8 / 10。】

韦东盯着那行字看了三秒。

挨卵啦。

他把最后一口粉汤喝完，抹了抹嘴，从背包里掏出手机自带的备忘录。

新建一条，工工整整记下三行字：

"1. 问题：苏家眼线打探。

2. 交办：覃小慧打听苏家底细，阿贵盯紧外人。

3. 时限：3天。"

存档，命名："评论办公本·001"。

韦东把手机扣过去，又嗦了一口粉汤。

下午3点整，门口传来五菱宏光发动机的声音，一辆白车停下，车门打开。

覃小慧从驾驶座下来，穿着改良版苗族短上衣加牛仔裤，手里拎着两个用粽叶包好的糯米粽。

她比18岁那年长高了5公分，皮肤被苗山日头晒得健康黝黑。

她梳着一根麻花辫，辫子末端用一只塑料蝴蝶发卡别着——蝴蝶是粉红的，但已经褪到几乎是灰色。

那只蝴蝶发卡是韦东12岁那年在融城县城三元店买的，5块钱。

他下意识看了一眼，目光停了不到一秒。

覃小慧先开口："卯崽，听说你被毕业咯？"

韦东抠后脑勺："消息几灵通。"

"村群你阿叔德才发的，配图你骑电瓶车的背影。"

"几卵难搞。"韦东笑了一下，"我连个秘密都没有了。"

覃小慧把粽子塞过来："阿婆让我带的。说你瘦了。"

韦东接过粽子，粽叶上还带着竹蒸的水汽。

她身上有一股淡淡的紫苏味——是苗山自己长的紫苏，跟母亲店里那种一样。

韦东看了一眼她手机壳——壳上别着一个二维码贴纸，写着"小慧·苗山非遗 | 抖音"。

他点开抖音搜了一下，"小慧·苗山非遗"，8.2万粉，主页全是芦笙、苗绣、坡会的短视频。

她注意到韦东在搜什么，没说话。

"明天的视频你帮我拍下一条，"韦东说，"我教你怎么剪。"

她愣了："凭什么？"

"凭你阿婆让你给我送粽子。"

覃小慧脸一红，翻了个白眼："韦东你借个事——"

她话没说完，转身上车，五菱宏光发动机响了一下，开走了。

车开过去十米又停下，副驾驶车窗摇下来，她探出半个头："明天下午三点，弄崖村风雨桥头。"

韦东嗯了一声，车开走了。

他蹲在地上嗦完最后一口粉汤，打开抖音，搜索框输入："苏宇航 融城"。

第一个跳出来的账号——"航宇苏"，2.3万粉。

头像是个染了灰色头发的小伙，主页置顶视频标题《我爸又开会了》，是录他爸苏明远在公司会议室拍桌子的尬视频，2.7万播放。

第二条视频是他在融城"夜色酒吧"打卡。

第三条是开他妈的捷豹去江滨大道兜风。

韦东滑了五条，关掉视频界面，点开私信。

收信人："航宇苏"。

韦东打字，没用语音："听说你爹找我有事？过几天我请他嗦粉。"

发送。

屏幕上跳出三个字："已送达"。

韦东把手机扣进口袋，蹲下来又嗦了一口螺蛳粉。

汤已经凉了。

但酸笋还是那个酸笋。

苏宇航回不回，都在韦东的预料之中。
