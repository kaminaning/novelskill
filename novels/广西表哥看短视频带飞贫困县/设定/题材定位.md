# 题材定位

## 基本信息

- **题材类型**：都市脑洞（系统 + 基建 + 种田 + 短视频 + 扮猪吃虎）
- **核心梗**：被裁回乡仔绑定县城科技树系统 + 全国万人排行榜 + 短视频评论值→DP 转化，把广西柳州融城县从全国垫底拍成全国第一
- **微创新点**：
  1. **科技树 + 万人榜 + 短视频评论 三系统融合**（吸纳对标书《评论办公》核心机制 + 县城科技树系统创新）
  2. **季度淘汰制**（每季末末位 100 名淘汰）持续危机感
  3. **柳州地域深度绑定**（与广西其他地方、广东、川渝彻底区分）
  4. **苗侗融合风情**（融水苗 + 三江侗 + 工业柳州三重内核）

## 广西柳州特色定位（与已有 县城科技树 项目最大差异）

> ⚠️ 已有 县城科技树 项目地域内核是泛广西 + 桂西山区 + 壮族 + 甘蔗经济（实际偏河池/百色），与柳州不符。本书内核是**柳州桂中工业城市 + 融水苗 + 三江侗 + 工业基因**。
> 详见 `设定/世界观/柳州元素强制清单.md`。

### 本书的差异化核心（强制约束）

1. **方言**：桂柳话（西南官话），不是粤语 / 南宁白话 / 桂林话。详见强制清单 1 节
2. **饮食**：螺蛳粉 / 紫苏炒田螺 / 螺蛳鸭脚煲 / 三江油茶 / 地羊。**严禁**老友粉 / 啤酒鱼 / 早茶
3. **风俗**：融水苗系列坡会群 + 三江侗鼓楼 / 风雨桥 / 大歌 + 拉鼓节。**严禁**用刘三姐 / 歌圩当融水标识
4. **工业**：五菱 / 柳钢 / 柳工 / 螺蛳粉产业链。**柳州工业产值占广西 1/4** 这一身份是反差爽点的根基
5. **人文**：龙城雅称 + 柳侯祠 + 柳江穿城 + 桂中商埠

### 与对标书的地域差异

| 维度 | 对标书《评论办公》凉水县（甘肃） | 本书融城（广西柳州） |
|------|---------------------------|-------------------|
| 地理 | 西北黄土高原 / 山地 | 桂中喀斯特山水 / 柳江 |
| 气候 | 干冷 | 湿热（湿冷的冬天） |
| 饮食 | 面食 / 牛羊肉 | 螺蛳粉 / 紫苏田螺 / 酸笋 |
| 方言 | 西北官话 | 桂柳话（西南官话） |
| 民族 | 汉族 | 汉 + 苗 + 侗 + 瑶 |
| 工业 | 几乎为零 | 重工业基因（五菱 / 柳钢 / 柳工） |
| 风俗 | 黄土风情 | 苗侗坡会 + 鼓楼 + 风雨桥 |
