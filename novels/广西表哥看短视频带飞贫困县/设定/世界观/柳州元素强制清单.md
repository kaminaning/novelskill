# 柳州元素强制清单（写正文时必须照本宣科）

> **核心原则**：本书地域内核 = 柳州（桂中），融城原型混合融水苗族自治县 + 三江侗族自治县 + 融安 + 柳城 + 鹿寨。
> 写正文时**必须**严格遵守本清单。任何与本清单冲突的用词/桥段/设定都视为"地域错误"，相当于把广东话写进东北戏，属于一票否决项。
>
> **资料来源**：百度百科桂柳话、维基百科柳州话、柳州市政府门户网、新华网、知乎专栏、广西文化和旅游厅官网、中国民族文化资源库、复旦大学医学人类学《"广西话"的"另类标准化"》、Foodaily、新浪柳州等。本清单引用日期 2026-05-17。
>
> **核心避雷三句话**：
> 1. **柳州话是西南官话，不是粤语**——"吃饭/喝茶/上街"，不是粤语的"食饭/饮茶/去街"。
> 2. **柳州不是沿海/不是石漠化山区/不是粤语圈**——柳州是桂中工业城市，工业产值占广西 1/4。
> 3. **融城内核 = 苗（融水）+ 侗（三江）+ 汉（柳城/鹿寨/融安）**，不是壮族主导。壮族是广西最大民族但柳州下辖区域中**汉族 + 苗 + 侗 + 瑶**占主导。

## 一、方言强制清单（最高优先级）

### 1.1 方言归属

| 项 | 内容 |
|------|------|
| 方言 | **桂柳话**（柳州话），属**汉语西南官话**，不是粤语 |
| 历史 | 明代北方军屯移民带来的官话，后融合粤语借词形成 |

> ⚠️ **难点警告**："搞咩""咩"在柳州话里有借入但发音 / 用法和粤语不同。柳州的"咩"作疑问助词偏静态（"走咩？"），不像粤语"做咩"那么有侵略感。写正文时**优先用"揍马 / 做嘛"**，少用"搞咩"，杜绝"搞乜嘢"。

### 1.5 主角口头禅库（强制限定）

**韦东（主角）必须高频使用的柳州口头禅**：

1. **"莫搞这些有的没的"**（别整这些没用的——委婉拒绝）
2. **"挨卵啦"**（糟了 / 完了——遇挫调侃）
3. **"几卵难搞"**（很难搞——感叹）
4. **"答你都困"**（懒得理你——怼人）
5. **"狗肉们"**（朋友们——招呼自己人）
6. **"算了卵"**（算了吧——服软 / 妥协）
7. **"够水"**（有胆识——夸人）
8. **"借个事好搞"**（这件事好办——揽事）
9. **"我醒水的"**（我清楚的 / 我警觉着呢——回应担心）
10. **"困得鬼都走"**（累得不行——抱怨）
11. **"你表哥我"**（自称——招牌口头禅，与书名《广西表哥》直接呼应；西南官话+粤语区通用的亲切自称，桂柳话区高频使用。例："你表哥我醒水的咯""你表哥我搞过这种事")
12. **"几个表弟"**（招呼比自己年轻的男性同辈——和"狗肉们"互补使用）

**绝对不允许出现的"假柳州"口头禅**：
- ❌ "得不得嘛"（南宁白话）
- ❌ "搞咩呢"（粤语 / 南宁）
- ❌ "傻仔 / 戆居"（粤语）
- ❌ "鸡乸 / 乜嘢"（粤语）
- ❌ "老俵"（江西话 / 湖南话——但"表哥/表弟/表妹"作为亲属称呼或自称是可以的）

> ✅ **关于"表哥"**：作为亲属称呼和**自称**（"你表哥我"），是桂柳话 / 广西全境通用用法，**不属于避雷词**。与书名《广西表哥看短视频带飞贫困县》直接呼应，应作为主角招牌口头禅。注意区分：**"老俵"是江西/湖南话才避雷**，"表哥/你表哥我"不是。

## 二、饮食强制清单

### 2.1 柳州招牌 7 件套（必须高频出现）

| 食物 | 关键特征 | 不可混淆点 |
|------|---------|-----------|
| **螺蛳粉** | 米粉 + 螺蛳骨牛骨熬底 + **酸笋**（灵魂！）+ 炸腐竹 + 木耳 + 花生 + 萝卜干 + 酸豆角 + 辣椒油。酸辣鲜爽烫 5 字诀 | ❌ 不是桂林米粉（卤汁干拌）、不是南宁老友粉（炒制酸笋猪肉粉）。**2021 年列国家级非遗** |
| **螺蛳鸭脚煲** | 鸭脚油炸 → 与田螺、酸笋、芋头、腐竹一起入螺蛳汤煲 | 柳州夜市之王，**配冰啤酒** |

## 三、风俗节庆强制清单

### 3.1 融水苗族系列坡会群（国家级非遗，融城核心招牌）

- **时间**：每年农历正月初三至十七，**每天一个坡会**（连续半个月）
- **核心活动**：芦笙踩堂、芦笙比赛、斗马、苗歌对唱、芦笙打同年、芒篙（戴面具的神灵化身）
- **芦笙柱**：杉木 10-20 米高，顶部雕锦鸡 / 鸟兽，中部水牛角，底部鼓模型衬托
- **百鸟衣**：芦笙队节日盛装，土黑亮布 + 五色绣花帕 + 白色鸟羽绒，男女款不同
- **踩堂舞**：分文打（轻柔三进一退）、武打（自由奔放男女合舞）
- **2006 年列国家级第一批非遗**

### 3.2 苗年

- **时间**：农历九 / 十 / 十一月**卯日（兔日）或丑日（牛日）**
- **活动**：踩鼓、跳芦笙、赛马、斗牛、斗鸟、游方（恋爱）
- **持续**：5-10 天

### 3.3 拉鼓节

- **归属**：融水苗 + 三江侗都有
- **频率**：大节 13 年一次（鼓 1 丈 3 尺）/ 小节 3 年一次（鼓 5 尺）
- **时间**：农历九 / 十月
- **核心**：泡桐木挖空 + 牛皮制鼓 → 鼓社双方拼劲抢鼓（类似拔河）→ 鼓后踩堂舞
- **送鼓**：拉鼓当天送鼓回岩穴或鼓棚，少数村寨**推下悬崖任其碎毁**

### 3.4 三江侗族文化

| 项目 | 关键 |
|------|------|
| **鼓楼** | 侗族村寨的中心 + 灵魂。木结构重檐建筑，中央火塘四周长凳。议事 / 典礼 / 迎宾 / 歌舞场所。三江有 230 多座。三江鼓楼"世界第一鼓楼" |
| **风雨桥 / 花桥** | **不用一钉一铆**，全杉木榫卯衔接。三江有 108 座。程阳永济桥、岜团桥列国家级文保。侗家人称"福桥"，月也节庆时桥上迎宾送客 |
| **侗族大歌** | **无伴奏无指挥的多声部民歌**。代表作《蝉之歌》《大山真美好》《装呆傻》《松鼠歌》。**"汉人有字传书本，侗族无字传歌声"** |
| **多耶（哆耶）** | 侗族集体歌舞，鼓楼前广场踩堂。柳州办过"中国侗族多耶节" |
| **月也** | 侗族**集体做客**习俗（A 寨请 B 寨全体来吃饭交流） |
| **百家宴 / 合拢饭** | 全寨每家一道菜并桌共食。长桌宴 |
| **行歌坐夜 / 坐妹 / 坐寨** | 青年男女恋爱社交（融水苗叫坐妹 / 坐寨，与黔东南"游方"对应） |

## 五、人文气质强制清单（主角性格基底）

### 5.1 柳州人 / 广西表哥的性格基线

1. **松弛感**：不内卷、不过度焦虑、嗦碗粉先（用户明确要求）
2. **直率**：嘴贱但心善，敢怼但讲理
3. **义气**：狗肉之间情义重于利益
4. **嗜辣嗜酸**：饮食偏好直接塑造性格——"无辣不欢"
5. **重工业基因**：祖辈三代柳钢 / 柳工 / 五菱的家庭很多，对"做实事"有信仰
6. **山水边缘 + 工业内核**：既有山水浪漫又有务实精神

### 5.2 与其他广西人 / 广东人的性格差异

| 维度 | 柳州 | 南宁 | 桂林 | 广东 |
|------|------|------|------|------|
| 节奏 | 中等偏快（工业城市） | 慢（养老 / 闲适） | 慢（旅游 / 文艺） | 极快（商业） |
| 性格底色 | 直率松弛 + 务实嘴贱 | 慢热 + 老友式相处 | 雅致 + 文人气 | 精明 + 商业 |
| 嗜好 | 螺蛳粉 + 紫苏炒田螺 + 牛杂 | 老友粉 + 酸嘢 + 烧烤 | 桂林米粉 + 啤酒鱼 | 早茶 + 烧腊 |
| 方言 | 桂柳话（官话） | 白话（粤语） | 桂林官话（更近湘） | 粤语 |
| 内核 | 工业 + 山水 | 政治 + 闲适 | 山水 + 旅游 | 商业 + 海洋 |

### 5.3 ❌ 主角性格避雷

| ❌ 严禁主角行为 | 原因 |
|---------------|------|
| 像精明算计的广东人那样反复打小算盘 | 不符柳州松弛感 |
| 像桂林文青那样满嘴诗词 | 不符柳州工业市井气质 |
| 像东北人那样大开大合咋咋呼呼 | 不符广西人内敛 |
| 满口"得不得嘛 / 友仔 / 食饭" | 是南宁白话，不是柳州话 |
| 张嘴就是"啤酒鱼 / 醋血鸭 / 老友粉" | 这些不是柳州菜 |
| 用粤语骂人 / 表情 | 柳州不说粤语 |
| 过度强调"穷山沟丢人"内核 | 柳州工业城市，没有"丢人"感，反而有"五菱出我家"的傲气 |

## 六、用法约束（写正文时的强制 SOP）

### 6.1 章节级别的"柳州含量"配额

> **每章必须至少出现 3 处柳州元素**（饮食 / 方言 / 风俗 / 工业 / 人文任选）。
> **每 5 章必须出现 1 处"避雷区"防错检查**——确保没有混入南宁 / 桂林 / 广东 / 闽南元素。

### 6.2 方言密度配额

- 主角对白：每段对白至少**穿插 1 个柳州方言词**（口头禅库 10 选 1，或方言词典任选）
- 配角对白：本地配角同上；外地配角不用方言
- 旁白叙述：避免大段方言，但**关键转折点用 1 句方言点睛**

### 6.3 饮食密度配额

- 每 3-5 章至少出现 **1 处吃喝场景**
- 吃喝场景必须是清单第二节内的食物
- **螺蛳粉**应该是主角"思考道具"——遇大事先嗦碗粉（已是项目惯例）
- **酸笋**的酸味（独特发酵气味）是柳州气味标识，每出现螺蛳粉必描写

### 6.4 风俗密度配额

- 第一卷（前 30 章）必须**有 1 次完整坡会 / 多耶 / 油茶宴 / 拉鼓节**场景
- 这场景必须**承担剧情功能**（不是装饰）——爆点 / 转折 / 共情都行

### 6.5 工业元素配额

- 主角科技树解锁可挂钩柳州真实产业链：螺蛳粉 / 五菱代工 / 柳工配件 / 柳钢小批量加工 / 新能源汽车零部件
- 配角阵营可设：五菱退休工程师、柳钢车间主任、螺蛳粉非遗传承人、侗族鼓楼掌墨师

### 6.6 排行榜元素的地域感

- 排行榜对手不是抽象数字，是**具体的全国其他县**：东北的、东南沿海的、华北的、西北的——通过对手反衬柳州独特性
- 主角看其他县报告时可吐槽："那是东北农场，跟我们桂中山区不一样" / "广东那个是沿海经济区，没法比"
