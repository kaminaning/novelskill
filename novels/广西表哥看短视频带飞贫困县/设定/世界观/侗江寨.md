# 侗江寨 — 世界观设定卡

> 三江侗族风格村寨虚化版 / 主角侗族血脉根据地 / 陈年酸笋发酵基地 / 侗族油茶根源地。

## 一、基本信息

- **名称**：侗江寨（弄崖乡下辖侗族村寨）
- **类型**：行政村（三江侗族风格山区村寨）
- **行政归属**：融城县弄崖乡（与弄崖村 + 苗排村 + 弄潭村同属弄崖乡）
- **民族构成**：**侗族 85% / 苗族 10% / 汉族 5%**（三江真实侗寨的虚化版）
- **核心特色**：3 座国家级文保鼓楼 + 1 座清代风雨桥 + 侗族大歌传承基地

## 二、地理特征

- 弄崖乡侗族村寨
- 与弄崖村 + 苗排村相邻
- 山地 + 杉木林 + 鼓楼 + 风雨桥
- 远眺融江白光

## 三、人文与历史

### 侗族文化（柳州元素强制清单 3.4 节合规）：

- **侗族 85%**——三江真实侗寨的虚化呈现
- **3 座国家级文保鼓楼** + 1 座清代风雨桥（参照三江有 230+ 座鼓楼 + 108 座风雨桥）
- **侗族大歌传承基地**（无伴奏无指挥的多声部民歌）
- **侗族油茶（打油茶）**：又叫"侗族咖啡"——茶叶炒香 + 加水煮 + 加糯米 / 花生 / 葱花 / 盐 / 姜
- **酸笋发酵**：18 个月发酵的"陈年酸笋"是螺蛳粉灵魂（柳州元素强制清单 2.1 节）

### 关键人物：

- **龙阿婆**：71 岁 / 苗绣老艺人 / 嫁到苗排村前是侗江寨人（侗族）/ 嫁过来后成为弄崖乡"侗苗融合"血脉的代表
- **吴守德**：71 岁 / 侗江寨本地农技手 / 第 19 章弄崖村农技培训站师资

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **3 座国家级文保鼓楼**：木结构重檐建筑，中央火塘四周长凳，议事 / 典礼 / 迎宾 / 歌舞场所
2. **1 座清代风雨桥**：与弄崖村风雨桥同款工艺（杉木榫卯无一钉一铆）—— 弄崖村风雨桥实质是侗江寨侗族建筑文化通过母亲血脉传入的
3. **龙阿婆地窖**：放陈年酸笋的地下储存室（**18 个月发酵酸笋的源头**）
4. **酸笋坛子**：贴着"2024 年 10 月" / 土黄色细绳缠口（第 9 章主角拿 10 公斤陈年酸笋的具体场景）
5. **风雨桥边的夜市螺蛳粉摊**：本地侗族 50 岁老板娘 / 与母亲熟识 / 第 9 章主角回程嗦的夜市粉摊
6. **杉木林 + 鼓楼前广场**：侗族大歌彩排场地
7. **笋干棚**：第 20 章主角带 22 名学员到侗江寨笋干棚学发酵工艺 + 张秀芳副局长在杉木门槛上的 23 分钟谈话场地

## 五、出场章节与剧情功能

- **第 9 章**：主角去三江侗寨乡找龙阿婆拿 10 公斤陈年酸笋 / 龙阿婆送主角"亲手绣的护腰带"（侗江寨实际是侗寨乡的村寨之一，弄崖乡内部地理设置）
- **第 20 章**：主角带 22 名学员到侗江寨笋干棚学发酵工艺 / 张秀芳副局长在杉木门槛上 23 分钟谈话场地（**正面登场**）
