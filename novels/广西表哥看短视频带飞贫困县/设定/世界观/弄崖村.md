# 弄崖村 — 世界观设定卡

> 主角故乡 / 全卷主舞台 / 苗汉混合山区村寨。内核混合融水苗族自治县 + 融安县山区风格。

## 一、基本信息

- **名称**：弄崖村
- **类型**：行政村（苗汉混合山区村寨）
- **行政归属**：广西柳州融城县弄崖乡（下辖 4 个自然屯）
- **民族构成**：**苗族 65% / 汉族 25% / 侗族 10%**
- **海拔**：800-1200 米
- **气候**：亚热带季风湿润气候，冬季湿冷 0-8℃ 无暖气，**潮湿**
- **常住人口**：980 人（户籍 2300+ 人，青壮年外出务工率 65%+）

## 二、地理特征

- **位置**：融城县西北山区
- **到县城距离**：28 公里盘山路 / 47 个弯
- **山系**：苗岭余脉 + 桂中喀斯特峰林峰丛地貌
- **地质**：石灰岩 + 风化层 + 地下溶洞分布（1957 年柳钢勘探队测过 17 处溶洞，主角第 17 章通过覃成福手绘图获取这一资料）
- **河流**：融江支流穿村而过
- **交通**：盘山公路 28 公里 47 弯 / 五菱小卡 + 桑塔纳 + 五菱宏光是主要交通工具

## 三、人文与历史

### 民族文化构成：

- **苗族（65%）**：主要为融水苗族风格 / 节日含拉鼓节 / 坡会 / 苗年 / 跳坡 / 苗绣传承
- **汉族（25%）**：1958 年柳钢调来的工人后代 + 1965 年返村的工人家庭（如主角爷爷韦保根）
- **侗族（10%）**：来自三江侗寨乡的婚姻人口（如主角外婆龙阿婆从侗江寨嫁到苗排村，再生下主角母亲覃秀英嫁到弄崖村汉族韦家）

### 历史背景：

- **1958 年**：柳钢成立，部分山区青壮年被招进柳钢
- **1965 年**：主角爷爷韦保根从柳钢调回弄崖村（带回 LG-187 小型锻造冲压机，挂在吊脚楼下）
- **1976 年**：村东头生产队仓库建成（即后来主角作坊地址）
- **1990 年代**：柳钢改制 / 大批工人下岗 / 弄崖村 + 周边村大量男人去东莞 / 广东打工
- **2008 年**：父亲韦海军从柳钢下岗 / 去东莞打工 / 半年后失联（L01 父亲失踪 18 年的起点）
- **2026 年**：主角被裁返乡 / 系统激活 / 弄崖村作坊建设

### 主要风俗：

- 苗族系列坡会群（融水苗族国家级非遗）
- **拉鼓节**（3 年小节 / 13 年大节，1998 年外公和父亲最后一次拉过，2026 年主角第 13-15 章复活）
- 苗绣传承
- 油茶（侗族传统，由侗江寨血脉传入）
- 香糯种植 + 紫苏 + 笋干 + 酸笋发酵
- **村群微信**：200+ 人（村主任韦德才作为村群管理人，第 2 章发"被裁回乡摆烂"消息）

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **村口褪色牌坊**：写着"**全国 GDP 后 10% 县——融城欢迎您**"（红字已褪色）—— 第 1 章主角桑塔纳过弯时看到的反讽视觉锚
2. **盘山公路 47 个弯**：从弄崖村到县城 28 公里，第 18 个弯是主角的"看融江独白位"（第 21 / 30 章呼应）
3. **村东头废弃 1976 年生产队仓库**：800 平米屋顶塌一半但承重柱完整 + 70 年代混凝土地基坚固——主角作坊地址，故意选在韦德才家斜对面让对手"每天看见"
4. **吊脚楼**：木板墙 + 廊下 + 杉木结构 + 地下室藏旧工具（爷爷辈柳钢锻造机 LG-187 / 1957 年勘探地质图等）
5. **村口清代风雨桥**：木瓦黛色 + 全杉木榫卯无一钉一铆 / 桥下融江支流伸手可触水 / 100 年来抗住百场雨 / 主角第 30 章卷末 50 分钟独坐场地
6. **祠堂前青石板广场**：村民代表大会场地 + 主角第 22 章蹲会场地（五人没椅子没桌子蹲在青石板上）
7. **弄崖村小卖部**：阿贵家开的，是村中信息枢纽
8. **村中阿贵爸韦满堂家**：与韦德才家形成"挺主角 vs 反主角"两户中心
9. **山脚 20 亩荒地**：主角第 22 章自建酸笋发酵基地（18 度恒温 28 天发酵周期）
10. **吊脚楼地下室旧柜子**：覃成福家（M02 工业血脉伏笔的具体物理位置）

## 五、产业 / 经济

- **传统产业**：香糯种植 / 紫苏 / 笋干 / 油茶 / 苗绣 / 酸笋发酵
- **2026 年新增产业**：螺蛳粉作坊（弄崖村作坊）/ 苗绣礼盒 / 短视频运营 / 农技培训站 / 酸笋发酵基地 / 端午限定礼盒
- **外出务工人口**：78 户在外打工（深圳 / 东莞 / 广州 / 柳州市区）
