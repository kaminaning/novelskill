# 远辉鑫源楼 — 世界观设定卡

> 苏氏远辉实业集团总部 / 主反派大本营 / "工业老板大本营"视觉地标 / 第 29 章鸿门宴场地。

## 一、基本信息

- **名称**：远辉鑫源楼
- **类型**：商业地产 / 写字楼（融城镇地标性建筑）
- **行政归属**：融城县融城镇（县城中心商务区）
- **建造年代**：2018 年建成
- **楼层**：18 层（顶层 28 楼整层是苏氏家族总部）
- **业主 / 物业方**：远辉商业地产（苏氏远辉系子公司）

## 二、地理特征

- **位置**：融城镇县城中心
- **观景**：落地玻璃外是融江穿城而过
- **周边**：县城核心商务区 + 老街附近
- **交通**：县城主干道沿线

## 三、人文与历史

### 苏氏家族总部：

- **28 楼整层** = 苏氏远辉实业集团总部
- 苏明远的董事长办公室
- 18 米长会议桌的董事会议室
- 落地玻璃 + 融江景观
- 墙上挂着远辉实业旗下 5 家公司的成绩单

### 与"工业柳州"内核的对照：

- **远辉鑫源楼"奢华金融范"**（深灰西装 + 浪琴名匠手表 + 南珠手串 + 18 米长会议桌）
- **VS 弄崖村吊脚楼"工业市井气"**（光脚踩田埂 + 大白纸督办墙 + 螺蛳粉灶台）
- 这种对照是本书"新秩序 vs 旧秩序"主题的视觉化

## 四、关键场景 / 物件细节（写正文时可直接调用）

### 28 楼董事会议室：

1. **18 米长会议桌**：主反派阵营的"权力桌"，苏明远食指敲桌面三下的标志性动作场地
2. **落地玻璃 + 融江景观**：与弄崖村吊脚楼"窗外能看见祠堂前青石板广场"形成对比
3. **墙上 5 家公司成绩单**：远辉物流 / 远辉商超 / 远辉文创 / 苏家茶园 / 远辉商业地产
4. **黑色定制西装 + 浪琴名匠手表 + 南珠手串**：苏明远的标志性配件（F01 伏笔）
5. **87 页厚资料**：第 20 章关于主角的厚厚资料（信息掌控 A+ 级实力展示）

### 三楼包间（第 29 章鸿门宴场地）：

6. **三楼包间**：与 28 楼董事会议室的"开会式权力空间"不同，是"广西式商务宴"的私密空间
7. **桌上一桌广西式商务宴**（地域饮食合规）：
   - **紫苏炒田螺**（柳州招牌 / 不是九层塔！）
   - **螺蛳鸭脚煲**（柳州夜市之王）
   - **柳钢老车间风味牛杂火锅**（柳州独有）
   - **三江侗酸鱼**（三江侗族特色）
   - **苗山香猪**（融水苗山特色）
   - **一壶融水家酿米酒**（融水苗族特色）
   - **一壶柳州本地紫苏茶**（主角点的非酒精饮品）

### 顶楼天台：

8. **顶楼天台**：第 29 章末苏明远抽烟 + 打电话给爸爸苏建国的场地（"爸，这小子比我们想的难搞" / "27 天他熬不过"）

## 五、出场章节与剧情功能

- **第 8 章 ⭐**：远辉鑫源楼 28 楼 / 苏明远早会 / 反派阵营立体化登场（**反派大本营首次亮相**）
- **第 20 章**：苏明远 18 楼董事长办公室 + 87 页厚资料 + 4 个商标抢注布局
- **第 29 章 ⭐**：三楼包间鸿门宴 + 顶楼天台苏明远打电话苏建国（**全卷反派最高潮场景**）

## 六、剧情功能

### 前 30 章承担：

- 主反派大本营的视觉地标
- "工业老板大本营 vs 民间作战指挥部"对照的反派一端
- 苏氏家族议事 + 战略布局的物理空间
- 鸿门宴 + 反派轻视段 + 反派阵营立体化的核心舞台
- 苏明远敲桌面三下 / 南珠手串 / 浪琴名匠等反派识别符的展示场

### 长期弧线：

- 第二卷：远辉鑫源楼成为主角全面对抗的对象
- 第三卷：苏明远倒台后远辉鑫源楼易主或被纪委查封
