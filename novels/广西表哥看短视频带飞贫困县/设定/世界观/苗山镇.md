# 苗山镇 — 世界观设定卡

> 融水大苗山的虚化版 / 主角文化树 T1 拉鼓节复活地 / 国家级非遗坡会群核心承办地。

## 一、基本信息

- **名称**：苗山镇
- **类型**：乡镇 / 苗族传统文化核心区
- **行政归属**：融城县（融城原型混合了融水苗族自治县 / 三江侗族 / 融安 / 柳城 / 鹿寨）
- **民族构成**：**苗族 78% + 汉族 15% + 其他 7%**（融水大苗山的虚化版）
- **海拔**：800-1500 米（山区）
- **到弄崖村距离**：50 公里盘山路（与弄崖村同属弄崖乡北部山区方向）

## 二、地理特征

- **位置**：融城县北部山区
- **山系**：融水苗岭支脉
- **地形**：高山盆地 + 杉木林 + 梯田
- **核心场所**：芦笙堂 + 坡会广场 + 斗马场 + 苗排村（外公覃老六所在）

## 三、人文与历史

### 民族文化（融水苗族系列坡会群核心区）：

- **拉鼓节**（柳州元素强制清单 3.3 节）：
  - 3 年小节（鼓 5 尺）/ 13 年大节（鼓 1 丈 3 尺）
  - 时间：农历九 / 十月
  - 核心：泡桐木挖空 + 牛皮制鼓 → 鼓社双方拼劲抢鼓（类似拔河）→ 鼓后踩堂舞
  - **1998 年**：外公覃老六和父亲韦海军最后一次拉过（M02 工业血脉伏笔时间锚）
  - **2026 年**：本来要办临时取消（队伍凑不齐 / 老的拉不动 / 年轻的全在广东）→ 主角第 13-15 章接系统任务复活
- **苗年**（农历九 / 十 / 十一月卯日或丑日 / 持续 5-10 天）
- **苗族系列坡会群**（国家级非遗 / 时间农历正月初三至十七 / 每天一个坡会连续半个月）
- **芦笙文化**：芦笙队 + 芦笙柱
- **斗马**（融水苗族国家级非遗）
- **苗绣传承**

### 历史背景：

- 1990 年代乡镇企业辉煌期（与北河镇类似有过鞋厂等小厂）
- 2000 年代后年轻人外流严重
- 2026 年：拉鼓节 + 苗绣 + 文旅成为镇政府"待激活"资源

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **芦笙堂**：详见 [苗山镇芦笙堂.md](苗山镇芦笙堂.md)（外公覃老六劈鼓木 + 拉鼓节场所）
2. **坡会广场**：拉鼓节 + 踩堂舞 + 直播大爆点的场地（青石板地面 + 周围杉木林 + 月光环境）
3. **斗马场**：国家级非遗活动场地

5. **苗排村**：苗山镇辖下苗族村寨之一（覃家外公外婆所在，详见 [苗排村.md](苗排村.md)）
6. **盘山公路**：50 公里弯绕从弄崖村到苗山镇
7. **本地特产**：紫苏 / 香糯 / 笋干 / 苗绣 / 紫苏籽（老吴家田螺养殖户 64 岁壮族也住苗山镇）
