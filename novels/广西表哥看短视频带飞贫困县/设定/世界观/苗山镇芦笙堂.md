# 苗山镇芦笙堂 — 世界观设定卡

> 外公覃老六劈鼓木的工作场所 / 拉鼓节传统场所 / 主角第 13-15 章文化树 T1 副本核心场地。

## 一、基本信息

- **名称**：苗山镇芦笙堂
- **类型**：苗族传统芦笙文化场所（含芦笙队训练 + 鼓木加工 + 拉鼓节备料）
- **行政归属**：苗山镇苗排村（详见 [苗山镇.md](苗山镇.md)）
- **建造年代**：未明（推测明清始建，多次翻修）
- **核心功能**：芦笙队彩排 + 拉鼓节备料 + 苗绣展示 + 老人聚会

## 二、地理特征

- **位置**：苗山镇苗排村
- **离弄崖村距离**：22 公里盘山路
- **周边**：杉木林 + 山坡 + 田垄
- **建筑形态**：木结构 + 杉木横梁顶 + 通风开放式（不完全封闭）

## 三、人文与历史

### 拉鼓节传承场所：

- **1998 年**：外公覃老六和父亲韦海军最后一次在这里拉过鼓（M02 工业血脉伏笔的时间锚 + 父亲失踪 18 年伏笔的前夜场景）
- **1998-2026 年**：芦笙堂边的鼓棚因为山猪拱塌 + 没人维护，逐渐荒废
- **2026 年第 13 章**：外公在芦笙堂边的杉木林劈鼓木 → 主角接下"6 天搞回拉鼓节"任务

### 芦笙文化：

- **芦笙柱**：10-20 米高杉木，顶部雕锦鸡 / 鸟兽，中部水牛角，底部鼓模型衬托（柳州元素强制清单 3.1 节）
- **芦笙队**：约 14-18 人，传统配置
- **百鸟衣**：苗族盛装（土黑亮布 + 五色绣花帕 + 白色鸟羽绒）

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **杉木横梁顶**：开放式建筑，主角第 13 章蹲在鼓木堆边看到的"芦笙堂顶部杉木横梁"
2. **杉木林**：芦笙堂边的杉木林，外公在里面劈鼓木
3. **鼓木堆**：泡桐木堆放区域，等待加工成拉鼓节用鼓
4. **石匠凿 + 劈刀**：外公的工具（与主角第 17 章覃成福石匠凿形成传统手艺线）
5. **苗绣作业区**：外婆龙阿婆71 岁苗绣老艺人的工作区（在屋内打苗绣）
6. **18 年没修的鼓棚**：芦笙堂附近 / 已经被山猪拱塌一半
7. **传统乐器陈列**：芦笙 / 木鼓 / 铜鼓 / 锣
8. **百鸟衣陈列**：外婆龙阿婆手绣的传统苗绣盛装

### 第 13 章核心场景细节：

- **外公穿黑色对襟苗服**：拉鼓节准备时的传统服饰
- **外公的劈刀靠在鼓木上**：传统手艺人的"沉思动作"
- **外婆从屋里出来塞一块烧炙到主角手里**：融水 + 融安特色（猪网油包馅 + 油炸 + 炭火炙烤）
- **外公讲 1998 年"我跟你阿爸最后一次拉过"**：L01 父亲失踪 18 年长线伏笔的本章正面触发

## 五、出场章节与剧情功能

- **第 13 章 ⭐**：主角骑五菱小卡 22 公里盘山到苗排村芦笙堂 / 找外公覃老六 / 接下"6 天搞回拉鼓节"任务 / 外公讲 1998 年父亲最后一次拉鼓的故事（**父亲软肋首次正面触发**）
- **第 14 章**：芦笙柱筹备 / 鼓木鼓皮筹备
- **第 15 章 ⭐**：苗山镇坡会广场凌晨四点搭台 / 与芦笙堂联动（拉鼓队从芦笙堂出发到坡会广场）

## 六、剧情功能

### 前 30 章承担：

- 父亲软肋首次正面触发的物理场景（外公讲 1998 年那段话）
- 系统文化树 T1 副本的关键空间
- 苗族非遗（拉鼓节 / 芦笙 / 苗绣）的视觉化展示
- 外公覃老六 / 外婆龙阿婆作为主角母系家族支柱的物理依托
- 共情角色阶梯第三层（苗山镇龙阿婆芦笙非遗老人 101-200 章爆款）的承接地

### 长期弧线：

- 第二卷：芦笙堂复兴 / 成为"苗山镇芦笙堂直播基地"
- 第三卷：芦笙堂传承下一代苗族青年 → 文化产业链完整闭环
