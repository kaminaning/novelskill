# 弄崖村吊脚楼 — 世界观设定卡

> 主角家堂屋 / 全卷"作战指挥部" / 母亲油茶店所在 / 工业血脉伏笔藏匿地。

## 一、基本信息

- **名称**：韦家吊脚楼（同时容纳主角家堂屋 + 母亲油茶店）
- **类型**：苗汉风格山区吊脚楼（木板墙 + 杉木结构 + 廊下 + 地下室）
- **行政归属**：弄崖村村口斜坡上
- **建造年代**：未明确（推测 1970-80 年代）
- **所有人**：韦家三代——爷爷韦保根 / 父亲韦海军 / 主角韦东
- **使用功能**：
  - 一楼：母亲油茶店"覃记油茶 1998"店面
  - 二楼：堂屋 + 厨房 + 主角卧室
  - **地下室**：放工具 / 旧物 / 柳钢锻造机（M02 伏笔关键空间）

## 二、地理特征

- 弄崖村村口斜坡上
- 楼后是杉木林 + 山地
- 楼前看向村中道路 + 远眺融江白光
- 紧邻祠堂前青石板广场

## 三、人文与历史

### 三代柳州人血脉：

- **第一代（1958-1965）**：爷爷韦保根 / 柳钢工人 / 1965 年带 LG-187 小型锻造冲压机回弄崖村
- **第二代（1965-2008）**：父亲韦海军 / 柳钢工人 / 1990 年代下岗 / 2008 年去东莞失踪
- **第三代（1998 - 现今）**：主角韦东 / 28 岁 / 1998 年快乐期 → 2008 年家从富足变贫困 → 2026 年返乡

### 母亲油茶店"覃记油茶 1998"：

- 详见 [母亲油茶店.md](../势力/母亲油茶店.md)
- 招牌挂在一楼店面外，1998 是父亲在柳钢工作最后一年（同年开油茶店）

### 堂屋"作战指挥部"：

- 第 6 章后主角把八仙桌挪开 + 49 张大白纸全部铺满墙和地板
- 红色马克笔写"作战图"三个字（很大）
- 后期成为弄崖村作坊的"灵魂室"

## 四、关键场景 / 物件细节（写正文时可直接调用）

### 一楼（油茶店店面）：

1. **木牌"覃记油茶 1998"**：边缘已被柴火熏黑
2. **斑驳的木板墙**：第 2 章 18 秒视频背景
3. **半旧的茶炉**：母亲打油茶 18 年的老物件
4. **墙上 2008 年全家福**：父亲韦海军失踪前最后一张全家福
5. **母亲的"百鸟苗绣"**：18 年前结婚时绣的最高难度纹样

### 二楼（堂屋 + 厨房 + 卧室）：

6. **堂屋八仙桌 + 两条长板凳**：传统农村摆设
7. **2009 年长虹电视**：墙角，开过又落灰
8. **49 张大白纸 + 红色马克笔**：第 6 章后主角的作战图墙
9. **绿色"评论办公本"**：从县城新华书店买的，60 页规格小学生作文本款式
10. **墙角的三盒云南白药膏 + 骨痛灵酊 + 甲钴胺**：母亲腰椎药物（视觉上的"母亲软肋"锚点）

### 地下室（M02 伏笔关键空间）：

11. **柳钢 1958 锻造车间 LG-187 小型锻造冲压机**：爷爷 1965 年从柳钢带回来的，大概 1 米 5 高 / 落满灰 / 上面挂着发黑的铜铭牌"柳钢 1958 锻造车间 LG-187"
12. **1957 年柳钢勘探队手绘地质图**（藏在覃成福家吊脚楼地下室 / 与主角家同一类吊脚楼）：折了八层的牛皮纸 + 一把锈得几乎认不出的石匠凿 + 一本 1957 年柳钢工人手册
13. **1980 年代柳钢小型石灰窑**：被山猪拱过但还能认（覃成福家地下室）
14. **苗绣布盖的工具**：传统手艺人的工具传承

### 楼前楼后视野：

15. **楼前视野**：能看见祠堂前青石板广场 + 弄崖村中道 + 远处融江白光
16. **楼后**：杉木林 + 山地

## 五、出场章节与剧情功能

- **第 1 章**：主角桑塔纳到达弄崖村首夜（楼前牌坊"全国 GDP 后 10% 县"反讽视觉）
