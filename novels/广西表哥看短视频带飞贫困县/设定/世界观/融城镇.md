# 融城镇 — 世界观设定卡

> 融城县城所在地 / 政商谈判核心舞台 / 民营企业总部聚集区 / 主反派与体制盟友所在地。

## 一、基本信息

- **名称**：融城镇
- **类型**：县城（行政区中心）
- **行政归属**：融城县
- **海拔**：约 200 米（与山区弄崖村 800-1200 米形成显著海拔差）
- **常住人口**：约 6-8 万人（县城规模）
- **核心定位**：融城县政治、商业、文化中心

## 二、地理特征

- **位置**：融城县中心位置
- **河流**：融江穿城而过（柳江支流）
- **到弄崖村**：28 公里盘山路 47 个弯
- **附近**：120 公里盘山高速 + 国道连接柳州市区
- **公交**：每日 6 班至柳州

## 三、人文与历史

### 行政架构核心区：

- 县委 / 县政府 / 县人大 / 县政协（四大班子）
- 县政府二号楼 + 三号办公区（详见 [融城县政府.md](../势力/融城县政府.md)）
- 县纪委办公区
- 县文旅局 / 县农业农村局 / 县扶贫办

### 商业核心区：

- **远辉鑫源楼**（详见 [远辉鑫源楼.md](远辉鑫源楼.md) / 苏氏家族总部）
- 县物流园（融江物流公司·融城调度站）
- 县商业街 + 商超 + 餐饮聚集区

### 老街区：

- **融城老街**：传统市井气息 + 螺蛳粉摊 + 老茶馆 + 老字号
- **龙城阁茶馆**（详见 [龙城阁茶馆.md](../势力/龙城阁茶馆.md) / 老李柳钢退休车间主任）
- **莫阿姨粉摊**（第 29 章末主角嗦粉时打听苏家底细 / 老板娘开店 23 年）
