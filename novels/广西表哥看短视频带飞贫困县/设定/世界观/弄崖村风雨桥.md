# 弄崖村风雨桥 — 世界观设定卡

> 卷末灵魂场景 / 主角许誓地 / 100 年抗百场雨的清代风雨桥 / 侗族建筑文化在弄崖村的传承。

## 一、基本信息

- **名称**：弄崖村风雨桥（村民也称"通桥"或"桥头"）
- **类型**：清代风雨桥（木瓦黛色 + 全杉木榫卯无一钉一铆）
- **行政归属**：弄崖村村口
- **建造年代**：清代（确切年份未明，约 100+ 年历史）
- **跨越**：融江支流（桥下水流，伸手可触水）
- **建造工艺**：侗族传统风雨桥工艺（与三江侗族 108 座风雨桥同款手艺）

## 二、地理特征

- 位置：弄崖村村口（与吊脚楼方向相反，村中道路尽头）
- 跨越：融江支流（柳江上游分支）
- 周边：风雨桥栏边 + 山坡 + 远眺融水山系
- 桥下水流：从融水苗山发源，往柳州市区流，往珠江流，往南海流

## 三、人文与历史

### 文化渊源：

- **侗族建筑文化**（柳州元素强制清单 3.4 节）：
  - 全木结构，无一钉一铆
  - 杉木榫卯衔接，整桥靠咬合
  - 三江有 108 座风雨桥（程阳永济桥、岜团桥列国家级文保）
  - 侗家人称风雨桥为"福桥"
- **弄崖村风雨桥**：是侗族建筑文化通过母亲覃秀英血脉（外婆龙阿婆侗江寨苗 + 母亲嫁汉家）传入弄崖村的具象
- **存在意义**：村民乘凉 + 躲雨 + 月也节庆迎宾送客

### 主角许誓地：

第 30 章卷末灵魂场景——主角一个人坐在桥栏上 50 分钟，对融江许下"1 年后融城摘掉贫困县帽子"的第一个誓。

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **木瓦黛色**：木瓦在 100 年雨水下变成黛色（不是黑色，是带青的深灰）
2. **全杉木榫卯无一钉一铆**：是清代侗族传统工艺的活样本
3. **桥栏**：可坐人 + 可斜倚 + 可伸手摸水
4. **桥下融江支流**：水流可见 + 凌晨 1 点白光 + 夏天能下水 + 冬天结薄冰
5. **桥顶覆盖**：木瓦 + 茅草顶（防雨）
6. **桥两端**：村口入口 + 村中道路尽头
7. **桥的隐喻**："桥上没钉子没螺栓全靠榫卯咬合 / 100 年来 100 场雨桥还在" = 主角"弄崖村也会还在"的隐喻锚点
8. **桥下水流的隐喻**："这条江从融水苗山发源，往柳州市区流，往珠江流，往南海流 / 18 年前父亲沿着这条江坐火车去了东莞" = 父亲失踪线索的隐喻锚点

## 五、出场章节与剧情功能

- **第 11 章**：风雨桥头看融江"白光"——主角七天战报压力下的内心独白第一次
- **第 16 章**：风雨桥头给外公电话联系喀斯特石匠

- **第 24 章**：龙城阁茶馆"看融江"内心独白（与风雨桥独白形成桂中三连之一）

## 六、剧情功能
