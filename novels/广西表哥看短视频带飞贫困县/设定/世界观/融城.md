# 融城（虚构县）—— 世界观核心

> 融城是本书虚构的广西柳州下辖县。内核混合了**融水苗族自治县 + 三江侗族自治县 + 融安 + 柳城 + 鹿寨** 5 个真实柳州下辖区域的特征，避免与现实一一对应。
> **核心约束**：所有方言 / 饮食 / 风俗 / 工业 / 人文必须符合 `设定/世界观/柳州元素强制清单.md`。

## 五、主要乡镇 / 村寨

### 5.1 弄崖乡（主角故乡）

- **位置**：融城县西北山区，距县城 28 公里盘山路
- **海拔**：800-1200 米
- **主要村寨**：弄崖村（主角家）/ 弄潭村 / 苗排村 / 侗江寨
- **民族**：苗族 65% / 汉族 25% / 侗族 10%
- **产业**：香糯种植 / 紫苏 / 笋干 / 油茶
- **风俗**：每年正月**初九有坡会**（融水系列坡会群之一）
- **常住人口**：980 人（户籍 2300+ 人，青壮年外出打工）

### 5.2 苗山镇

- 融水真实大苗山的虚化版
- 民族：苗族 78%
- 特色：芦笙堂 + 斗马场 + 苗绣

### 5.3 侗寨乡

- 三江真实侗寨的虚化版
- 民族：侗族 85%
- 特色：**3 座国家级文保鼓楼** + 1 座清代风雨桥 + 侗族大歌传承基地

### 5.4 北河镇

- 县城周边平地镇，汉族为主，米粉店 + 螺蛳粉店密集
- 1990 年代有过乡镇企业辉煌（鞋厂 / 五菱配件小厂），90 年代末倒闭

## 八、本地经济命脉（金手指可挂钩）

### 8.1 螺蛳粉产业链（柳州顶级名片）

- 全县螺蛳粉店：427 家（密度全县第一）
- 螺蛳粉工厂：0 家（全部在柳州市区）
- **机会**：袋装螺蛳粉产业链中，**酸笋发酵 / 干米粉 / 螺蛳熬制汤包**可下沉到县级。融城气候适合酸笋发酵
- 主角科技树解锁路径：T1 螺蛳粉作坊 → T2 半自动化生产线 → T3 全产业链

### 8.2 五菱 / 柳工配件代工

- 县内现有 2 家小型代工厂：北河五菱座椅泡棉厂（剩 40 人）/ 北河柳工螺栓厂（剩 60 人）
- **机会**：承接五菱 / 柳工微小零部件订单，主角可通过短视频拉来订单
- 主角科技树解锁路径：T2 零部件代工 → T3 新能源汽车配件

### 8.3 苗侗文旅

- 现状：年游客 4 万人次（仅坡会期间集中）/ 民宿 12 家 / 收入低
- **机会**：芦笙堂 + 鼓楼 + 风雨桥 + 坡会 + 油茶 是抖音爆款素材
- 主角科技树解锁路径：T1 短视频内容化 → T2 文旅小镇 → T3 全国 4A 景区

### 8.4 香糯 / 紫苏 / 茶叶 / 金桔

- 优质农产品但缺品牌
- 主角科技树解锁路径：T1 产业品牌化 → T2 电商 → T3 一二三产融合

## 九、读者代入感锚点

写正文时**必须经常呈现**的"融城"画面：

1. **嗦粉**：清晨县城十字街老螺蛳粉店冒着酸笋蒸汽，5 块钱一碗加蛋
2. **风雨桥**：侗寨乡村口的清代风雨桥，木瓦黛色，夏天乘凉冬天躲雨
3. **鼓楼**：侗寨夜晚火塘 + 长凳 + 老人讲古
4. **芦笙柱**：苗山镇坡会广场的杉木芦笙柱，红绸缠绕
5. **柳江支流融江**：穿县城而过，岸边老榕树
6. **五菱小卡**：每个赶集日上的运货主力，挤满后斗的红薯 / 笋干 / 鸡笼
7. **盘山公路**：从县城到弄崖村 28 公里 / 47 个弯，会晕车
8. **油茶**：侗寨乡早晨家家炒茶香气，米花 + 花生 + 葱花
9. **柳工螺栓厂烟囱**：北河镇唯一的现代化痕迹，已停产 80%
