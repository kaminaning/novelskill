# 县城科技树系统 + 全国万人排行榜 + 评论值→DP 三合一金手指

> 本书的核心金手指。**沿用既有 县城科技树 项目的科技树+排行榜 双系统**，**新增评论值→DP 转化机制**（对标《我一个县长，看评论办公怎么了》的评论办公核心）。

---

## 一、系统全名

**「县城逆袭科技树·全国挑战赛」**

绑定时系统播报：

> 「检测到宿主满足条件：户籍县城 GDP 全省后 10% / 宿主年龄 35 岁以下 / 个人 GPA 大于 0 / 返乡创业意愿值 ≥ 80%。」
> 「绑定成功。当前宿主：广西柳州融城县弄崖村韦东。」
> 「全国挑战赛初始排名：#9527 / 10000。」
> 「⚠️ **季度末末位 100 名淘汰**。下次结算：90 天后。」

## 二、三层结构

### 2.1 第一层：科技树（基础层 - 提供升级蓝图）

#### 科技树形态

```
┌─【一级根】融城县——「县城综合发展科技树」
│
├─ 产业树（核心）
│   ├─ T1 产业 → T2 产业 → T3 产业 → T4 产业
│   │   ├─ 螺蛳粉作坊 → 自动化生产线 → 全产业链 → 行业标准
│   │   ├─ 紫苏种植 → 紫苏深加工 → 中药提取 → 健康食品
│   │   ├─ 香糯种植 → 糯米精加工 → 糯米衍生品 → 米业王国
│   │   ├─ 笋干加工 → 酸笋发酵厂 → 酸笋出口 → 国际品牌
│   │   ├─ 茶园 → 茶叶精制 → 油茶产业化 → 健康茶饮
│   │   └─ 五菱配件代工 → 新能源零部件 → 整车配套 → 产业园
│   
├─ 基建树
│   ├─ 村道硬化 → 县道升级 → 高速接线 → 高铁站
│   ├─ 自来水 → 污水处理 → 综合管廊 → 智慧水务
│   └─ 4G 升级 → 5G 全覆盖 → 数据中心 → 算力枢纽
│
├─ 文化树
│   ├─ 苗侗非遗保护 → 非遗体验馆 → 文旅小镇 → 4A 景区
│   ├─ 坡会数字化 → 文创周边 → 沉浸式演艺 → 国际节庆
│   └─ 短视频内容 → MCN 矩阵 → 网红孵化 → 流量小镇
│
├─ 人才树
│   ├─ 农技培训站 → 职业学校 → 应用型大学分校 → 研究院
│   └─ 返乡引才 → 创业孵化 → 风投基金 → 上市公司

└─ 民生树
    ├─ 医疗下沉 → 县医院升级 → 三甲医院 → 区域医疗中心
    ├─ 教育扶持 → 学校翻新 → 名校分校 → 教育高地
    └─ 养老服务 → 老有所养 → 银发经济 → 国家级试点
```

#### 解锁规则

- 每个 T1 节点：50-100 DP（开发点数）
- 每个 T2 节点：200-500 DP
- 每个 T3 节点：1000-3000 DP
- 每个 T4 节点：5000-10000 DP
- **前置条件**：必须先点亮上一级节点
- **跨树解锁**：达到一定节点数后可激活跨树协同（如产业 T2 + 基建 T2 → 解锁「智慧农业」混合节点）

#### 解锁后获得物

- 「蓝图」：详细到工艺参数、设备清单、操作流程的完整文档（自动写入主角记忆）
- 「专家咨询通道」：解锁后 30 天内可向系统提问 3 次（系统会调取该领域知识库回答）
- 「**保密机制**」：蓝图不允许电子化储存（一旦截图 / 拍照立即蒸发），只能口头传授和手抄

### 2.2 第二层：全国万人排行榜（中盘层 - 提供竞争 / 危机感）

#### 排行榜结构

- **总榜**：全国 10000 个绑定者（每个绑定者对应中国某个 GDP 后 10% 的县）
- **三级架构**（用户要求）：
  - **全国榜**：10000 人，每周更新排名
  - **省级榜**：以广西为例，约 30-50 人（其他省份配比类似）
  - **区域榜**：本地区域前 10 名（如桂中区域）
- **排名算法**：综合分 = 县城 GDP 增速 50% + 民生指标改善 20% + 系统点亮节点数 15% + 短视频流量 / 评论 15%

#### 季度淘汰机制（用户拍板的核心机制）

- **每季度末末位 100 名淘汰**
- **被淘汰者**：系统解绑 → 当前已解锁科技树节点保留（融城已建项目不消失）→ 但**失去系统升级权 + DP 累计**
- **被淘汰者的影响**：意味着当地县城**永远停留在被淘汰瞬间的发展水平**——这对主角是巨大压力
- **重新进入资格**：原绑定者无法再次绑定。新绑定者需满足条件
- **特殊保护**：连续 4 次冲入前 100 的可获得"季度淘汰豁免"1 次

#### 排行榜信息维度（主角能看到的）

主角打开排行榜可看到：

- 自己的排名 + 上下 10 名的对手
- 每个对手的"县城名 + 主营产业 + 当前等级 + 系统年龄"
- 前 100 名的排名（不显示细节）
- 末位 100 名的"危险区"红色标记
- **看不到**：竞争对手的具体策略、DP 余额、解锁的科技树节点

### 2.3 第三层：评论值→DP 转化（金手指燃料·对标书核心）

#### 核心机制

**主角的短视频 / 直播 / 抖音 / 视频号评论区**是 DP 的主要来源。

#### 转化规则

| 行为 | DP 转化率 | 备注 |
|------|---------|------|
| 评论：1 条普通评论 | 0.1 DP | 基础量 |
| 评论：1 条 50+ 字真实留言 | 1 DP | 长留言 |
| 评论：1 条带照片 / 视频 | 3 DP | 内容贡献 |
| 评论：1 条求助 / 民生诉求 | 5 DP | **★ 触发"评论办公"特殊机制** |
| 点赞 | 0.01 DP / 个 | 海量基础 |
| 转发 | 0.5 DP / 次 | 传播 |
| 完播率 80% 以上的播放量 | 0.05 DP / 次 | 内容质量 |
| 涨粉 | 0.5 DP / 人 | 长线 |

#### "评论办公"特殊机制（对标书《评论办公》核心）

**当主角收到「民生求助」类评论时**：
1. 系统会**自动识别**该评论是否为"真实民生诉求"
2. 如是真实诉求，触发**「评论办公」副本**：
   - 主角必须**在 7 天内回应** + 在 30 天内推动解决
   - 解决后：评论值 ×5 倍 + 当地民生改善 +1 + DP 大额奖励
   - 未解决：被该评论关注者打分降低 + DP 倒扣
3. **副本类型**：道路 / 自来水 / 教育 / 医疗 / 养老 / 就业 / 残疾人 / 留守儿童 等
4. **"工工整整记笔记本三行字"**（沿用对标书桥段）：每条触发副本的评论必须在「评论办公本」上记三行：**问题 / 交办 / 时限**
5. **"大白纸督办墙"**（沿用对标书桥段）：所有副本任务挂在主角直播间背景的大白纸上，**完播率最高的画面**

#### 评论 → DP → 解锁科技树 闭环

```
[主角拍短视频] → [评论涌入] → [评论办公副本触发] →
     ↓                                    ↓
[转化 DP] ← [民生改善值 +] ← [副本完成]
     ↓
[解锁科技树节点] → [融城实质性变化] → [更多素材 / 流量] →
     ↓
[更多评论] → 循环放大
```

## 三、系统的"奇葩规则"（极端化算子）

### 规则 1：保密绑定

- 主角**严禁向任何人公开透露系统存在**
- 违规：系统会"假装罚款"——扣除 50 DP 并发出"⚠️ 警告" 提示
- **第 23 章左右主角发现**：系统其实没有真正惩罚机制，只是吓唬人——但**全国 9999 个其他绑定者都被骗了**。这个"保密"是系统的伪规则，但主角选择继续保密（认知反差）

### 规则 2：季度淘汰末 100 名

- 已述。**核心危机感来源**

### 规则 3：评论办公副本必须 7 天回应 + 30 天解决

- 强制规定，违反扣 DP
- **现实意义**：主角不能光做大方向，必须**接受微观民生压力**——这是代入感的关键

### 规则 4：跨绑定者合作的"加成机制"

- 两个绑定者跨县合作 → 双方 DP +10% / 项目
- 三方合作 → 三方 DP +15% / 项目
- **这是反"零和博弈"的设计**——主角能突破苏明远的零和思维，因为系统鼓励合作

### 规则 5：「人格化系统」（轻度）

- 系统的语音偶尔带情绪——主要是**柳州工业老板风格的傲娇 + 毒舌**
- 系统会在主角偷懒时说："韦东，你阿婆都比你勤快"
- 主角连续做得好时会说："勉强够水"（柳州话"勉强算有点本事"）
- **这个人格化是潜移默化的——不是元 2.1 强烈系统人格化，是轻度调味**

## 四、系统的限制与边界

- **不能凭空生产**：科技树只给"蓝图"，不给原材料 / 设备 / 工人——主角必须自己整合资源
- **不能干预政治**：科技树不能直接解决"批文 / 政策 / 关系网"问题
- **不能干预人心**：科技树不能让人喜欢主角或讨厌反派——人物关系靠主角自己经营
- **不能跨县干预**：主角只能影响融城，无法直接改变其他县（但可以通过合作机制间接帮其他绑定者）
- **不能预测未来**：系统不告诉主角"未来会发生什么"——只给当前的蓝图和数据
- **不能复活 / 治病**：系统不能救母亲的腰椎 / 找回失踪的父亲——这些必须靠现实努力

> **这些限制让主角的胜利变得"努力可达成"而不是"系统送的"**——避免无脑爽。

## 五、系统的真相（伏笔·400 章后揭示）

> **以下为长线伏笔，正文期前 300 章不揭示**：
>
> 「县城逆袭科技树·全国挑战赛」**不是天降系统**，是一个 2020 年代国家级"乡村振兴 + AI 工程"的实验项目。10000 个绑定者是**算法从 1.4 亿县城青年里筛选出的"高潜力返乡创业者"**。
>
> 季度淘汰机制是**算法的强化训练**——优胜劣汰让系统不断学习"哪种创业模式能让县城真的逆袭"。
>
> 在系统第 400 章的真相揭示里，**主角是 10000 人里第一个意识到"我们是被实验"的**——这开启全书第四幕的格局升级。
