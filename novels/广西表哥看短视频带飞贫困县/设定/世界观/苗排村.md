# 苗排村 — 世界观设定卡

> 主角母系家族（覃姓苗族）根据地 / 外公外婆所在地 / 弄崖乡苗族村寨。

## 一、基本信息

- **名称**：苗排村
- **类型**：行政村（苗族山区村寨）
- **行政归属**：融城县弄崖乡（与弄崖村 + 弄潭村 + 侗江寨同属弄崖乡）
- **民族构成**：以苗族为主（覃姓苗族家族聚居）
- **海拔**：800-1500 米（山区）
- **到弄崖村距离**：22 公里盘山路
- **核心特色**：覃姓苗族家族 + 苗排村芦笙队（与苗山镇苗山镇芦笙堂联动）

## 二、地理特征

- 弄崖乡苗族村寨之一
- 与苗山镇芦笙堂在地理上较近（覃家在苗排村，芦笙堂在苗排村）
- 山地 + 杉木林 + 梯田

## 三、人文与历史

### 主角母系家族（覃姓苗族）：

- **外公覃老六**：73 岁 / 弄崖乡芦笙队队长 / 隐藏的拉鼓节传承人
- **外婆龙阿婆**：71 岁 / 苗绣老艺人 / 从侗江寨嫁来（侗族苗族复合血脉）
- **母亲覃秀英**：48 岁 / 苗族 / 从苗排村嫁到弄崖村韦家
- **二伯覃国宏**：67 岁老党员（第 7 章村民代表大会拄拐杖支持主角的"我是看着他长大的"金句的人物）
- **覃小慧**：25 岁 / 主角青梅竹马 / 苗族芦笙艺人 / 抖音 8.2 万粉博主"小慧·苗山非遗"
- **覃小燕**：23 岁 / 主角堂姐
- **覃小蓉**：21 岁 / 主角堂姐
- **覃国伟**：32 岁 / 主角表哥 / 2017 年茶园失败 / 现回乡（卷纲规定第二幕共情角色）
- **覃满全**：42 岁 / 覃小慧二堂叔 / 做过 5 年小工 / 第 9 章作坊改造灶台砌建
- **覃永田**：67 岁 / 苗排村本地农技手 / 第 19 章弄崖村农技培训站师资

### 龙姓家族（外婆血脉）：

- **龙阿婆**：71 岁 / 外婆 / 苗绣老艺人
- **龙阿萍**：17 岁辍学回乡 / 母亲 2 年前在广东打工没回 / 父亲在柳州市区打工 / 跟外婆相依为命 / 主角姨表妹 / 第 19 章培训站学员

## 四、关键场景 / 物件细节（写正文时可直接调用）

1. **覃家老吊脚楼**：与弄崖村吊脚楼同样风格 / 外公外婆居住地
2. **苗排村芦笙队的训练场**：约 6 人芦笙队 + 鼓队（与苗山镇芦笙堂联动）
3. **传统的苗族风格村落**：杉木 + 砖瓦 + 山坡上层叠分布
4. **梯田 + 紫苏种植**：苗排村本地农产业
5. **油茶林**：第 20 章主角带 22 名学员到苗排村油茶林学结实期管理
6. **覃家祖宅**：外公劈鼓木 + 外婆打苗绣的祖宅（与苗山镇芦笙堂相邻）
7. **苗排村小卖部**：阿贵在弄崖村，苗排村有类似的本地小卖部

## 五、出场章节与剧情功能

- **第 9 章**：主角骑电瓶车 50 公里到苗山镇拉螺，回程中途经过苗排村
- **第 13 章 ⭐**：主角骑五菱小卡 22 公里盘山到苗排村找外公（**正式登场**）/ 接下拉鼓节复活任务 / 父亲失踪 18 年伏笔正面触发
- **第 14 章**：覃小慧带苗排村两位堂姐 + 苗排村芦笙队 6 件百鸟衣到弄崖村凑队
- **第 19-20 章**：苗排村学员 6 人 / 农技培训站第二天田间地头到苗排村油茶林学习

## 六、剧情功能

### 前 30 章承担：

- 主角母系家族（覃姓苗族）的根据地
- 外公覃老六 / 外婆龙阿婆 / 覃小慧 / 覃国伟等关键人物的物理依托
- "弄崖乡内部苗汉联动"（苗排村苗族 + 弄崖村汉族混血 + 侗江寨侗族）的样板
- 系统文化树 T1 副本（拉鼓节）的人物资源池

### 长期弧线：

- 第二卷：苗排村加入"五乡联动"产业链
- 第三卷：龙阿萍 / 覃国伟等返乡青年的核心来源地
- 全卷：苗排村芦笙队成为"苗山镇芦笙堂直播基地"的核心人员

- 与对标书"乡村振兴 + 党员先锋"模式的差异化——本书是"民族家族网络 + 苗汉融合"模式
- 覃姓家族 + 龙姓家族的交错血脉是本书地域文化深度的核心来源
