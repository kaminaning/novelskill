# 融城县文旅局 — 势力卡

## 一、基本信息

- **势力名**：融城县文化和旅游局
- **类型**：县政府组成部门（文旅 + 非遗 + 体育 + 文物 + 广电融合）
- **所在地**：融城镇县政府办公区（非二号楼）
- **行政归属**：融城县人民政府
- **主管**：周振华副县长（分管文旅 / 商务 / 招商）

## 三、势力范围与实力

### 管理范围：

- 融城县非遗保护（含融水苗系列坡会群 / 拉鼓节 / 苗绣 / 油茶 / 螺蛳粉等申报）
- 县文旅产业规划（年游客 4 万人次的底子，主要在坡会期间集中）
- 142 个行政村文化资源摸底
- 弄崖村作坊"村集体合作社"的县级监督方角色（第 12 章周振华谈判中明确）

### 政治资源：

- 县文旅局副局长身份让周国强能**给主角合规通道**（批文 / 政策口径 / 评审）
- "等了八年的回乡仔"——主角是周国强政治版图中的关键变量
- 与县扶贫办覃志强、县农业农村局张秀芳、副县长周振华形成"实地主义干部"非正式联盟

### 周国强的工作日常：

- "管的就是非遗这一摊"
- 跟县长打了八年报告"要发展苗侗文旅 / 螺蛳粉作坊 / 短视频"被上头一句"融城不是桂林搞那些没用"驳回
- 第 11 章受主角邀请，从"文旅局副局长"切换到"老地方茶馆"穿深蓝衬衫见面（去办公室外的状态）

## 六、剧情功能

### 前 30 章承担：

- 体制内盟友三层级的最底层（周国强 → 周振华 → 张秀芳的金字塔最底层）
- 主角"评论办公"模式从村级到县级跃迁的桥梁
- 第 24 章金句章核心交锋场景（"我不打蛇，把路修宽"金句的对话方）
- 黄大发偷税线索的关键传递人（M05 伏笔启动的关键变量）

### 后续弧线：

- 第二卷：周国强升副县长 / 主角推动周国强的政绩
- 第三卷：陈志强县长退休 / 周国强是否能接班县长（长线悬念）
- 全卷：周国强成为主角"体制内核心盟友"的稳定值

## 七、关键场景细节

- **龙城阁茶馆**：周国强当年还在文旅局副科级时常去的小茶馆，老板是柳钢退休车间主任老李（65 岁戴老花镜慢慢摆茶具），桌上是金骏眉 + 糟辣豆角 + 凉拌口水鸡——与第 29 章鸿门宴的"远辉鑫源楼大席"形成体制内 vs 民营老板的桌面对比
- **A4 纸折成方块的票据线索**：周国强不发微信不发邮件，用纸笔写线索递过来——是体制内人的"留痕方式"和"安全方式"
- **深蓝色衬衫**：周国强"不在办公室"的着装信号
