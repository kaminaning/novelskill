- **势力名**：苏氏远辉实业集团（旗下"远辉系"）

- **所在地**：融城镇远辉鑫源楼 28 楼（总部）

## 三、势力范围与实力

### 旗下 5 家子公司：

1. **远辉物流**（苏明亮）——本地货运垄断（但县内具体物流由小舅子黄大发的融江物流执行）
2. **远辉商超**（苏婉婷）——融城镇 3 家连锁，覆盖本地商品 70% 流通
3. **远辉文化创意有限公司**（"远辉文创"，苏文清）——商标截胡的法律主体（5 月 6 日抢注"弄崖村苗侗礼盒"等 4 个商标）
4. **苏家茶园**（苏明远直管）——融城北部茶园基地，韦德才接受好处的请客点
5. **远辉商业地产**——远辉鑫源楼物业方

### 经济实力：

- 控制融城县内民营经济 70%
- 五菱配件代工厂（融城工业代工）
- 与北海祖宅（苏家发家地，南珠手串伏笔的源头）保持联系
- 资产沉淀：融城江景豪宅区 1800 万别墅 + 远辉鑫源楼 28 楼整层
- 信息掌控 A+ 级：对主角动作三周内整理 87 页厚资料

### 政治资源：

- 副市长堂哥周永刚（市级保护伞）
- 县商会 + 文旅 / 商务 / 招商口都有眼线网络
- 韦德才（弄崖村村主任，每月 500 块 + 免费购物卡收买）

## 四、与主角的关系

- **立场**：**主反派阵营**（贯穿全书 1-200 章的最大商业对手）
- **冲突本质**：新秩序（短视频 + 流量 + 万人榜）vs 旧秩序（资源控制 + 关系网 + 零和博弈）
- **金句对锚**：苏明远"广西就这么大块蛋糕"（F03 伏笔） vs 主角"蛋糕是做大的，不是抢小的"

### 全卷弧线预设：

- 第 50 章：鸿门宴升级（40% 隐性控股要求）
- 第 80-150 章：全面对抗（融城市域运营博弈）
- 第 150-200 章：决战（苏明远孤注一掷 → 副市长堂哥切割 → 倒台）
- 第 200 章：苏氏家族末代 / 苏宇航是否反水成关键悬念

## 七、致命弱点

苏家三大致命弱点（鸿门宴主角已戳穿其中两点的微表情）：

1. **祠堂规矩**：苏建国老爷子的封建家族观念束缚——苏明远不能完全自由决策
2. **网瘾儿子苏宇航**：主角抖音粉丝（F02 伏笔），鸿门宴上主角已亮牌
3. **副市长堂哥的边界**：周永刚的政治风险敏感度——一旦苏明远触红线副市长会切割
4. **对短视频认知盲区**：远辉系商超下架弄崖村酸笋的"低判断"（苏婉婷提议被苏明远否决但反映集体盲区）
5. **北海祖宅 / 南珠手串左侧那颗略小的珠子**：F01 伏笔的远端钩子
