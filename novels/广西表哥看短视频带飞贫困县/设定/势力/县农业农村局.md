# 融城县农业农村局 — 势力卡

## 一、基本信息

- **势力名**：融城县农业农村局
- **类型**：县政府组成部门（农业产业 + 乡村振兴 + 农技推广 + 地理标志注册申报口）
- **所在地**：融城镇县政府办公区
- **管辖**：弄崖乡 / 苗山镇 / 侗寨乡 / 北河镇 等 9 乡镇农业产业

## 三、势力范围与实力

### 管理范围：

- 弄崖村农技培训站省级农技推广点（张秀芳挂牌承诺）
- 紫苏 / 香糯 / 笋干 / 油茶 等本地农产品产业指导

- 融城北部山区五乡联动产业规划（张秀芳第 20 章主动提的合作方向）

### 政治资源：

## 六、剧情功能

### 前 30 章承担：

- 体制内盟友三层级中间层（周国强 → **张秀芳** → 县级层）
- "实地主义干部"形象样板（卷纲规定的代表角色）
- 地理标志注册申报权 = 主角应对商标截胡的关键反击武器
- 龙阿萍母亲在番禺电子厂没消息的信息透露（弄崖村失踪外出务工人家属群体共情升级）
- "融城北部山区五乡联动"概念提出 = 第二卷融城摘帽主线的政策依据

### 后续弧线：

- 第二卷：地理标志正式批 → 远辉系"弄崖村"商标被撤销 → 商标截胡彻底翻盘
- 第三卷：张秀芳升农业农村局长 / 主角推动"五乡联动"成为融城摘帽核心样板

## 七、关键场景细节

- **张秀芳的旧捷达 + 米色风衣 + 普通牛仔裤**：与周振华"领导范"形成对比，是融城县实地主义干部气质代表
- **蹲在笋干棚的杉木门槛**：广西人蹲文化的县级官员版本
- **"卯崽，借个事你晓得"**：第一次用本地方言对主角说话的县级干部，比周振华"卯崽你这话讲得有点意思"更亲近
- **认识龙阿萍妈妈**：丰满了"弄崖村失踪外出务工人家属群体"的共情网络
