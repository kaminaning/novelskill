# 融城县扶贫办 — 势力卡

## 一、基本信息

- **势力名**：融城县扶贫办（已并入"乡村振兴局"但日常仍以扶贫办称呼）
- **类型**：县政府组成部门（扶贫资金审批 + 项目实施 + 乡村振兴政策落地）
- **所在地**：融城镇县政府大楼后面一栋小楼
- **行政归属**：融城县人民政府
- **门口字幅**："**群众事 无小事**"（覃志强主任办公室门口挂着）

## 二、领导人 / 核心成员

| 人物 | 职务 | 年龄 | 立场 | 备注 |
|------|------|------|------|------|
| **覃志强** | 主任 | 59 | **务实老干部 / 主角远亲** | 本地土著、覃姓远亲（与主角母亲覃秀英家族关系），开门见山型，先卡流程后特事特办 |

## 三、势力范围与实力

### 管理范围：

- 县扶贫专项资金（80 万扶贫款是第 12 章批给主角弄崖村作坊的关键资金）
- 142 个行政村扶贫项目实施
- "申报 → 评审 → 拨款"三步流程（正常 14 天）

### 政治资源：

- 与县农业农村局 / 县文旅局协同
- 周振华副县长可以打"特事特办"招呼

### 关键资源：

- **80 万扶贫款**——第 12 章周振华谈判后批准 / 第 16 章被压缩到 8 小时到账（覃志强特事特办执行）
- 流程审批权——主任的"流程一步都不能省"是政治正确口径，但能根据"特事特办"调节

## 四、与主角的关系

- **立场**：**主角隐性盟友 / 远亲族人**（覃姓家族关系）
- **关键事件**：
  - **第 16 章**：主角下午两点进扶贫办 → 蹲在地砖上 + 自热螺蛳粉 + "借个事好搞" → 用 8472 单 × 50 万损失/周的数据说服 → 覃志强打电话给周振华汇报 → 周振华"特事特办" → 晚上 9 点 80 万入账村集体合作社账户（**正式入场**）
- **核心对话**：
  - 覃志强："**80 万的事周副跟我打过招呼了，但流程一步都不能省。**"
  - 主角："**主任，借个事好搞——这 8472 单要发货，村道五菱小卡现在进不去。村道修不通，融城就少 50 万营业额一周。**"
  - 覃志强："**卯崽，你这账算得几卵狠。**"

## 五、首次出场章节

- **第 16 章**：覃志强办公室 → 蹲在地砖上谈判 → 80 万特事特办到账（**正式登场**）

## 六、剧情功能

### 前 30 章承担：

- 评论办公副本 2（弄崖村村道硬化）的资金通路
- 主角"政商效率"型爽点的承接体（14 天流程压缩 8 小时）
- 县级体制盟友"务实老干部"样板（与张秀芳、周国强、周振华形成三层级）
- "覃姓远亲"线（隐性家族网络，覃秀英 / 覃小慧 / 覃国伟 / 覃老六 / 覃志强 / 覃成福 / 覃永田 / 覃满全 / 覃国宏 / 覃永福 / 覃志强 / 覃小燕 / 覃小蓉 等覃姓家族在融城北部山区的网络）

### 后续弧线：

- 第二卷：覃志强升乡村振兴局长 / 主角扶持其他乡镇扶贫项目
- 第三卷：覃志强退休 / 主角推荐覃国伟接班（家族传承）

## 七、关键场景细节

- **办公室门口字幅"群众事 无小事"**：延续对标书"实地主义 + 大白纸督办墙"的物件叙事
- **缺口搪瓷茶杯**（与县政府二号楼小会议室同款）：体制内人的"老干部物件"
- **韦东蹲在地砖上的姿态**：广西人的蹲文化在县级办公场景的"格格不入但有效"展示
- **自热螺蛳粉撕开倒水**：本书"嗦粉式思考"在县级办公场景的极端化表达
