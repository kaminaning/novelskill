# 鹿寨广运冷链 — 势力卡

## 一、基本信息

- **势力名**：鹿寨广运冷链运输有限公司
- **类型**：跨县民营冷链物流公司
- **所在地**：柳州市鹿寨县（融城原型周边真实区域之一，公路一小时车程）
- **行业地位**：鹿寨县内中型冷链物流公司，承接广西区内中短途冷链业务
- **从属关系**：**主角弄崖村作坊的"备用物流盟友"**（绕开融江物流的核心通路）

## 三、势力范围与实力

### 业务范围：

- 鹿寨县 + 柳城县 + 融安县 + 融城县（融城原型涵盖区）跨县冷链运输
- 广州 / 深圳零担货运合作通路
- 鹿寨 → 柳州市区主干线

### 服务承诺（主角对接条件）：

- 三个月合作框架
- 加价 18%（比融江物流贵，但稳定）
- 货车出问题莫总赔
- 现金结算

### 实力配置：

- 冷链车队（具体规模未详写）
- 与跨省零担货运平台对接
- 柳州工业基因背景（莫总柳钢退休 + 鹿寨本地资源）

## 四、与主角的关系

- **立场**：**主角"绕开融江物流"的备用物流核心**（第 22 章正式签约）
- **战略意义**：
  - 突破黄大发"卡脖子"的关键通路
  - 工业柳州血脉延伸（莫总舅舅 = 父亲当年柳钢车间同事）
  - 主角"我不打蛇，我把路修宽"哲学的具象载体
  - 父亲失踪线索的早期种子（莫总舅舅是未来线索人物）

## 六、剧情功能

### 前 30 章承担：

- 主角"绕开融江物流"的物理通路
- 工业柳州血脉伏笔的早期承接（莫总舅舅 = 父亲柳钢车间同事）
- "广西人不在融城内部死磕"的格局展示载体
- 第二幕跨县合作的物流基底

### 中长期弧线：

- 第二卷：莫总舅舅正面出场 / 父亲失踪线索的早期获取
- 第三卷：鹿寨广运 + 黄文秀（梧州蒙山）+ 陈志远（钦州灵山）形成"跨县盟友物流圈"

## 七、与外部势力关系

- **联盟**：弄崖村作坊 / 黄文秀（蒙山）/ 陈志远（钦州灵山）跨县绑定者
- **对立**：融江物流（黄大发）—— 鹿寨广运的存在直接打破融江物流的县内垄断
- **隐性线索**：莫总舅舅 = 主角父亲柳钢车间同事（L01 父亲失踪长线伏笔的早期种子）
