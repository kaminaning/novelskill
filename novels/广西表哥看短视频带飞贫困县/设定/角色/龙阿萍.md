# 龙阿萍 — 设定卡（17 岁辍学少女·留守儿童代表·主角姨表妹）

## 一、基本信息

- **姓名**：龙阿萍
- **性别**：女
- **年龄**：17 岁
- **籍贯**：广西柳州融城县弄崖乡苗排村
- **民族**：苗族
- **身份标签**：苗排村 17 岁辍学少女 / 母亲 2 年前去广州番禺电子厂落了一段时间后没消息 / 父亲在柳州市区打工偶尔寄钱 / 与外婆龙阿婆相依为命 / 初三辍学 / 弄崖村农技培训站第一批学员 / **韦东姨表妹**（外婆龙阿婆是覃秀英母亲）
- **角色定位**：⭐ 留守儿童 + 留守青年双重身份共情角色 / 第 14 章 M03 评论区留守儿童家长留言伏笔的"具象化身"
- **一句话功能**：用一句"东哥——我能帮我外婆把苗绣卖到全国吗？"，把"东莞陈女士"那条评论区留言变成弄崖村本地的具体人物，让主角"为弄崖村失踪外出务工人家属群体"的动机有了一个能看见的脸

## 二、外貌特征

- **关键词**：身高 1 米 58 / 17 岁但比同龄人显小 / 苗山日头晒出的黑里透红肤色 / 长辫子（苗族少女传统款）
- **常见装束**：旧校服改的卫衣 + 牛仔裤 + 旧运动鞋
- **记忆点动作**：眼眶红红的不说话 + 第二排座位一笔一画记笔记 / 比所有学员都认真

## 三、性格关键词

1. **沉默 + 努力**：辍学回乡但仍想学农技养活自己 + 外婆
2. **柔弱 + 倔强**：眼眶红了无数次但没哭过
3. **善良 + 真诚**：第一次跟主角说话就是"我能帮我外婆把苗绣卖到全国吗"

## 四、出场章节与剧情功能

- **首次出场**：第 19 章（农技培训站第一批学员 / 22 人中"包括一位 17 岁辍学回乡的女孩龙阿萍"）
- **关键场景**：
  - 第 19 章 ⭐：第二排座位一笔一画记笔记 + 卷末签到时小声问"**东哥——我能帮我外婆把苗绣卖到全国吗？**"——M03 伏笔具象化
  - 第 20 章 ★：田间地头实操紫苏地一笔一画记笔记 + 张秀芳副局长那句"这丫头我认识她妈——她妈 2 年前在番禺一个电子厂落了一段时间，后来没消息了"——母亲失踪线索铺设
  - 第 25-30 章：18 个村妇围坐绣帕群像中跟随外婆学苗绣的少女
- **长线规划**：
  - 第 50-100 章：龙阿萍正式成为弄崖村作坊核心员工
  - 第 200 章：龙阿萍母亲在番禺被找到（与主角父亲失踪线索可能交叉）
- **推进作用**："弄崖村失踪外出务工人家属群体"具象化身 / 留守儿童 + 留守青年双重共情 / 第二代非遗传承的种子

## 五、与主角关系

- **立场**：核心共情配角 / 主角姨表妹（家族关系）
- **互动模式**：很少说话但每次开口都让主角心里一震 / 主角对她用平等的"阿萍"称呼 / 不抢戏但每次出场都让读者揪心

## 六、关键台词

> 桂柳话 + 苗语夹杂（17 岁苗排村少女，苗语更多）

1. **"东哥——我能帮我外婆把苗绣卖到全国吗？"**（第 19 章金句 ⭐ / 与"东莞陈女士"评论留言形成镜像）

## 七、代入感锚点（共情角色必填）

- **普世痛感**：17 岁初三辍学 + 母亲 2 年前去广州番禺电子厂落了一段时间后没消息 + 父亲在柳州市区打工偶尔寄钱 + 与外婆相依为命 + 苗排村青年外流殆尽——这是"留守儿童成长为留守青年"全过程的浓缩
- **软肋（让读者共情）**：17 岁但想养活外婆 / 不哭只一笔一画记笔记 / 第一次跟主角说话是为外婆问的——这是中国千万留守儿童的浓缩
- **道德爽感**：辍学但不躺平 / 第一时间想的不是自己是外婆 / 想学农技养活家——读者会感到"这种孩子值得回报"
