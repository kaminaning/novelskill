# 阿贵 — 设定卡（发小·杂货店老板·主角铁杆）

## 一、基本信息

- **姓名**：阿贵（本名韦阿贵）
- **性别**：男
- **年龄**：26 岁
- **籍贯**：广西柳州融城县弄崖乡弄崖村
- **民族**：苗族（弄崖村本地）
- **身份标签**：弄崖村小卖部老板 / 韦东发小（从小一起长大）/ 韦满堂之子 / 弄崖村作坊"地头蛇式情报员"
- **角色定位**：⭐ 主角第一个跟着干的人 / 弄崖村铁杆 / 地头蛇资源调动者
- **一句话功能**：用一辆电瓶车 + 一辆拉货五菱小卡 + 一句"借个老叔几卵恶心"的怼人金句，把主角"村中人脉"的第一根钉子钉死

## 二、外貌特征

- **关键词**：身高 1 米 70 / 圆脸 + 大眼睛 + 笑起来露出虎牙 / 微胖（26 岁开始有点小肚子）
- **常见装束**：T 恤 + 大裤衩 + 人字拖（夏天）/ 卫衣 + 牛仔裤 + 帆布鞋（冬天）
- **记忆点动作**：嘴里叼着没点的烟 / 骑电瓶车后座载主角 / 啃烧炙

## 三、性格关键词

1. **直率 + 义气**：第 2 章韦德才嘲讽主角时直接骂出"借个老叔几卵恶心"
2. **散漫 + 靠谱**：杂货店日常很懒散，但主角喊他干啥立刻到位
3. **嘴贱 + 嘴稳**：怼人快，但主角的事不会乱说

## 四、出场章节与剧情功能

- **首次出场**：第 1 章（主角刚回村借电瓶车）/ 第 2 章正面登场
- **关键场景**：
  - 第 2 章：发现主角回村第一句"卯崽，你真回来咯？"+ 韦德才嘲讽时骂"借个老叔几卵恶心"
  - 第 5 章 ★：母亲腰椎复发凌晨开拉货五菱小卡 3 分钟到 / 50 时速翻 47 个弯送医
  - 第 6 章：傍晚送米粉看见整面墙作战图惊呼"卯崽，你这是要打仗？"+ 跑去给爸爸打电话"我也要加入"
  - 第 7 章：村民代表大会前给爸爸（韦满堂）送主角熬的螺蛳粉
  - 第 9 章：苗山镇拉螺 + 三江侗寨拿酸笋 全程跟随
  - 第 10 章：开张首日订单暴涨时的现场调度
  - 第 11 章：第一次完整剪辑视频"剪得有点抖但主角说'够水的咯'"
  - 第 21 章 ⭐：黄大发掐渠道后凌晨观察记录 + 在物流园对面小杂货店租二楼作为观察点
  - 第 22 章：祠堂蹲会五人之一（叼烟的那个）
  - 第 26 章：礼盒爆单时去备包装盒
  - 第 27 章 ⭐：三线情报收集"阿贵线"——盯紧融江物流县物流园出入车辆 + 拍照存档 + 凌晨 2-4 点"非常规出货"
  - 第 28 章：商标截胡夜核心团队成员
- **推进作用**：地头蛇资源调动 / 情报收集执行 / 主角"返乡仔身份的本地合法性"原始钉子 / 笑点担当 / 关键时刻凌晨送医

## 五、与主角关系

- **立场**：发小铁杆 / 第一个跟着干的人 / 地头蛇情报员
- **互动模式**：互怼但不真怼 / 主角喊一声"狗肉"就到位 / 主角嘴贱他嘴更贱 / 但提到主角爸爸他立刻闭嘴

## 六、关键台词 / 口头禅

> 纯桂柳话 + 弄崖村本地口音

1. **"卯崽，你真回来咯？"**（第 2 章重逢）
2. **"借个老叔几卵恶心。"**（第 2 章祠堂前怼韦德才 ⭐）
3. **"卯崽，你这是要打仗？"**（第 6 章看见作战图）
4. **"我也要加入。"**（第 6 章给爸爸打电话）
5. **"东哥，刚刚陈大头的车队又把我们的货扣下来了。"**（第 26 章紧急通报）
6. **"收到，东哥。"**（第 24 章秒回情报指令）

## 七、代入感锚点（重要配角必填）

- **普世痛感**：26 岁弄崖村本地 / 没大学文凭 / 杂货店日营业额 200 / 想找个媳妇但村里没年轻女性 / 跟着主角是他唯一的"翻身机会"
- **软肋（让读者共情）**：发小情谊 / 杂货店是父亲韦满堂给的家底 / 26 岁还跟父母住一起 / 但跟主角干第一时间不问钱
- **道德爽感**：义气 + 不计算 + 关键时刻凌晨救人 + 怼反派不留情 + 跟主角混不抢戏——读者会感到"每个广西人身边都有一个阿贵"
