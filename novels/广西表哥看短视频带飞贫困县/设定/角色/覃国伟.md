# 覃国伟 — 设定卡（表哥·返乡失败青年·第二幕共情角色）

## 一、基本信息

- **姓名**：覃国伟
- **性别**：男
- **年龄**：32 岁
- **籍贯**：广西柳州融城县弄崖乡苗排村
- **民族**：苗族
- **学历**：广西大学商业管理本科（2017 年毕业）
- **身份标签**：弄崖乡上一届返乡青年（**2017 年返乡失败案例**——开过茶园 3 年亏 48 万回深圳）/ 现在主角拉动重启返乡 / 弄崖村作坊兼职财务 + 运营搭档 + 农技培训站助教 / 韦东表哥（覃秀英堂兄之子）
- **角色定位**：⭐ **第二幕共情角色**（31-100 章核心 / 卷纲明确规划）/ "想留下来但留下来真的能活吗"代言人 / 主角商业管理副手
- **一句话功能**：用一段"我 2017 年茶园亏 48 万"的失败史 + 一句"留下来真的能活吗" + 一台开第一车螺蛳粉过新村道的五菱小卡，把"返乡失败青年"群体的共情和"这次不一样"的希望同时拉到全网

## 二、外貌特征

- **关键词**：身高 1 米 76 / 偏瘦 / 短寸头发 / 笑起来眼角有 32 岁的细纹（比同龄人显大）
- **常见装束**：旧 polo 衫 + 牛仔裤 + 解放鞋（与主角同款，是兄弟也是身份）+ 一只搪瓷杯（永远捏在手里 / 第 21 章手抖留茶渍是物件叙事锚点）

## 三、性格关键词

1. **务实 + 自嘲**：失败过的人不怕再失败但怕再失败
2. **专业 + 谦卑**：商业管理本科 + 3 年茶园实战 + 5 年深圳打工——业务能力很强但不抢主角风头
3. **温和 + 隐痛**：32 岁未婚 + 没给妈过 60 大寿 + "我也想留下来啊"——这是一代失败返乡青年的浓缩

## 四、出场章节与剧情功能

- **首次出场**：第 17 章（村道副本溶洞难题时带队 17 人施工 + "东哥，这咋整？填？绕？"）
- **关键场景**：
  - 第 17 章：村道施工队带队 + 溶洞危机时的现场协调
  - 第 18 章 ★：村道点亮日开五菱小卡载第一车螺蛳粉过新村道 + 那段"东哥——我 2017 年茶园亏完回深圳那时候，我以为弄崖村再不会有这种凌晨四点干活的日子了"——第二幕共情角色铺垫
  - 第 19 章 ⭐：农技培训站开课讲 23 分钟"**我 2017 年那个茶园是怎么亏掉的**" + 全网级金句首次落地"**我也想留下来啊——但留下来真的能活吗？我 2017 年回来时跟你们一样，以为我有大学文凭就行。结果三年亏了 48 万，连给我妈过个 60 大寿的钱都没有。这就是为什么我跟着东哥再回来——这次不一样**"——M03 评论区"我也想回去"留言 1873 条触发
  - 第 20 章：作坊宿舍跟主角规划"五乡联动"方案
  - 第 21 章：祠堂蹲会五人之一（捏青草的那个 / 搪瓷杯撞大白纸右下角留茶渍 ⭐ 物件叙事锚）
  - 第 22 章 ★：祠堂蹲会金句段"东哥，14 万 18 天，每天差不多 8000 块"
  - 第 26 章：礼盒爆单时的紧急调度搭档
  - 第 27 章：作为兼职财务，以"上月对账争议"为由对融江物流写正式询证函 → 三线情报收集中最关键的合规手段执行人
  - 第 28 章 ⭐：商标截胡夜 30 分钟法律分析段——"在先使用抗辩能保留 / 不能撤销远辉系商标 / '弄崖村'单独撤销 60% 可能 6-12 个月 / 地理标志注册要 6 个月"——专业人设具象化
- **长线规划**：
  - 卷纲第二幕共情角色（31-100 章爆款）
  - 第 100 章：覃国伟茶园重启（带着主角的资源 + 系统蓝图）
  - 第 200 章：覃国伟成为弄崖村作坊副总
- **推进作用**：第二幕共情角色全网爆款 / 主角商业管理副手 / "返乡失败青年"群体代言 / 三线情报收集合规手段执行人

## 五、与主角关系

- **立场**：表哥兼弟兄 + 创业搭档 + 第二幕共情角色
- **互动模式**：32 岁比主角大 4 岁但听主角的 / 但主角凡涉及法律 / 财务 / 商业流程都先问他 / 兄弟之间不互相吹捧但能在凌晨四点工地上互相递保温杯

## 七、代入感锚点（核心共情角色必填）

- **普世痛感**：32 岁 / 大学文凭 / 返乡创业 3 年亏 48 万 / 没给妈过 60 大寿 / 灰溜溜回深圳打工 5 年 / 再被主角拉回弄崖村——这是中国千万 1985-1995 返乡失败青年的浓缩
- **软肋（让读者共情）**：3 年茶园亏掉父母给的首付钱 / 给妈 60 大寿那一年没回家 / 失败后 5 年深圳每天加班到凌晨 / 这次回来时 32 岁未婚——这是失败者的代价
- **道德爽感**：失败过敢公开讲失败 / 主动在直播间说"我 48 万亏在哪里" / 不抢主角风头但关键时刻给法律分析撑场——读者会感到"这种表哥值得跟"
