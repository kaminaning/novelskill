# 覃成福 — 设定卡（78 岁喀斯特石匠·三代失踪的中间人）

## 一、基本信息

- **姓名**：覃成福
- **性别**：男
- **年龄**：78 岁
- **籍贯**：广西柳州融城县弄崖乡弄崖村（村南）
- **民族**：苗族（弄崖村本地）
- **身份标签**：喀斯特石匠世家第二代传人（1948 年生 / 父亲覃永福是 1957 年柳钢勘探队临时工 / 1998 年父亲去打工没回来后封手 28 年）/ 弄崖村吊脚楼旧屋主人（M02 伏笔实体）
- **角色定位**：⭐ 三代失踪的中间见证人（爷爷辈柳钢 / 父亲辈下岗 / 主角辈返乡）/ 喀斯特溶洞地质的关键技术持有人 / 柳州工业血脉的本地化身
- **一句话功能**：用一把锈得几乎认不出的石匠凿 + 一张 1957 年牛皮纸地质图 + 一句"弄崖村这地下，是龙城的根"，把"工业柳州 70 年血脉传承"压缩到主角一条村道里

## 二、外貌特征

- **关键词**：身高 1 米 65 / 78 岁但腰还不弯 / 满头银白短发 / 满脸喀斯特山地刻出的深皱纹
- **常见装束**：旧蓝色解放装 + 布鞋 + 拐杖（节后才用 / 进溶洞时不用）
- **记忆点动作**：摸石头三个动作（看 / 敲 / 听）/ 18 年没敲过石灰岩 / 第一下震麻第二下稳了

## 三、性格关键词

1. **手艺 + 谦卑**：78 岁石匠世家但讲话只说"18 年没碰石头，手忘了。但石头没忘"
2. **沉默 + 厚重**：父亲 1998 年失踪后封手 28 年 / 现在因主角再敲第一锤
3. **传统 + 实在**："弄崖村这地下，是龙城的根"——这是一辈子守地之人的话

## 四、出场章节与剧情功能

- **首次出场**：第 16 章末（外公电话推荐 "村南 78 岁覃成福，他爹是石匠世家。但他 18 年没干这活了"）
- **关键场景**：
  - 第 17 章 ⭐：溶洞危机化解的核心——进入溶洞观察 17 分钟 + "**18 年没碰石头，手忘了。但石头没忘——这个洞是 1957 年柳钢勘探队碰到过的，我爹给柳钢供过这边的石灰，留过一张图。**" + 1957 年牛皮纸手绘地质图 / 1957 年柳钢工人手册 / 1980 年代小型石灰窑（M02 伏笔实体）+ 手抖握 18 年没动过的石匠凿 + "**后生仔——你晓不晓得，我爹生前总说，弄崖村这地下，是龙城的根**"
  - 第 18 章 ⭐：村道点亮日"78 岁老工匠 18 年后再敲石灰岩"直播切片 + 第一下震麻第二下稳 + 路边敲三下新石灰岩验收
  - 第 19 章：杉木手刻"弄崖村农技培训站"匾额（覃成福老工匠手艺延续）
- **推进作用**：喀斯特溶洞地质核心技术 / 柳州工业血脉 70 年传承的本地化身 / 三代失踪的中间见证人

## 五、与主角关系

- **立场**：核心共情盟友 / 工业柳州血脉化身 / 18 年后重拾手艺的人
- **互动模式**：很少说话 / 但每个动作都是 18 年的重量 / 主角对他用尊重的"覃叔" / 第 17 章主角讲蓝图工艺给他听时他听到第三句就拿起石匠凿

## 六、关键台词

> 桂柳话 + 苗语自然混合

1. **"18 年没碰石头，手忘了。但石头没忘——这个洞是 1957 年柳钢勘探队碰到过的，我爹给柳钢供过这边的石灰，留过一张图。"**（第 17 章 ⭐）
2. **"后生仔——你晓不晓得，我爹生前总说，弄崖村这地下，是龙城的根。"**（第 17 章金句 ⭐）
3. **"我爹叫覃永福，1957 年柳钢勘探队的临时工，懂壮苗汉三种话，能跟苗山的人交涉。"**（第 17 章 / 三代失踪叙述）

## 七、代入感锚点（核心共情角色必填）

- **普世痛感**：父亲 1998 年去打工没回来 + 自己封手 28 年 + 弄崖村吊脚楼地下藏着 1957 年的工业血脉 + 78 岁还能拄拐杖下溶洞——这是中国千万守地老人的浓缩
- **软肋（让读者共情）**：父亲失踪 / 自己 28 年没敲过石头 / 看见外孙辈的主角回来才重新拿起凿子 / 18 年的克制在第一锤震麻时全部松开
- **道德爽感**：守地不守财 / 18 年封手不卖父辈手艺 / 主角找他他不收钱只说"搞得动就搞" / 一辈子的话压成一句"弄崖村这地下，是龙城的根"——读者会感到"这才是中国老石匠"
