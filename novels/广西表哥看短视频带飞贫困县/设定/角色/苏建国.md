# 苏建国 — 设定卡（苏家第一代·祠堂规矩制定者·真正的最终拍板者）

## 一、基本信息

- **姓名**：苏建国
- **性别**：男
- **年龄**：75 岁
- **籍贯**：广西柳州融城县融城镇（祖籍湖南，明代军屯入柳）
- **民族**：汉族
- **身份标签**：苏氏家族第一代实际创始人 / 1980 年代从国营柳钢辞职创业 / 1990 年代国企改制中低价收购融城几家乡镇企业 / 2000 年代借五菱产业链向上爬 / 2010 年代借物流和商超下沉到村 / 现退居二线但每周一在祠堂主持家族会议 / 真正的最终拍板者
- **角色定位**：⭐ 反派阵营幕后大佬 / 祠堂规矩制定者（苏明远第一致命弱点的源头）
- **一句话功能**：用一句"27 天他熬不过"和一条"不害穷人 / 不踩寡妇孤儿 / 不毁同乡根基"的祠堂规矩，把苏家从"商业反派"升级为"有家训的反派"，同时也成为主角能利用的反派致命弱点

## 二、外貌特征

- **关键词**：身高 1 米 70 / 75 岁瘦削 / 满头银白 / 戴老花镜 + 中山装（祠堂正装）
- **常见装束**：深色中山装 + 黑布鞋 + 一柄拐杖
- **记忆点动作**：祠堂主位上端坐不动 + 听完所有人发言才开口

## 三、性格关键词

1. **守业 + 不残忍**：祠堂规矩"不害穷人 / 不踩寡妇孤儿 / 不毁同乡根基"
2. **务实 + 老辣**：从柳钢出身 / 经历过 90 年代国企改制 / 见过比主角危险百倍的对手
3. **家族 + 父权**：每周一祠堂会议是苏家最高决策机制 / 苏明远做大事必须经过他

## 四、出场章节与剧情功能

- **首次出场**：第 8 章侧面提及（远辉鑫源楼 28 楼会议未直接出现 / 但通过"祠堂规矩"间接定位）/ 实质性首次"出场"是第 29 章电话
- **关键场景**：
  - 第 8 章：通过苏明远讲的"祠堂规矩"间接定位
  - 第 29 章 ⭐：鸿门宴后顶楼天台父子电话——苏明远说"爸，这小子比我们想的难搞" / 苏建国停顿 3 秒"那就让他过完今年的季度淘汰再说。还有 27 天" / "27 天他熬不过" / "那就好" 挂断 —— **本卷反派幕后大佬正式定位**
  - 第 30 章：通过季度淘汰倒计时 33 小时间接施压
- **长线规划**：
  - 第 100 章左右：苏建国在祠堂主持家族会议正式定调"对付韦东"
  - 第 150-200 章：决战前夕苏建国可能因祠堂规矩与苏明远内部矛盾——主角利用的关键时刻
- **推进作用**：反派阵营幕后大佬 / 祠堂规矩制定者 / 苏明远第一致命弱点的源头

## 五、与主角关系

- **立场**：反派阵营幕后大佬 / 距离最远但份量最重
- **互动模式**：30 章之前不与主角正面接触 / 通过苏明远间接定调 / 主角能利用"祠堂规矩"绕过苏明远直接戳到苏建国

3. **75 岁年龄**：体力 + 反应力下降 / 但威望仍在
