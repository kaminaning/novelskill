# 黄国平 — 设定卡（融城县委书记·中立偏正·关键时刻站队）

## 一、基本信息

- **姓名**：黄国平
- **性别**：男
- **年龄**：52 岁
- **籍贯**：广西柳州融城县（本地土著）
- **民族**：汉族
- **职务**：融城县委书记
- **角色定位**：⭐ 县级最高政治资源 / 中立偏正 / 关键时刻站队 / 苏明远"上限"——苏明远的县级政治资源未触及黄国平
- **一句话功能**：作为融城权力金字塔顶端的"未被反派渗透的最后一块"，给主角"扳倒苏明远必须经过县委书记"的长线政治路径设定一个明确的天花板

## 二、外貌特征

- **关键词**：身高 1 米 73 / 中等身材 / 短发偏分 / 戴细框眼镜 / 永远穿黑色西装
- **常见装束**：黑色西装 + 白衬衫 + 红色或深蓝色领带（正式场合）

## 三、性格关键词

1. **保守 + 有底线**：做事保守但有底线
2. **本地土著 + 务实**：知道融城的难 + 知道苏家的根
3. **中立 + 待机**：30 章前不下场 / 但关键时刻一定站正确一边

## 四、出场章节与剧情功能

- **首次出场**：卷纲规划第 30 章正面出场 / 30 章细纲未直接出现（间接通过县长陈志强 + 副县长周振华的政治系统辐射）
- **关键场景**：
  - 第 7 章 / 第 10 章 / 第 12 章：通过周国强 / 陈志强 / 周振华间接关注主角
  - 第 30 章前后：正面出场（卷纲规划"中立偏正 / 关键时刻站队"）
- **长线规划**：
  - 第 80-100 章：黄国平在县委常委会上为主角说一句关键的话
  - 第 150-200 章：决战前夕黄国平的态度成为苏明远倒台的关键
- **推进作用**：县级最高政治资源 / 苏明远"上限" / 关键时刻站队
