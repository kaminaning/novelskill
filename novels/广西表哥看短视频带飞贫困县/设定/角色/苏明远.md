# 苏明远 — 主反派设定卡

## 一、基本信息

- **姓名**：苏明远
- **性别**：男
- **年龄**：48 岁
- **籍贯**：广西柳州融城县融城镇（土生土长融城人）
- **民族**：汉族（苏氏家族祖籍湖南，明代军屯入柳，与桂柳话历史同源）
- **职务 / 身份**：
  - 苏氏家族第二代实际掌门人
  - 融城远辉实业集团董事长（旗下 5 家公司）
  - 融城商会会长（任期 8 年）
  - 县政协常委
- **角色定位**：主反派 + 中盘对手（不是终极反派，但是前 150 章主角的核心障碍）
- **一句话功能**：让"小县城里也有真实的体面反派"——精明、隐忍、克制、长线，主角必须用时间和盟友才能扳倒

## 二、外貌特征

- **关键词**：1.78 米精瘦 / 永远三件套西装 / 鼻梁挺直 / 笑起来眼睛不笑
- **常见装束**：定制西装（深灰 / 藏青）/ 浪琴名匠手表 / 半旧公文包
- **气场**：广西工业老板特有的"务实精明 + 山水气韵"复合气质——不像广东老板那么浮夸、也不像北方老板那么粗粝
- **记忆点动作**：思考时用食指敲桌面三下（柳州本地老板常见习惯）；签字前必先用钢笔轻点墨水瓶三次
- **标志性细节**：右手腕一串南珠手串（北海带来的，他每年去北海祖宅过暑假）

## 四、反派威胁三要素（必填）

### 要素 1：至少能赢主角 1 次的方式

**前 30 章必须发生一次"主角输给苏明远"**：

- **第 28-30 章左右**：主角推动弄崖村油茶店升级为"融城首家苗侗融合非遗体验馆"，需要县文旅局批文。苏明远通过县文旅局内部关系**让批文卡 3 个月**——这期间他**抢先拿下融城镇的"非遗体验馆"招牌**（注册商标 + 抢占名号），把主角的"首家"变成"第二家"，让主角的短视频流量被分走 40%。
- 主角输：流量分流 + 投入沉没 + 心态焦灼
- 苏明远赢：低成本截胡 + 主角第一次意识到"小县城里也有看不见的天花板"

### 要素 2：完整动机

苏明远不是"为坏而坏"。他的核心动机：

> **保住苏氏家族对融城的 70% 经济控制**。
>
> 这种控制是**祖辈三代积累**的——他父亲苏建国（75 岁）1980 年代从国营柳钢辞职创业，1990 年代国企改制中低价收购了几家融城乡镇企业，2000 年代借五菱产业链向上爬，2010 年代借物流和商超下沉到村。
>
> 苏明远的认知：**"融城就这么大一块蛋糕，多一个人切蛋糕，所有人都吃不饱"**。
>
> 韦东带来的不是"新增蛋糕"（他不信），而是"新的分蛋糕者"。这种零和博弈的本能让他必须把韦东压回去。
>
> **关键**：他不会主动作恶到伤害弱者（韦东母亲、村民），但**会让韦东的事业进展处处被卡**——批文 / 流量 / 物流 / 供应链 / 资金 / 媒体，每一个环节都是他能伸手的地方。

### 要素 3：与主角的核心矛盾

**不是个人恩怨，是结构矛盾**：

| 维度 | 苏明远（旧秩序代表） | 韦东（新秩序代表） |
|------|------------------|------------------|
| 蛋糕观 | 蛋糕固定，要抢 | 蛋糕可做大，要做大 |
| 资源观 | 控制资源 = 利益 | 流动资源 = 利益 |
| 民生观 | 让他们活下去就行 | 让他们活得好 |
| 渠道观 | 线下关系网 | 线上流量 + 短视频 |
| 时间观 | 守业（保 30 年） | 创业（搏 5 年） |

**核心矛盾激化点**：当韦东证明"短视频 + 科技树"能把蛋糕做大 10 倍时，苏明远的认知必须破裂——要么承认错了（投降派），要么孤注一掷（决战派）。本书让他走**决战派**路径（更精彩）。

## 五、阵营 / 势力

### 苏氏家族核心成员

- **苏建国**（75 岁，父亲）：退居二线，但每周一在祠堂主持家族会议。**真正的最终拍板者**
- **苏明远**（48 岁，主反派）：实际掌门人
- **苏明亮**（46 岁，弟弟）：远辉物流总经理，与黄大发合作密切
- **苏婉婷**（43 岁，妹妹）：远辉商超总经理
- **苏家三代子女**（10-25 岁）：在柳州市区 / 南宁 / 广州读书或工作

### 利益同盟（县级马甲）

- **黄大发**（55 岁）：融江物流公司老板，苏明远小舅子兼合作伙伴
- **韦德才**（51 岁）：弄崖村村主任，被苏家长期收买
- **某副县长**（分管工业）：苏明远华南理工同班同学
- **县文旅局某副局长**：与苏家有 20 年交情
- **县发改局某副局长**：苏明远表哥的同学

### 市级保护伞

- **周永刚**（柳州市某副市长，苏明远堂哥）：**上限**——主角第一卷无法触及

## 六、性格关键词

1. **精明 + 隐忍**：能等 3 个月就不出 3 天
2. **务实 + 不残忍**：不主动作恶，但会冷静切割威胁
3. **江湖气 + 体面感**：广西工业老板的复合气质——讲规矩、给面子、但触及底线就翻脸
4. **格局有 + 上限不高**：能看到柳州市区，看不到全国
5. **家族观念重**：苏家三代的祠堂规矩是底线

## 七、致命弱点（必填——主角能够利用的反派弱点）

1. **苏建国的祠堂规矩**：苏家祖训"不害穷人 / 不踩寡妇孤儿 / 不毁同乡根基"——苏明远不能公开违背
2. **儿子苏宇航的"网瘾少年"**：18 岁高中生，沉迷短视频和直播，看主角视频是粉丝。**这是苏明远完全无法管控的家庭软肋**
3. **副市长堂哥的"边界"**：周永刚不会无限制保他，触碰省委 / 国务院红线时会切割
4. **对短视频 / 互联网的认知盲区**：48 岁的他骨子里不相信"流量能变蛋糕"

## 八、与主角的对抗节奏

| 章节 | 苏明远的动作 | 主角应对 |

| 31-50 | 通过黄大发掐断主角物流 + 通过韦德才搅局村委 + 通过远辉商超挤压主角线下渠道 | 主角寻找盟友（周国强、张秀芳、外公） |
