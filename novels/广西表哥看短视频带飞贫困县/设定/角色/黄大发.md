# 黄大发 — 设定卡（融江物流老板·二级反派·苏明远小舅子）

## 一、基本信息

- **姓名**：黄大发
- **性别**：男
- **年龄**：55 岁
- **籍贯**：广西柳州融城县融城镇
- **民族**：汉族
- **身份标签**：融江物流公司老板（融城物流垄断 70%）/ 苏明远小舅子（苏明远妻子黄翠的哥哥）/ 远辉系利益同盟核心成员 / 长期偷税漏税 / M05 伏笔核心证据持有人
- **角色定位**：⭐ 中型反派（10-50 章 / 卷纲规定）/ 物流卡脖子者 / 苏家阵营脏活执行者
- **一句话功能**：用一辆品字形堵车 + 47 条恶意差评 + 一辆陈大头的金叶烟车队，把"基础设施垄断者 vs 新经济创业者"的矛盾压到读者眼前

## 二、外貌特征

- **关键词**：身高 1 米 75 / 中等偏壮 / 油光满面 / 戴一条粗金链 / 笑起来眼睛眯成一条缝
- **常见装束**：唐装 + 黑色西裤 + 一只名牌劳力士手表（与苏明远的浪琴形成档次对照——苏明远高雅 / 黄大发暴发）
- **记忆点**：与苏明远食指敲桌面三下不同 / 黄大发是抠耳朵 / 抽中华烟（比韦德才的红双喜高档但比苏明远的更糙）

## 三、性格关键词

1. **贪婪 + 短视**：长期偷税漏税积累"安全感"是他敢做事的底气
2. **暴力倾向 + 不直接动手**：通过远辉物流卡车队 + 远辉商超保安队间接调度
3. **小舅子 + 自卑**：依附姐夫苏明远但也想证明自己能独当一面
4. **比苏明远更脏**：苏明远不主动作恶 / 黄大发主动作恶（47 条差评针对母亲苗绣礼盒就是他的主意）

## 四、出场章节与剧情功能

- **首次出场**：第 8 章（苏明远早会被点名"明亮，让黄大发那边把弄崖村的酸笋和香糯收购价压一压"）/ 实质性首次正面登场是第 21 章
- **关键场景**：
  - 第 8 章：苏明远开会授意"压收购价 10%"——黄大发被动出场
  - 第 10 章末：苏明远敲桌面三下 + 黄大发电话响——M04 卡物流伏笔预兆
  - 第 20 章：苏明远办公室开会"姐夫，这小子要不要直接掐？"——主动请战
  - 第 21 章 ★：融江物流县物流园品字形堵 9 吨甘蔗 + 现场负责人陈大头（戴远辉商超 VIP 腕带）+ 司机抽金叶烟——M01 主动出手 + 第一次正面卡渠道
  - 第 24 章：周国强递的 A4 纸"黄大发名下融江物流近三年异常票据"——M05 偷税情报第一份
  - 第 26 章末：第一条差评的承运公司"融江物流"+ 阿贵报"陈大头的车队又把我们的货扣下来了"
  - 第 27 章 ⭐：47 条差评在 24 小时内涌入 / 时间集中 5 月 18-22 日 / 78% 在广西区内 / 破损率最高是有母亲苗绣帕的礼盒——**最恶毒细节**（直接攻击主角母亲软肋）
  - 第 29 章：鸿门宴前 + 阿贵报"陈大头凌晨 2:38 私车进物流园办公室 40 分钟出来手里多了一个文件袋"——M05 偷税证据持续收集
- **长线规划**：
  - 第 36 章：通过纪委李国栋递交偷税证据 + 媒体曝光 → 黄大发被税务局带走（卷纲规划"中期爽点"）
- **推进作用**：物流卡脖子者 / 苏家阵营脏活执行者 / M05 偷税证据核心载体 / 47 条差评关键事件制造者

## 五、与主角关系

- **立场**：中型反派 / 物流卡脖子者
- **互动模式**：从未与主角正面对话 / 一直通过陈大头等代理人作恶 / 47 条差评针对母亲苗绣礼盒——是主角"嘴贱树敌"过敏点最毒触发者之一

## 六、关键台词

> 桂柳话 + 融城本地市井气

1. **"姐夫，这小子要不要直接掐？"**（第 20 章主动请战 ⭐）
2. **"那条线我们公司这两天调度紧张，要等。"**（第 21 章电话调度借口）

## 七、致命弱点（反派必填）

1. **长期偷税漏税**：跨省虚开发票 / 虚高运费——M05 伏笔的核心载体 / 第 36 章被纪委李国栋拿下的关键证据
2. **依附姐夫**：没有姐夫苏明远保护就什么都不是
3. **暴力倾向**：主动作恶——苏明远的祠堂规矩"不害穷人 / 不踩寡妇孤儿 / 不毁同乡根基"在他这里不成立
4. **针对母亲苗绣礼盒的恶意差评**：47 条差评最毒一招让主角"嘴贱树敌过敏点"全开 + 触发主角"等了 10 天"的预谋反击
