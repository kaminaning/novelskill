# 林晓峰 — 设定卡（大学同学·前同事·经济线伙伴）

## 一、基本信息

- **姓名**：林晓峰
- **性别**：男
- **年龄**：27 岁
- **籍贯**：广西柳州融城县北河镇
- **民族**：汉族（融城北河镇汉族多数区）
- **学历**：广西大学（与韦东同校，2017 年毕业）
- **身份标签**：原韦东深圳互联网中厂前同事 → 被裁后回融城北河镇做酸笋经销 → 主角拉来当客服 / 物流对接 / 深圳人脉枢纽
- **角色定位**：⭐ 主角"狗肉之首"（卷纲规定）/ 深圳人脉延伸 / 物流 / 财务情报双线
- **一句话功能**：用一个深圳带回来的双肩包 + 一个曾经在深圳互联网中厂的工号 + 一份鹿寨广运冷链联系人，把主角"被裁打工人盟友圈"在弄崖村具象化

## 二、外貌特征

- **关键词**：身高 1 米 72 / 偏瘦 / 短发 / 戴黑框眼镜（程序员风格但回乡后摘掉了）
- **常见装束**：旧 polo 衫 + 牛仔裤 + 旧帆布鞋 + **深圳带回来的双肩包**（永远背着 / 物件叙事）
- **记忆点动作**：刷手机回评论秒回 / 包里永远有一个充电宝 + 三根数据线

## 三、性格关键词

1. **互联网思维 + 县城实操**：5 年深圳互联网 / 2 年北河镇酸笋经销 / 复合型经验
2. **靠谱 + 寡言**：跟主角做事不多话 / 但每件事完成度极高
3. **务实 + 关切**：被裁回乡 2 年看到主角回来"我也想跟"

  - 第 22 章 ★：祠堂蹲会五人之一（背深圳双肩包的那个）+ 与主角开五菱小卡跑鹿寨广运冷链 + 谈下三个月合作框架（替代物流首战）
  - 第 24 章：龙城阁茶馆后主角让他"查桂林某栋写字楼"——M04 伏笔追查

  - 第 26 章：端午限定礼盒最后冲刺直播提议人
  - 第 27 章 ⭐：三线情报收集"林晓峰线"——通过深圳前同事联系广州/深圳物流圈朋友打听融江物流"跨省虚开发票"行情

- **普世痛感**：广西大学毕业 / 深圳互联网中厂 5 年 / 27 岁被裁 / 回北河镇做小生意 2 年 / 看见主角回来跟着干——这是中国千万 90 后被裁打工人的浓缩
