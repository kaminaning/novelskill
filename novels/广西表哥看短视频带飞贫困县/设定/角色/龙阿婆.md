# 龙阿婆 — 设定卡（外婆·苗绣老艺人·非遗传承）

## 一、基本信息

- **姓名**：龙阿婆（本名龙月娥，弄崖乡苗排村人称龙阿婆）
- **性别**：女
- **年龄**：71 岁
- **籍贯**：广西柳州融城县弄崖乡苗排村（本地土著苗族）
- **民族**：苗族
- **身份标签**：苗绣老艺人 / 弄崖乡苗绣非遗候选传承人 / 韦东外婆 / 覃秀英母亲 / 覃老六妻子 / 龙阿萍外婆
- **角色定位**：⭐ 核心共情配角（苗族手工非遗代表 / 第三幕共情阶梯重要扩展）/ 苗绣资源源头
- **一句话功能**：用一双绣了 50 年苗绣的手 + 一块送给主角的护腰带 + 一句"听说她腰又坏了"，把"留守外婆"和"民族非遗"两层共情同时锚住

## 二、外貌特征

- **关键词**：身高 1 米 52 / 微驼背 / 满头银白盘髻（苗族传统发髻）/ 手指被绣线磨得有薄茧 / 眼神温和但锐利
- **常见装束**：靛蓝苗族对襟上衣 + 黑色阔腿裤 + 银耳坠 + 一条苗绣围裙（永远佩戴）
- **记忆点动作**：戴老花镜绣苗绣 + 蹲在地窖前整理酸笋坛子 + 把烧炙塞到外孙手里

## 三、性格关键词

1. **温柔 + 坚韧**：嫁覃老六 50 年 / 看着女儿覃秀英 18 年独自撑家 / 外孙女龙阿萍辍学
2. **沉默 + 体察**：话比覃老六还少，但塞东西塞得最准（护腰带、烧炙、苗绣帕）
3. **传统 + 慈悲**：苗绣老艺人但不固守"传男不传女"，是龙阿萍最早的苗绣老师

## 四、出场章节与剧情功能

- **首次出场**：第 9 章（三江侗寨乡 / 主角去拿陈年酸笋时见到）
- **关键场景**：
  - 第 9 章 ★：地窖里 2024 年 10 月陈年酸笋（18 个月发酵）+ 亲手绣的护腰带 + 那句"东仔，你回弄崖村，顺道带这个给你阿妈——是我亲手绣的护腰带，听说她腰又坏了"——主角红了眼眶
  - 第 13 章：在苗排村芦笙堂屋里出来塞烧炙到主角手里——苗族外婆的标志性动作
  - 第 14 章：百鸟衣 + 苗绣 + 拉鼓节队伍组织的隐性后台支持
  - 第 19 章：外孙女龙阿萍辍学回乡 → 龙阿婆送龙阿萍来农技培训站学习（背景设定）
  - 第 25 章 ★：18 个村妇围坐绣帕群像核心成员之一（融水苗族苗绣传承一线代表）
  - 第 26 章：教 12 个村妇染五色糯米饭工艺协助者
- **长线规划**：
  - 卷纲第三幕共情角色阶梯第三位"**苗山镇龙阿婆**"——后续 101-200 章爆款"**我会跳坡的人都死光了**"（卷纲明示）
- **推进作用**：苗绣非遗资源源头 / 母亲软肋的代际加层 / 龙阿萍家庭背景 / 第三幕共情角色阶梯铺垫

## 五、与主角关系

- **立场**：核心共情盟友 / 苗绣非遗源头 / 长线共情角色（第三幕）
- **互动模式**：龙阿婆不说话只塞东西——护腰带、烧炙、苗绣帕；主角接过任何一样都红眼眶；外婆塞东西时永远说"听说……" / "我亲手……"

## 六、关键台词 / 口头禅

> 桂柳话 + 苗语自然混合，比覃老六更多苗语

1. **"东仔，你回弄崖村，顺道带这个给你阿妈——是我亲手绣的护腰带，听说她腰又坏了。"**（第 9 章 ⭐）
2. **"后生仔，鼓棚塌了快二十年了。"**（第 13 章配合覃老六叙述）
3. **"我会跳坡的人都死光了。"**（卷纲规划第三幕爆款金句 / 101-200 章）

## 七、代入感锚点（核心共情角色必填）

- **普世痛感**：71 岁苗绣老艺人面对传承断代 / 看着女儿独自撑家 18 年 / 外孙女 17 岁辍学 / 自己绣的护腰带最先想到的是给女儿——这是中国千万民族手工非遗老人的浓缩
- **软肋（让读者共情）**：50 年苗绣手艺 / 教过 32 个徒弟 28 个都去广东打工 / 龙阿萍是她最后的指望 / 看见外孙回来塞最好的烧炙
- **道德爽感**：苗绣老艺人但不固守 / 给外孙塞东西不说"我"只说"你阿妈" / 71 岁还在等女儿去广东的儿媳妇——这是这一代苗山外婆的浓缩
