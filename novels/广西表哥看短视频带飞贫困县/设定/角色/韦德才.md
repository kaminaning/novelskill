# 韦德才 — 设定卡（弄崖村村主任·村级反派·苏家收买）

## 一、基本信息

- **姓名**：韦德才
- **性别**：男
- **年龄**：51 岁
- **籍贯**：广西柳州融城县弄崖乡弄崖村
- **民族**：苗族（弄崖村本地）
- **身份标签**：弄崖村村主任（任期已 8 年）/ 韦东同族叔（韦东父亲韦海军的远房叔伯辈）/ 长期接受苏家"每月 500 块 + 远辉商超免费购物卡"的小恩小惠 / 笃信"出去打工不会有出息回来更不会"
- **角色定位**：⭐ 村级反派（1-15 章主要小反派 / 反派轻视段提前埋打脸的承载者）
- **一句话功能**：用一根两块钱红双喜 + 一件发黄的白衬衫 + 一句"你阿爸当年也是这样吹牛走了——一去 18 年"，把主角"嘴贱树敌"的过敏点和"父亲失踪"的最深软肋同时点爆

## 二、外貌特征

- **关键词**：身高 1 米 65 / 矮胖 / 脸像浮肿的酒鬼 / 牙齿被红双喜熏成黄色
- **常见装束**：发黄的白衬衫（永远不换）+ 黑色西裤 + 旧皮鞋 + 红双喜（2 块钱一包）+ 一串廉价佛珠手串
- **记忆点动作**：叼着烟阴阳怪气 + 哈哈大笑捂肚子 + 看人时眯眼

## 三、性格关键词

1. **仗辈分 + 内心自卑**：51 岁村主任仗着比主角大 23 岁却被被裁返乡仔反将
2. **贪小便宜 + 短视**：苏家每月 500 块 + 购物卡就把他买走
3. **嘴贱 + 怕事**：村委会羞辱主角时声音很大，但被韦满堂反呛后脸涨成猪肝色
4. **典型村级反派**：脸谱化程度比苏明远低不了多少，但有"贪小+短视+怕事"的真实底色

## 四、出场章节与剧情功能

- **首次出场**：第 2 章（祠堂门口阴阳怪气嘲讽主角）
- **关键场景**：
  - 第 2 章：祠堂门口"大城市混不下去的就回来当祖宗。读那么多书有揍马用？你看东仔，他阿爸 18 年前出去说过两天回，现在儿子也回来了"+ 在 200+ 人村群发"韦东被裁回村摆烂"消息
  - 第 4 章 ★：村委会临时会议"返乡青年三宗罪"——占地皮 / 不懂农村 / 不可持续 + 那句"**你阿爸当年也是这样吹牛走了——一去 18 年**"（主角软肋最毒辣的一次戳）+ 给苏家眼线发"程序员就是程序员"截图——反派轻视段提前埋打脸第一次埋设
  - 第 6 章：主角故意选村东头废弃仓库在韦德才家斜对面（让他每天看见）
  - 第 7 章 ⭐：村民代表大会三宗罪老套路被韦满堂当场反呛 + 投票 11:6:2 险胜 5 票 + 脸涨成猪肝色 + 两小时后给苏家发"要不要给点压力"——反派轻视段提前埋打脸的精彩兑现
  - 第 8 章：苏家眼线汇报中"昨晚 11 票支持 6 票反对 2 票弃权"
  - 第 13-15 章：韦德才在拉鼓节复活前后变得沉默（被主角的成功反向逼到角落）
  - 第 20-30 章：韦德才作为村级反派的能量逐步被苏家剥离（苏明远开始亲自下场不需要村主任级别的小角色了）
- **长线规划**：
  - 第 35 章：弄崖村村民大会罢免韦德才（卷纲规划"中期目标"）
- **推进作用**：村级反派 / 反派轻视段提前埋打脸承载者 / 父亲软肋最毒辣的攻击者 / 苏家小马甲的代表

## 五、与主角关系

- **立场**：村级反派 / 苏家小马甲 / 主角嘴贱树敌的本地原型
- **互动模式**：仗辈分嘲讽但被反将 / 苏家收买但收的太少 / 关键时刻被自己村人反呛——经典反派轻视段角色

## 六、关键台词 / 口头禅

> 纯桂柳话 + 弄崖村本地口音 + 苗族老一辈男性的傲慢腔

1. **"我跟你们讲哦，大城市混不下去的就回来当祖宗。"**（第 2 章 / 祠堂门口嘲讽）
2. **"读那么多书有揍马用？"**（第 2 章）
3. **"你阿爸当年也是这样吹牛走了——一去 18 年。"**（第 4 章 / 主角软肋最毒攻击 ⭐）
4. **"程序员就是程序员，搞这些花里胡哨的本本。七天？我看七天后他自己都忘了。"**（第 4 章 / 反派轻视段经典金句 ⭐）
5. **"要不要给点压力？"**（第 7 章 / 给苏家发的求救短信）

## 七、代入感锚点（村级反派必填）

- **不需要"共情角色锚"，但需要"真实反派底色"**：
  - 51 岁村主任拿 1500 元月工资 + 苏家 500 + 商超购物卡 = 真实贪小经济驱动
  - 8 年村主任任期 + 7 户亲戚靠他吃饭 = 真实利益绑定
  - "出去打工不会有出息"是他自己 1998 年想去东莞但没去的合理化辩护
