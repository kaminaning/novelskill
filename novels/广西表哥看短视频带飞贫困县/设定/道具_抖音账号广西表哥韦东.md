## 五、与系统机制的关联

### DP 转化率（系统第三层机制）：

- 1 条普通评论 = 0.1 DP
- 1 条 50+ 字真实留言 = 1 DP
- 1 条带照片 / 视频 = 3 DP
- **1 条求助 / 民生诉求 = 5 DP**（触发"评论办公副本"特殊机制）
- 点赞 = 0.01 DP / 个
- 转发 = 0.5 DP / 次
- 完播率 80% 以上的播放量 = 0.05 DP / 次
- 涨粉 = 0.5 DP / 人

### 评论办公副本触发：

- 主角拍短视频 → 评论涌入 → 系统识别"真实民生诉求" → 触发副本
- 副本必须 7 天内回应 + 30 天内解决
- 解决后：评论值 ×5 倍 + 当地民生改善 +1 + DP 大额奖励
- 未解决：被该评论关注者打分降低 + DP 倒扣
