# 大白纸督办墙 — 关键道具设定卡

> 对标书《评论办公》"大白纸督办墙"桥段的本土化升级 / 主角民间作战指挥部的视觉化标志 / 仪式感锚 #2。

## 一、基本信息

- **名称**：大白纸督办墙
- **类型**：纸质工作流可视化系统（50 张 A1 大小白纸 + 红色马克笔）
- **出处**：第 4 章主角从县城邮政带回 50 张大白纸 + 红色马克笔
- **物理位置**：
  - **第 4 章**：弄崖村村委会会议室墙上（首次出场，仅 1 张）
  - **第 6 章后**：主角家堂屋 + 地板 + 墙（49 张全部铺开 = 弄崖村作战指挥部）
  - **第 7 章**：49 张全部贴到村委会会议室墙上（村民代表大会用）
- **核心功能**：作战图 + 督办墙 + 数据墙的三合一

## 二、外形 / 视觉特征

1. **50 张 A1 大小白纸**：标准邮政白纸
2. **红色马克笔**：粗头 + 显眼颜色
3. **"作战图"三个字**：第 6 章主角在墙上最显眼位置写的（字写得很大）
4. **第 21 章覃国伟手抖留下的茶渍**：大白纸右下角 / 物件叙事的复利锚点 / 第 27 章主角写第 11 行督办任务时特意"挨着茶渍"写
5. **多张大白纸的内容**：每张承载一类信息（销售数据 / 督办任务 / 战略路径 / 数据分析）

## 五、剧情功能

### 前 30 章使用方式：

- 对标书《评论办公》"大白纸督办墙"桥段的本土化升级
- 但本书的差异点：这墙不在政府机关，而在弄崖村一栋老吊脚楼的堂屋里 = "民间作战指挥部"
- 主角"言出必行 + 数据驱动 + 可视化作战"的人物气质载体
- 反派"程序员就是程序员搞这些花里胡哨"被打脸的视觉化武器
- 第 7 章村民代表大会"用眼睛投票"的关键道具
- 第 21 章覃国伟手抖留下茶渍的物件叙事锚点（贯穿后续章节）
- 第 27 章主角"挨着茶渍写第 11 行"的物件复利

### 长期规划：

- 第二卷：大白纸督办墙扩张到弄崖村作坊办公室（多个房间多面墙）
- 第三卷：主角对外媒体采访时背景墙
- 全卷：大白纸督办墙的"作战图"成为主角创业 600 章的完整可视化档案

## 六、与对标书桥段的本土化升级

- **对标书《评论办公》李铮**：政府机关的大白纸督办墙（行政指令 + 政绩督办）
- **本书韦东**：弄崖村吊脚楼堂屋的大白纸督办墙（民间作战 + 创业可视化）
- **共同点**：49 张大白纸 + 督办墙 + 工工整整写字
- **差异点**：本书加了"作战图"三字仪式感 + 物件叙事茶渍锚点 + 民间空间感

## 七、与"作战指挥部"概念的关系

- "作战指挥部"是大白纸督办墙 + 评论办公本 + 红色马克笔三件道具组成的"视觉系统"
- 这个系统让弄崖村吊脚楼堂屋从"民间作战指挥部"具象化为可被读者识别的画面
- 与远辉鑫源楼 28 楼"工业老板会议桌"形成视觉对比
