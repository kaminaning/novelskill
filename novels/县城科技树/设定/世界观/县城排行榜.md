# 县城排行榜 — 世界观设定

## 基本信息
- 主题类型：核心设定（竞争机制）
- 首次出现：第2章
- 重要程度：核心设定（全书竞争驱动力）

## 排行榜概述

**全称**：县域发展竞赛排行榜

**本质**：全国共有 50 名系统绑定者，分布在 50 个国家级贫困县。排行榜追踪所有绑定者的发展成果，每周更新排名。

**核心规则**：排名越高，获得的系统奖励越丰厚；排名垫底的绑定者将被系统"降级"——减少每日签到DP、限制可解锁节点。连续 4 周垫底，系统绑定转移给同县另一位符合条件者。

---

## 排名维度

综合排名由五个维度加权计算：

| 维度 | 权重 | 衡量指标 |
|------|------|---------|
| 经济增长 | 30% | GDP增速、财政收入、企业数量、产业多元化 |
| 人口态势 | 25% | 常住人口变化、人才流入、返乡率 |
| 基础设施 | 20% | 交通通达度、通信覆盖率、公共服务评分 |
| 民生福祉 | 15% | 人均收入、就业率、教育医疗评分 |
| 文化影响力 | 10% | 文化品牌、旅游人次、知名度 |

## 奖惩机制

### 周度奖励

| 排名区间 | DP 奖励 | 特殊奖励 |
|----------|---------|---------|
| #1-3 | 800-1000 DP | 专属节点解锁券×1 |
| #4-10 | 400-700 DP | 随机资源包 |
| #11-30 | 200-350 DP | 无 |
| #31-45 | 100-150 DP | 无 |
| #46-49 | 50 DP | 系统警告 |
| #50 | 0 DP | 降级惩罚 |

### 降级惩罚

连续排名垫底（#50）的后果：
- 第 1 周：每日签到 DP 减半（5 DP）
- 第 2 周：无法解锁 T3 及以上节点
- 第 3 周：已解锁节点效率降低 50%
- 第 4 周：系统绑定转移

### 月度大奖

每月综合排名第一的绑定者获得：
- "模范县长"称号
- 3000 DP 额外奖励
- 一个 T4 节点的免费解锁权（无视前置条件）
- 全国新闻推送（现实中出现对该县的媒体报道）

## 排行榜的特殊规则

1. **合作奖励**：两个绑定者如果在现实中有跨县合作项目，双方获得额外 DP 加成（最高 20%）
2. **恶意竞争惩罚**：通过不正当手段打压其他绑定者县域发展（如恶意挖企业），被系统检测到后扣除大量 DP
3. **"越级解锁"**：排名前 10 的绑定者可以无视部分前置条件，直接解锁高级节点——这是排行榜最大的战略价值
4. **赛季制**：每 50 章为一个赛季，赛季结束时根据最终排名发放大奖。赛季第一获得"传说级"解锁权限

## 与主线关系

排行榜是全书的外部压力和竞争引擎：
1. **短期**：每周排名公布是小爽点/小危机，制造节奏
2. **中期**：与其他绑定者的互动（合作/竞争/冲突）丰富人际关系线
3. **长期**：从 #45 逆袭到 #1 的过程就是全书主线
4. **隐性功能**：排行榜把"建县城"这个慢过程游戏化，让读者有"追番"的期待感

排行榜数据变化与科技树解锁互为因果——解锁越多节点，县域数据越好，排名越高，获得更多 DP 和奖励，解锁更多节点。这个正向飞轮是全书的核心爽感来源。
