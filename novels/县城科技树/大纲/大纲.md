# 全书大纲

## 基本信息
- 书名：开局绑定科技树，把贫困县建成一线城市
- 题材：都市脑洞（系统+基建+种田+扮猪吃虎）
- 目标平台：番茄小说
- 预计字数：350万字
- 预计章数：140章
- 主角：韦东

## 一句话梗概
被裁打工人回到广西贫困县，绑定县城科技树系统，在排行榜50名竞争者中从倒数逆袭到第一，把弄安县建成全国标杆。

## 卷级结构

### 第一卷：回乡觉醒（第1-15章，约5万字）
- 功能：铺垫+起步
- 核心事件：韦东从深圳被裁回乡，绑定科技树系统，发现排行榜排名#45，解锁第一个T1节点（甘蔗精加工），完成首个任务
- 起始状态 → 结束状态：从灰溜溜的失业青年 → 初步理解系统的"准县长"
- 本卷卖点：金手指展示+排行榜震撼+广西方言暴击+第一个小爽点

### 第二卷：破土动工（第16-35章，约7万字）
- 功能：发展+第一个大冲突
- 核心事件：甘蔗产业升级、基建T1解锁、三月三文化突破、与地头蛇黄大发全面商业战争
- 起始状态 → 结束状态：从单打独斗 → 组建核心团队+击退黄大发+排名进入前30
- 本卷卖点：产业建设快感+文化IP打造+扮猪吃虎打脸+排行榜首次大幅跃升

### 第三卷：招兵买马（第36-55章，约6万字）
- 功能：团队成型+中阶对手登场
- 核心事件：人才引进、职教中心建设、苏家势力浮现、跨县合作首秀
- 起始状态 → 结束状态：从草台班子 → 正规化团队+触及深层权力网络+排名进入前20
- 本卷卖点：人才争夺战+政治博弈入门+排行榜盟友/对手互动

### 第四卷：深水区（第56-80章，约8万字）
- 功能：高潮+转折
- 核心事件：T3节点解锁（铝土矿开采/生态旅游）、苏家全面对抗、政治危机、人口回流拐点
- 起始状态 → 结束状态：从本地商人 → 县域经济核心人物+瓦解苏家网络+排名进入前10
- 本卷卖点：工业升级+深层权力斗争+人口数据拐点的情感爆发+排行榜前10

### 第五卷：决战桂西（第81-110章，约9万字）
- 功能：大高潮
- 核心事件：高铁站争夺、高速公路建设、与李沐阳直接竞争、T4节点解锁、弄安县摘帽贫困县
- 起始状态 → 结束状态：从区域玩家 → 全省焦点+基础设施质变+排名前3
- 本卷卖点：基建大战+高速铁路+全国关注+排行榜巅峰对决+贫困县摘帽

### 第六卷：山河巨变（第111-140章，约10万字）
- 功能：终局+收尾
- 核心事件：排行榜登顶之战、弄安县S评级冲刺、系统真相揭示、百万人口目标达成
- 起始状态 → 结束状态：从全国知名 → 传奇+弄安县彻底蜕变+系统终极奖励
- 本卷卖点：终极决战+系统揭秘+全书情感收束+广西崛起的集体共鸣
