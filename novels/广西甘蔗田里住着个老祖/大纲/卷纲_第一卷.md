# 第一卷卷纲：村霸鱼桶

## 卷次信息

| 项 | 内容 |
|---|---|
| 卷次 | 第一卷 |
| 卷名 | 村霸鱼桶 |
| 章节范围 | 第 1-30 章 |
| 字数预计 | 约 8 万字 |
| 写作节奏 | 30 天日更 6000 字（含 5 天集中铺前 5 章存稿） |
| 卷次定位 | 死亡开局 + 第一个完整身份揭露循环（武学高手→民国宗师） |

## 起始 → 结束

### 起始状态（第 1 章开篇）
- **韦运福**：在那安口蹲点钓鱼，被村霸踹翻鱼桶
- **小亮**：837 粉的不火主播，正举着手机找素材
- **王大牛**：村霸+糖厂老板小舅子，欺压蔗农收保护费
- **村里**：和往常一样的农忙日子
- **直播间在线**：7 人（包括小亮父母）

### 结束状态（第 30 章末）
- **韦运福**："民国宗师后人"身份在历史界小有名气；村里人开始隐隐意识到福哥不简单
- **小亮**：5 万粉主播，与父亲达成初步和解；开始有意识保护福哥不被过度曝光
- **王大牛**：在监狱
- **王老板**：刚出狱，正在筹谋报复（第二卷开场冲突）
- **新角色**：隐世道观主持林清远即将出场（第二卷开始）
- **直播间在线**：千万级关注（武协前辈+历史教授双重背书后）

## 风险与应对

| 风险 | 应对 |
|------|------|
| 反派王大牛 25 章中段消失（被抓） → 后期反派真空 | 第 22 章引入老刀（中段反派）；第 30 章王老板出狱预告卷二 |
| 30 章节奏过密 → 读者疲劳 | 第 8、13、28、29 章安排"日常缓冲" |
| 修真戏出现太晚（第 30 章才有林清远） → 主线驱动力不足 | 通过物件钩子（拳谱/米酒罐/青铜碑/烟杆）维持悬念 |
| 广西本地化滥用方言 → 非本地读者门槛高 | 方言只用句末语气词；关键剧情用普通话 |
| 主角太过保留 → 缺乏爽感 | 第 5、10、16、20、25、30 章保证有大爽点释放 |
