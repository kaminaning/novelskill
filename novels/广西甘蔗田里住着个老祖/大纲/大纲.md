# 《广西甘蔗田里住着个老祖》总大纲

## 一句话梗概

**「广西甘蔗田独居的怪福哥韦运福，是上古修真界玄霄宗末代老祖韦渡尘。邻居小亮的不火直播间，意外曝光了他三百年隐藏的所有身份。」**

## 全书规模

| 项 | 数据 |
|---|---|
| 总卷数 | 5 卷 |
| 预计总字数 | 200-300 万字 |
| 单章字数 | 2500-3000 字（按番茄传统文标准） |
| 预计总章节 | 800-1000 章 |
| 写作节奏 | 日更 6000 字（首秀期稳定双更） |

## 全书大纲（5 卷概览）

### 第一卷：村霸鱼桶（第 1-30 章，约 8 万字）

**功能**：建立人设 + 死亡开局 + 第一个完整身份揭露（武学高手→民国宗师）

**核心冲突**：村霸欺压蔗农 → 主角护短 → 武协老前辈识破"玄霄派拳谱" → 历史教授鉴定民国大佬遗物

**起始 → 结束**：
- 起始状态：837 粉小主播的不温不火日常 + 怪邻居福哥的独居生活
- 结束状态：5 万粉小主播 + 福哥"民国宗师后人"身份在历史界小有名气

**6 个完整爽点循环**（每 5 章 1 个）：
1. 第 1-5 章：村霸事件 → 力气大震慑（**死亡开局 + 第 1 章打脸**）
2. 第 6-10 章：功夫佬挑战 → **武学高手身份揭露**（第 1 层）
3. 第 11-15 章：武协前辈拜访 → **民国宗师线索浮现**
4. 第 16-20 章：网络黑公关攻击小亮 + 父亲冲突 → **主角护短**
5. 第 21-25 章：历史教授鉴定米酒罐 → **民国大佬遗物身份揭露**（第 2 层）
6. 第 26-30 章：王老板出狱 + 修真界遗孤预告 → **第一卷收尾**

### 第二卷：道观来客（第 31-80 章，约 13 万字）

**功能**：身份揭露第 3-4 层（古代名人遗后 → 修真界散修）+ 修真界遗孤上门

**核心冲突**：玄霄宗某弟子的后人"林清远"（隐世道观主持）找上门 → 主角不愿暴露身份 → 王老板出狱寻仇 → 网络全国关注

**关键事件**：
- 第 35-40 章：林清远以"游客"身份接近主角，看出端倪
- 第 50-55 章：林清远揭穿主角"修真界散修"身份
- 第 60-65 章：主角第一次正式动用修真法力（御物 + 简单符箓）
- 第 70-75 章：王老板出狱寻仇，被主角"刚好路过"反杀
- 第 76-80 章：海外华人富商"叶承志"（祖上玄霄宗弟子）找上门，**第二卷收尾**

### 第三卷：南疆蛊宗（第 81-150 章，约 18 万字）

**功能**：身份揭露第 5 层（玄霄宗执事级老祖）+ 魔界余孽现身 + 广西壮族文化与修真融合

**核心冲突**：南疆蛊宗传人"苏南疆"（壮族出身）找上门 → 揭开主角与"骆越部落"的渊源 → 魔界余孽"血煞"派人毁坏弟子牌位 → 主角失控暴走

**关键事件**：
- 第 90-100 章：苏南疆出场，揭开主角壮族姓氏"韦"的来源
- 第 110-120 章：第一次发现魔界余孽踪迹（在百色山区）
- 第 130-140 章：弟子牌位被破坏 → 主角失控 → 全村人见证主角真正实力
- 第 145-150 章：海外网友炸锅 + 国家某神秘部门"林处长"开始关注 → **第三卷收尾**

### 第四卷：渡尘归来（第 151-220 章，约 18 万字）

**功能**：身份揭露第 6 层（玄霄宗末代掌门 + 守护人间渡劫者）+ 师妹白凝玉簪揭露 + 全书最大泪点

**核心冲突**：师妹白凝的遗物玉簪被发现 → 三百年前的真相揭开 → 魔界主力即将再次入侵 → 主角必须复立玄霄宗

**关键事件**：
- 第 160-170 章：海外华人后人带来"白凝家族"线索
- 第 180-190 章：玉簪揭露 + 主角崩溃 + 三百年前真相完整呈现
- 第 200-210 章：林处长正式接触，国家与修真界开始合作
- 第 215-220 章：主角宣布复立玄霄宗 → **第四卷收尾**

### 第五卷：守山再战（第 221-300+ 章，约 25-50 万字）

**功能**：大结局，主角与魔界终极决战，玄霄宗复立

**核心冲突**：魔界主力入侵 → 主角带着新弟子（小亮+各代证人后人+国家力量）与魔界决战 → 玄霄宗以"现代修真社团"形式复立

**关键事件**：
- 第 230-250 章：主角招收第一批现代弟子（小亮成为第一弟子）
- 第 260-280 章：魔界先锋袭来，多地战斗
- 第 290-300 章：终极决战
- 大结局：玄霄宗以新形式复立，主角与白凝玉簪诀别（也可能是白凝转世归来——预留双结局空间）

## 全书三大主题

### 主题 1：身份与孤独
**核心问题**：当你活了三百年，所有同辈都死了，你还要不要"重新做人"？
**贯穿表现**：主角每年三月三祭弟子；师妹玉簪藏酸笋坛底；与小亮的羁绊从拒绝到接受。

### 主题 2：传承与延续
**核心问题**：宗门覆灭了，老祖的责任是什么？
**贯穿表现**：玄霄宗的"火种"（拳谱/隐世传人/壮族渊源）逐步浮现；最终主角决定复立宗门。

### 主题 3：现代与古老
**核心问题**：上古修真者如何在现代社会安身？
**贯穿表现**：主角不会用智能手机/嗦粉日常/广西壮族文化融合；最后宗门以"现代修真社团"形式复立。
