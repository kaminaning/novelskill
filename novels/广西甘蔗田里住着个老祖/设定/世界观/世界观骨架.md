# 世界观骨架

## 双世界设定

本书是 **「现代社会 + 隐世修真界」** 的双世界融合设定。

### 现代社会（明世界）
- 时间：当代（2026 年左右）
- 地点：中国广西崇左市江州区新和镇蔗园村（虚构村，区域真实）
- 状态：科技高度发达，直播 / 短视频 / 网红经济成熟
- 普通人对修真：**完全不知**（修仙界严格遵守"不显于凡人"的祖训）

### 隐世修真界（暗世界）
- 时间线：上古时代发祥，鼎盛于先秦-汉，衰落于唐宋
- 现状：大部分宗门已在三百年前的"魔界入侵"中覆灭，幸存者隐居各地
- 灵气：现代灵气只有上古的 **1/100**，强者难以维持境界，新人难以入门
- 修真界与现代社会的关系：完全隔离，相互不知晓存在

## 修真界力量体系

### 境界划分（玄霄宗体系）

| 境界 | 凡间表现 | 本书出现 |
|------|---------|---------|
| 炼气境（前期） | 强壮、反应快 | 小亮跟主角学完入门 |
| 炼气境（后期） | 力气大、能持久 | 部分功夫佬接近此境界 |
| 筑基境 | 内家高手、长寿（活到 80+） | 民国宗师"玄霄居士"水平 |
| 金丹境 | 御物、轻功、简单术法 | **主角现代灵气下的真实战力** |
| 元婴境 | 御剑、千里传音、寿命 200+ | 主角晋阶后展示 |
| 化神境 | 神通广大，凡人无法测度 | 主角真实底蕴 |
| 渡劫境 | 半步飞升，强如鬼神 | **主角原本境界，灵气稀薄下退到金丹** |
| 大乘境 / 飞升 | 飞升仙界 | 玄霄宗祖师叶天衍 |

### 主角境界设定（关键）

- **本质境界**：渡劫境后期（玄霄宗历史上最强之一）
- **现代表现**：因灵气稀薄，真实战力相当于金丹境
- **极限战力**：可以临时催发渡劫境秘法，但每用一次"耗损根基"（境界倒退）
- **设计意图**：让主角既能"装比打脸"又"不能无脑赢"

### 修真界主要分支

1. **玄霄宗（主角宗门，已覆灭）** — 守护人间，三百年前全员战死
2. **天罡门** — 上古五大宗门之一，已分散为散修
3. **太一观** — 道家修真，部分传承在道观中延续（第三卷"隐世道观老道"出处）
4. **南疆蛊宗** — 南方少数民族修真传承，与壮族文化有渊源（**广西地域 + 修真融合的伏笔**）
5. **东海仙岛** — 海岛修真，与现代消失（伏笔留待后续）

### 反派阵营

- **魔界余孽** — 三百年前侵入人间的魔族残余势力，潜伏至今
- **前妖兽王后人** — 上古时代被玄霄宗封印的妖兽王的后代/分身
- **现代社会反派** — 村霸、糖厂老板、网络黑公关、抢热点大主播、海外网红等

## 灵气稀薄的世界观规则

这是本书核心的"代入感+爽点平衡"机制。

### 规则 1：灵气稀薄 = 修为难涨
- 上古时代炼气境只需 1 年，现代需要 100 年
- 普通人几乎不可能修炼到筑基境
- 强者也在被动退境

### 规则 2：灵气浓厚的地点存在
- **那安** = 小型洞天福地，灵气是普通地区的 10 倍
- 主角能在那安内恢复元婴境实力
- 离开那安实力下降

### 规则 3：耗损根基的代价
- 主角每使用一次上古级法术，"根基"扣 1%
- 100% 用完就会境界永久下降
- 设计了"主角虽强但不能滥用"的张力

### 规则 4：灵气复苏机制
- 当代某事件导致灵气开始缓慢复苏（每年 +0.1%）
- 这是第四卷的核心伏笔，也是反派复出的原因

## 广西地域 + 修真的融合点

### 那安 = 喀斯特地貌
- 广西特色喀斯特地貌天然契合"洞府"设定
- 那安的钟乳石 = 上古时代的"灵脉"凝结
- 这种本地化让"修真隐居广西"显得合理

### 壮族文化 + 南疆蛊宗
- 第三卷会引入"南疆蛊宗"传承的设定
- 与壮族"花婆神""布洛陀"等民族信仰有渊源
- **本书独特卖点**：把广西壮族文化与修真融合，避免同质化

### 三月三歌节 + 修真节日
- 主角每年三月三必去镇上买五色糯米饭
- 表面是壮族节日，实则是玄霄宗"祭祖日"（祖师叶天衍飞升日）
- 这种"双重含义"是韦运福日常的核心张力

### 干栏建筑 + 镇灵阵眼
- 主角的老干栏房表面破败，实则是"镇灵阵眼"
- 干栏的"上层住人下层养牲畜"被改造成"上层住人下层储灵"
- 第二卷反派攻击干栏房 = 触发阵眼反扑

## 关键提醒

- **本书不是"修仙文"**，是"现代脑洞文+修仙背景"。
- 修真界设定是"暗线"，明线是韦运福在广西甘蔗田的日常
- 修真术语要尽量"接地气"（如"渡劫境"=人间最强，不展开境界细节）
- 不要写大段功法描述（番茄读者不爱看）
- 主角的"装老野"是核心，修真戏只在关键爆点出现
