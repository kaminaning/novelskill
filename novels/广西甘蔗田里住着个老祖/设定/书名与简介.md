# 书名与简介

## 书名

### 主书名
**《广西甘蔗田里住着个老祖》**

### 副选（不用）
- 《广西甘蔗田里那个老头，是我们宗门老祖》（更长但记忆点更强）
- 《我邻居韦运福是修真界活祖宗》（被动感稍弱）

### 书名设计逻辑
- **地域锚定**：广西 = 强地域辨识度 + 差异化（vs 大部分都市文的虚构城市）
- **生活场景**：甘蔗田 = 接地气 + 视觉化（vs 抽象的"乡下""山村"）
- **悬念词**：老祖 = 修真/宗门联想 + 反差感（30 多岁外表 vs"老祖"称呼）
- **设问句式**：番茄爆款偏爱"地点+人物+反差"的设问句式
- **类比成功案例**：《长生四十六亿年，被妹妹首播曝光》《十八岁太奶奶驾到》

### 字数与排版
- 12 字 = 在番茄手机端列表页可完整显示
- 关键词分布：广西（地点）+ 甘蔗田（场景）+ 住着（动作）+ 老祖（身份悬念）

## 简介（标准版 - 番茄风 110 字）

> 崇左江州区蔗园村独居的怪老野韦运福，三十多岁，每天蹲在那安口钓鱼或砍甘蔗。
> 
> 邻居小伙李明亮拍了他三年，粉丝从未涨破八百三。
> 
> 直到那天，村霸踹翻了韦运福的鱼桶——
> 
> 「家人们！我哥单手把村霸扔进了河里啊啊啊！」
> 
> 在线人数：7。
> 
> 三个月后，在线人数：5000 万。
> 
> 武协老前辈、历史界泰斗、修真界遗孤、海外华人……谁来都跪。
> 
> 韦运福叼着烟杆，蹲在那安口钓鱼：「麻烦。」

**字数**：约 200 字（番茄简介限制是 200 字以内最佳）

## 简介迭代说明

### 第一版（参考对标书简介）
- 套用对标书"我活了 46 亿年，妹妹直播间一开，全网震惊"的结构
- 已弃用：太像对标书容易被读者抓"抄袭"

### 第二版（采用）
- 主语换成视角主角（小亮）的吐槽视角
- 关键悬念词：837 粉、单手扔、5000 万在线、玄霄宗末代老祖
- 主角叼烟杆的镜头作为收尾，标志性动作

### 第三版（短剧化）
- 删去铺垫，直接进入冲突
- 数字化反应层贯穿（837 / 7 / 5000 / 5000 万）
- 标志性物件（酸笋坛 / 锈锄头 / 烟杆）密集呈现
