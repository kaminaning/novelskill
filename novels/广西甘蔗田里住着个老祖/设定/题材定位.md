# 题材定位

## 平台与赛道

- **平台**：番茄小说
- **主分类**：都市脑洞
- **副分类**：直播文 / 现代修真
- **标签**：脑洞、隐世大佬、修仙、直播、广西、甘蔗、反向出场、爆款重生、男频

---

## 核心梗三分法

### 主梗（一句话）
**「广西甘蔗田独居的怪福哥韦运福，是上古修真界玄霄宗末代老祖韦渡尘。」**

### 副梗（卖点扩展）
1. **反向出场流**：主角嘴笨少话，所有身份揭露靠物件 + 直播 + 旁人证词
2. **修真隐世曝光**：邻居小伙天天直播主角日常，从 837 粉一路追到亿级
3. **广西地域文化**：崇左壮族 + 甘蔗田 + 酸笋 + 那安，差异化卖点
4. **跨时代证人金字塔**：功夫佬 → 武协前辈 → 历史教授 → 修真遗孤 → 上古妖兽
5. **悲情底色爽文**：主角孤独三百年，宗门覆灭负罪感是全书情感锚

### 副副梗（差异化）
1. **三百年的孤独**：每年三月三去镇上买五色糯米饭"祭祖"（实则祭 327 名弟子）
2. **物件钩子双层含义**：锄头 = 灵剑、酸笋坛 = 储物宝、烟杆 = 法器
3. **灵气稀薄的张力**：主角虽是渡劫境但每次出手都耗损根基

## 对标分析

### 主对标：《长生四十六亿年，被妹妹首播曝光》

| 维度 | 对标书 | 本书 |
|------|------|------|
| 在读量 | 72.6 万 | 目标 80 万+ |
| 题材 | 都市脑洞 + 长生 + 直播 | 都市脑洞 + 修仙 + 直播 |
| 主角金手指 | 46 亿年寿命（蓝星开服第一） | 玄霄宗末代老祖（修真渡劫境） |
| 视角主角 | 妹妹（亲属） | 邻居（非亲属） |
| 主角形象 | 钓鱼咸鱼老哥 | 钓鱼+砍蔗+腌酸笋的独居怪福哥（村里小孩外号"老野"）|
| 时代证人 | 抗鹰-红军-淞沪-民国-清代老兵 | 武协-历史界-修真遗孤-上古妖兽 |
| 情感锚点 | 林婉清 + 1125 封信（BE） | 327 弟子牌位 + 师妹玉簪（覆灭兄弟情 + 隐藏 BE） |
| 物件钩子 | 勋章 / 地契 / 绝笔信 | 锈锄头 / 酸笋坛 / 烟杆 / 那安 |
| 数据反应层 | 7 → 1 亿（标杆） | 837 → 亿级（学并差异化） |
| 反派 | 国家级反派 + 海外网友 | 修真界宿敌 + 现实反派 |

### 学什么
1. ✅ 反向出场法（主角少出现，靠物件+证人构建）
2. ✅ 时代证人金字塔（每个时代对应一个证人）
3. ✅ 物件钩子（每章一个具体物件作为钩子）
4. ✅ 数据可视化反应层（直播数字递进）
5. ✅ BE 情感锚（师妹玉簪 = 林婉清玉簪）
6. ✅ 5-8 章一个完整爽点循环

### 不学什么
1. ❌ 信息密度过高（对标书 34-43 章淞沪线密集叙事，番茄读者会喘不过气）
2. ❌ 反派翻车节奏过密（对标书 41-42 章+52 章+90 章太密）
3. ❌ 章节重号问题（对标书第 32 章重号）
4. ❌ 海外大佬连环登场堆砌（对标书后期权贵脸谱化）

### 完全差异化什么
1. ⭐ 广西地域 / 壮族文化 / 崇左甘蔗田（vs 对标书的苏州江南）
2. ⭐ 修真隐世 vs 长生历史
3. ⭐ 邻居视角 vs 亲属视角
4. ⭐ 跨修真界证人 vs 跨历史时代证人
5. ⭐ 主角外貌 30 多岁 vs 对标书的 20 岁
6. ⭐ 那安 / 干栏建筑 / 酸笋坛 / 烟杆 vs 对标书的紫檀盒 / 地契 / 勋章

## 短剧改编潜质

### 适配短剧的爆点
1. **第 1 章村霸事件**：30 秒可拍完整爆点，主角一只手举起来扔出去
2. **第 3 章功夫佬拜倒**：50 秒拍完，"老前辈"喊出来即爆
3. **第 10 章武协前辈拜访**：90 秒可拍核心冲突
4. **第 30 章历史教授认证民国宗师手笔**：60 秒爆点
5. **第 60 章三月三祭弟子（隐含揭露第一波）**：120 秒情感戏

### 短剧化亮点
- 主角颜值 30 多岁 + 独居怪人，是短剧最爱的"反差人设"
- 邻居小亮的"家人们"开头是短剧标准模板
- 直播间数字递进有强烈视觉冲击
- 广西方言点缀（不过度）有地域辨识度
- 物件钩子（锈锄头/酸笋坛/烟杆）有强 toy-friendly 道具感

## 风险与对应

| 风险 | 对应 |
|------|------|
| 同质化竞争 | 广西地域 + 壮族文化 + 修真融合是核心差异化 |
| 修真设定门槛 | 修真术语接地气，主线是"日常+曝光"不是"修炼" |
| 主角太强失张力 | 灵气稀薄 + 耗损根基的设定让每次大动作都付代价 |
| 视角主角工具化 | 小亮有自己的弧线（返乡青年逆袭+与父辈和解） |
| 方言滥用劝退非本地读者 | 方言只用句末语气词 + 几个常用词，关键剧情用普通话 |
| 节奏拖沓 | 每章必有钩子，每 5-8 章必有完整爽点循环 |
