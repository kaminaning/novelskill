- 平台：番茄小说

| 第 2 层 | 隐世武林宗师 | 武协老前辈 | 一本手抄"歪派拳谱"实为宗门内功 | 第一卷尾 |

| 第 6 层 | 玄霄宗末代掌门 + 守护人间渡劫者 | 上古妖兽/魔界余孽 | 那安深处的弟子牌位群 | 第四卷+ |

## ⭐ 代入感锚（必填）

### 主角共情锚（韦运福）

- **普世痛感**：宗门覆灭、所有弟子身死、独活三百年的孤独和负罪感（每 10-15 章触发 1 次回忆）
- **主角软肋**：那安深处的弟子牌位群（共 327 块）+ 当年师妹的玉簪遗物（藏在酸笋坛底）
- **主角理由链**：
  - 童年：上古时代孤儿，被玄霄宗祖师收留
  - 事件：300 年前魔界入侵人间，玄霄宗全员守关战死，主角作为掌门唯一活下来
  - 认知："活着比死更难，但我不能让宗门彻底消失"
  - 行为模式：藏身广西甘蔗田、独居避世、不与人深交、看似冷漠实则护短
- **道德爽感锚**：
  - 保护本村人（村霸欺压蔗农时出手）
  - 言出必行（答应小亮的事一定办到）
  - 不欺老幼（对村里孩子和老人格外温和）
  - 知恩图报（对救过他的村民后人格外照顾）
- **致命弱点**：
  - 修为虽高但人间灵气稀薄三百年，每次大量出手都要耗损根基
  - 弟子牌位被触碰会失控（被反派利用过 1 次）
  - 对师妹遗物（玉簪）的执念近乎偏执

### 视角主角共情锚（李明亮/小亮）

- **普世痛感**：返乡待业青年，大学毕业三年没找到稳定工作，被父母催婚催就业（番茄读者大共鸣群）
- **身份共鸣**：不火的小主播（粉丝 800 + 三年不涨）、月入 500、靠拍韦运福的怪人日常吃饭
- **缺口**：童年缺父爱（父亲在外打工长年不归），下意识把韦运福当父辈

## 番茄风承诺（前 3 章必兑现）

### 死亡开局类型

**冲突 + 反转 + 物件钩子三件套**：
- 开篇 200 字：村霸带人来收"保护费"，要韦运福交出甘蔗田。韦运福正在那安口钓鱼，懒得搭理。村霸踹翻鱼桶。韦运福反手一拎，把村霸单手举起来三米高扔出去。小亮举着手机当场拍下来，直播间在线人数从 7 跳到 343。

### 第 1 章打脸点

村霸被韦运福一只手举起来扔进河里，全程没看清主角怎么动的手。村霸找人来报复，反被韦运福用甘蔗砸得满头包。

### 第 3 章前的爆点

第 2 章末：小亮的视频被本地"功夫佬"刷到，对方判断这是"内家高手"，特意来挑战。第 3 章：韦运福一招制服功夫佬，对方跪地喊"老前辈"。直播间在线突破 5000。

### 第 10 章的新副本

第 9-10 章：武协老前辈亲自上门拜访，认出韦运福手抄的"歪派拳谱"竟是民国失传宗师"玄霄居士"的手笔。**第 1 层身份揭露（武学宗师）完成，进入第 2 层揭露。**

## 与对标书的差异化

| 维度 | 对标书《长生四十六亿年》 | 本书《广西甘蔗田里住着个老祖》 |
|------|------|------|
| 金手指 | 46 亿年寿命 + 蓝星开服第一人 | 玄霄宗末代老祖 + 修仙界传承 |
| 视角主角 | 妹妹苏念（亲妹） | 邻居小亮（非亲属，反而是返乡待业的小主播） |
| 主角形象 | 钓鱼咸鱼老哥 | 钓鱼+砍蔗+腌酸笋的独居老野（30 多岁外表） |
| 地点设定 | 苏州（江南文化） | **广西崇左江州（壮族甘蔗主产区，独特地域）** |
| 时代证人 | 跨清末-民国-抗战的老兵 | 跨上古-民国-现代的修仙界证人 |
| 情感锚点 | 林婉清 + 1125 封信（BE 爱情） | 弟子牌位群（覆灭兄弟情）+ 师妹玉簪（隐含 BE） |
| 物件钩子 | 勋章/地契/绝笔信 | 锈锄头/酸笋坛/烟杆/那安 |
| 数据反应层 | 直播在线 7 → 1 亿 | 直播在线 7 → 5000 → 5 万 → 千万级（参考对标书递进） |
| 反派 | 国家级反派 + 海外网友 | 修仙界宿敌（魔界余孽/前妖兽王后人）+ 现实势力 |

## 核心风险预警

| 风险 | 应对 |
|------|------|
| 广西方言滥用导致非本地读者看不懂 | 方言只用"句末语气词 + 几个常用词"，关键剧情用普通话 |
| "修仙隐世"被视为同质化 | 强差异化：广西地域 + 干栏+酸笋+蔗田的真实物质性 |
| 主角太强失去张力 | 灵气稀薄的设定让主角每次大动作都付代价 |
| 视角主角太工具化 | 给小亮自己的弧线：从混日子到真心想守护这片土地 |
| 反派只用现实势力会单薄 | 第二卷起引入修仙界宿敌，给主角真实威胁 |
