# 王大牛（第一卷大反派）

## 基本信息

| 项目 | 内容 |
|------|------|
| 本名 | 王大牛 |
| 外号 | 牛哥 / 大牛 |
| 年龄 | 32 岁 |
| 身份 | 蔗园村本村人，**村霸**，王老板（糖厂老板）的小舅子 |
| 籍贯 | 崇左市江州区新和镇蔗园村本地 |
| 外貌 | 1 米 8 大块头，皮肤黝黑，左眼有疤（年轻打架留的），常年穿黑色 T 恤 + 牛仔裤 + 金链子 |
| 标志物 | 手里永远抽红河烟、开一辆五菱宏光（黑色喷漆）、车后座两只藏獒 |

---

## 性格

- **欺软怕硬**：典型村霸性格，对老弱欺压，对真硬骨头怂
- **小聪明**：会算计但不会算大账（每次都吃眼前亏）
- **嘴硬**：被打了也死不认错，回头还要找场子
- **重利轻情**：跟姐夫姐姐其实关系不深，纯粹利用关系收"保护费"

---

## 真实威胁度（反派威胁 4 原则）

| 维度 | 表现 |
|------|------|
| 实力威胁 | 块头大、会几下散打、手下有 3-4 个跟班 |
| 资源威胁 | 姐夫是糖厂老板，垄断本地甘蔗收购价 |
| 智力威胁 | 第 8-10 章后会"升级"，开始用法律手段诬告主角（智斗升级） |
| 道德威胁 | 第 10-12 章会针对小亮（攻击主角软肋——朋友） |

---

## 与主角的冲突线（第一卷主线）

| 章节 | 冲突事件 |
|------|---------|
| 第 1 章 | 王大牛带人来收"保护费"，要韦运福交出甘蔗田。韦运福反手把他扔进河里。 |
| 第 2-3 章 | 王大牛纠集人手报复，再次被韦运福用甘蔗砸得满头包 |
| 第 4-5 章 | 王大牛叫姐夫王老板压低韦运福的甘蔗收购价（经济打击） |
| 第 6-7 章 | 王大牛雇外地"功夫佬"张师傅来挑战韦运福（结果张师傅拜服） |
| 第 8-10 章 | 王大牛升级手段：诬告韦运福私占村集体土地（**法律层面智斗**） |
| 第 11-14 章 | 王大牛找小亮麻烦，威胁断小亮的网线、找人砸三脚架 |
| 第 15 章 | 王大牛黔驴技穷，绑架了村里的莫世昌（莫阿婆的孙子），逼韦运福现身 |
| 第 16-17 章 | 韦运福首次"亮相"（半公开使用法力），救下莫世昌 |
| 第 18-20 章 | 王大牛被警方依法逮捕（韦运福通过武协周老爷子牵线让警方介入） |
| 第 25-30 章 | 王老板（姐夫）出狱后会成为第二卷的引子（更高层级反派） |

## 关键设计

### 为什么不能让王大牛一开始就被秒杀

**反派威胁度 = 主角爽感的基础**。如果王大牛第 1 章被扔进河里就消失，后面 10 章主角没有对手 = 红线触发。

王大牛的设计是：
- 第 1 章吃亏 = 让主角立威
- 第 2-10 章反复来 = 让冲突持续
- 升级手段（法律/经济/网络/朋友） = 让主角必须正面对待
- 第 15 章绑架莫世昌 = 触动主角软肋（道德爽感的反向触发）

### 王大牛的智力升级路径

避免"纸老虎"红线触发：
- 第 1-3 章纯武力威胁 → 失败
- 第 4-7 章经济威胁 → 失败但有真实压力
- 第 8-10 章法律手段 → 真实威胁主角合法身份
- 第 11-14 章针对朋友（小亮） → 触动主角软肋
- 第 15 章绑架弱者 → 触发主角真正出手

每一次升级都让王大牛"不那么蠢"。

## 经典台词

- "你个老野，敢动我？"（被扔进河之前）
- "我顶你个肺！姐夫，今天你必须给我做主！"（电话求姐夫）
- "老野，你等着，我要让你滚出蔗园村"（被打后撂狠话）
- "你以为你是谁啊？村里来的怪东西"（升级矛盾时）

---

## 失败案例（避免写成这样）

- ❌ 王大牛从头到尾只会喊"老子要弄死你"（智力降级）
- ❌ 王大牛第 1 章就被秒杀消失（反派纸老虎）
- ❌ 王大牛无理由作恶（必须有"姐夫撑腰收保护费"的动机链）
- ❌ 王大牛智商低到看不出主角不能惹（每次都吃亏但有学习能力）
