> 写完每章后必须自检：本章用了哪些方言 / 饮食 / 生活细节 / 地理气候？是否有不准确的"广东元素 / 四川元素 / 湖广元素"误入？

## 一、桂柳话词汇库（西南官话桂柳片，A 级常用）

### 高频替换词（写作时直接替换普通话）

| 普通话 | 桂柳话 | 例句 |
|--------|--------|------|
| **去** | **克** | "我克镇上嗦粉"（我去镇上嗦粉）|
| **鞋** | **孩** | "你的孩湿了"（你的鞋湿了）|
| **小孩** | **娃仔** | "细娃仔莫闹"（小孩别闹）|
| **青蛙** | **蚂拐** | "那安里好多蚂拐"|
| **干什么** | **揍么子 / 揍酿** | "你揍么子？"（你干嘛？）|
| **出来** | **渠来** | "你渠来一下"|
| **出去** | **出克** | "出克买烟"|
| **怎么样** | **嫩子** | "今天嫩子？"（今天怎么样？）|
| **糟糕** | **挨卵** | "挨卵了，下雨了"|
| **蟑螂** | **搔夹** | "厨房有搔夹"|
| **戳** | **笃 / dok** | "笃他一下"|
| **蹲** | **踎 / mou** | "踎在台阶上"|
| **烧焦** | **燶 / nung** | "饭燶了"|
| **个** | **过** | "一过人"（一个人，使用谨慎，不是每个场合都用）|
| **解** | **改** | / |
| **六** | **路（lu）** | / |

### 程度副词（"好卵 X"）

- "好卵热" = 非常热
- "好卵远" = 非常远
- "好卵贵" = 非常贵
- ⚠️ "卵"虽然听起来粗鄙，但日常用，本书可在**村霸/糖厂工人/农忙汉子**等粗豪角色嘴里用 1-2 次，**福哥不用**。

### 常用语气/称呼

| 词 | 含义 | 使用 |
|----|------|------|
| **得啦** | 行了/可以了 | 全员通用，福哥高频 |
| **晓得** | 知道 | 通用 |
| **莫** | 不要 | "莫闹" "莫管" |
| **嗯** | 默认/承认 | 福哥高频 |
| **嘞** | 句末完成态 | "搞定嘞" |
| **三八货** | 疯疯癫癫的人（戏称）| 配角间互称 |
| **夜屎佬** | 多管闲事的人 | 配角间调侃 |

### 桂柳话发音规律（村人对话需体现）

- **没有卷舌音**：zh/ch/sh/r → z/c/s/y
  - "是" → "xi"（村人念）
  - "知道" → "zi 道"
  - "日" → "yi"
- **n/l 可分**（与其他西南官话不同，桂柳是 n/l 分）
- **第二声往下走**（与普通话相反）—— 不需要写出来，但写人物对话时知道这个差异

## 二、南普（南宁普通话）词汇库（B 级，村人/年轻人用）

### 尾音特色

| 尾音 | 用法 | 例子 |
|------|------|------|
| **喂** | 句末加强语气 | "今天好热喂" |
| **捏** | "呢"的南普变体 | "你揍么子捏？" |
| **咩** | 反问/疑问 | "真的咩？" |
| **嘞** | 完成态 | "搞定嘞" |
| **咯** | 句末叹词 | "落雨咯" |

### 词序倒装（受壮语影响，但用汉字表达）

| 标准 | 南普 |
|------|------|
| 很冷 | **冷多** |
| 很热 | **热多** |
| 男友 | **友仔** |
| 女友 | **友女** |
| 两只鞋 | **孩两只** / **鞋两只**（量词后置） |
| 三个人 | **人三过**（罕用） |

### 南普发音错位（配角群像偶尔出现）

| 普通话 | 南普 | 原因 |
|--------|------|------|
| 痛 → 怼 | "好怼" | n/l 不分 + 卷平舌不分 |
| 你 → 里 | "里是谁" | n/l 不分 |
| 天 → 颠 | "颠颠向上" | t/d 不分 |
| 鬼 → 给 | "鬼来了"→"给来了" | k/g 不分 |
| 师 → 西 | "老师"→"老西" | 平翘不分 |

⚠️ **福哥说标准普通话**，发音错位不出现在福哥嘴里。这是福哥与村人的差异点。

## 三、桂南白话词汇库（C 级，王老板/糖厂工人/外来商人用）

桂南白话属粤语系，与广东粤语相似但有差异。本书设定王老板（糖厂老板）和镇上做生意的人说桂南白话。

### 常用词

| 桂南白话 | 普通话 | 使用场景 |
|---------|--------|---------|
| **嗨** | 语气词（叹词）| "嗨！你这老板！" |
| **得没得** | 行不行 | "晚上喝酒得没得？" |
| **搞掂** | 搞定 | "事情搞掂嘞" |
| **犀利** | 厉害 | "你犀利！" |
| **好嘢** | 好东西/真棒 | "这酸笋好嘢！" |
| **饮** | 喝 | "饮酒" |
| **食** | 吃 | "食饭" |
| **几** | 多少 | "几钱？"（多少钱）|
| **捻** | 想 | "我捻一下"（我想一下）|
| **没系** | 不是 | "没系咁样"（不是这样）|
| **冇** | 没有 | "冇钱" |
| **咩** | 什么 | "搞咩？" |
| **嗰** | 那个 | "嗰个老野" |

⚠️ **福哥不主说桂南白话**，因为崇左壮族区的中老年人主要说壮语+桂柳话，桂南白话主要在镇上做生意的人嘴里出现（王老板/糖厂老板/外地商人）。

## 五、广西饮食强制清单

### 早餐（每章正文涉及早晨必体现 1 项）

| 早餐 | 描述 | 适用场景 |
|------|------|---------|
| **老友粉** | 酸笋+豆豉+蒜蓉+辣椒+番茄+肉末（南宁起源，崇左常见）| 福哥每周去镇上嗦 |
| **生榨米粉** | 米团发酵 2-3 天，微酸的"馊味"（蒲庙特色）| 福哥自己舂的早餐 |
| **卷筒粉** | 米浆蒸薄皮+豆芽木耳肉末+黄皮酱 | 镇集市早摊 |
| **酸粥** | 米饭发酵制成，蔗农下地必备 | **福哥下地前必吃** |
| **龙州鸡肉粉** | 土鸡熬汤+手工米粉+白切鸡+柠檬汁 | 福哥去龙州时 |

### 主菜/小吃（必体现"酸""糯""粉"三大特色）

| 菜 | 描述 | 适用 |
|----|------|------|
| **酸笋** | 福哥自腌一坛一坛（实为储物法器）| 贯穿全书 |
| **酸嘢** | 青芒/木瓜/番石榴泡米醋+椒盐+辣椒 | 福哥送阿芳 |
| **五色糯米饭** | 白/黄/黑/红/紫五色，三月三必做 | **祭弟子核心物件** |
| **酸粥牛腩 / 酸粥鱼仔 / 酸粥黄豆** | 崇左特色 | 福哥家常 |
| **假蒌叶包猪肉** | 假蒌叶包肉末油炸 | 镇集市小吃 |
| **芭蕉叶糍粑** | 糯米+花生芝麻糖/红豆沙 | 福哥过节做 |
| **桄榔粉** | 桄榔树髓芯制食用淀粉，崇左龙州特产 | 福哥泡水 |

### 喝的

| 饮品 | 描述 | 适用 |
|------|------|------|
| **自酿米酒** | 福哥的招牌，5 年起步 | 福哥独饮 |
| **甘蔗汁** | 现榨 | 砍蔗后自饮 |
| **凉茶（草根/茅根）** | 广西夏天必备 | 福哥夏天喝 |
| ❌ 早茶/普洱 | 广东特色 | 不要用 |

### 福哥的禁忌

- **不吃牛肉**（壮族禁忌 + 当年坐骑青牛精转世）
- **不喝市面酒**（只喝自酿米酒）
- **不吃早茶**（嗦粉为生）

## 七、广西地理 / 气候强制清单

### 气候（影响每章天气）

- **南亚热带季风气候**：长夏无冬，春秋相连
- **3 月**（故事开始）：蔗收末期，15-25°C，雨水开始多
- **4 月**：清明前后，雨水多，热起来
- **5-9 月**：夏季漫长，闷热多雨，有台风
- **10-11 月**：秋凉短
- **12-2 月**：冬季短而温暖

### 地理

- **喀斯特地貌**：石山、溶洞（那安典型）、河流穿过
- **左江流域**：村里小河是左江支流
- **大青山 / 公母山 / 十万大山**：背景山脉

### 物产

- **甘蔗**（村里 70% 收入来源）
- **剑麻 / 八角 / 桂皮 / 玉桂**
- **沙田柚 / 武鸣沃柑**
- **百香果 / 火龙果 / 芒果 / 龙眼 / 荔枝**
