# Artifact 创建模板

各 artifact 的标准模板与创建指引。agent 在 Phase 2-3 过渡时按需加载此文件。

**模板列表：**
- 设定/角色/{角色名}.md（主角卡 / 配角卡 / 反派卡）
- 设定/势力/{势力名}.md
- 设定/世界观/{主题}.md
- 设定/关系.md
- 设定/题材定位.md
- 设定/书名与简介.md
- 大纲/卷纲_第X卷.md（含情绪弧线+反转规划）
- 大纲/细纲_第XXX章.md（含钩子设计+章节结构+爽点+关键场景细节）
- 追踪/伏笔.md
- 追踪/时间线.md
- 对标/{对标书名}/拆文报告.md
- 对标/{对标书名}/原文/第XXX章_{章名}.md

**层级关系：**
- 大纲.md = 全书鸟瞰（每卷一两句话定位）
- 卷纲_第X卷.md = 单卷规划（爽点节奏+情绪+人物+伏笔+反转）
- 细纲_第XXX章.md = 单章蓝图（事件+钩子+爽点+悬念，钩子直接写在细纲里）

## 对标/{对标书名}/拆文报告.md

此文件由 `story-long-analyze` 的快速模式或深度模式输出。write skill 的职责是**读取**，不创建。

如需手动创建简版对标摘要（未使用 analyze skill 时）：

```markdown
# {对标书名} 对标摘要

## 基本信息
- 书名：{}
- 作者：{}
- 题材/类型：{}
- 目标平台：{}
- 成绩：{均订/在读/热度}

## 核心发现
- 开篇钩子：{类型 + 手法}
- 爽点密度：{约 N 字/次}
- 节奏模式：{描述}
- 可借鉴套路：
  1. {}
  2. {}
  3. {}

## 禁止照搬
- {}（只学结构，不抄桥段）
```

## 大纲/大纲.md 修订记录字段

大纲.md 在 Phase 4.5 阶段回顾时可能需要更新。更新时在文件末尾追加修订记录表：

```markdown
## 大纲修订记录
| 日期 | 当前章节 | 修订内容 | 原因{阶段回顾发现偏差/用户要求/细纲续生成调整} |
|------|---------|---------|----------------------------------------------|
```

首次阶段回顾执行时，agent 应将此表追加到现有 大纲.md 文件末尾。后续每次修订在此表中追加新行。
